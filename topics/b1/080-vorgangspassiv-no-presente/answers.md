# Respostas: Vorgangspassiv no presente

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | wird | **Das Haus** é singular neutro, então usa **wird**. |
| 1.2 | werden | **Die E-Mails** está no plural, então usa **werden**. |
| 1.3 | werde | Com **ich**, a forma de **werden** é **werde**. |
| 1.4 | wird | **Der Vertrag** é singular masculino, então usa **wird**. |
| 1.5 | werden | **Die Kinder** está no plural, então usa **werden**. |
| 2.1 | geöffnet | O Partizip II de **öffnen** é **geöffnet**. |
| 2.2 | geschrieben | O Partizip II de **schreiben** é **geschrieben**. |
| 2.3 | repariert | Verbos terminados em **-ieren** geralmente formam Partizip II sem **ge-**. |
| 2.4 | gekocht | O Partizip II de **kochen** é **gekocht**. |
| 2.5 | kontrolliert | **kontrollieren** forma **kontrolliert**, sem **ge-**. |
| 3.1 | Der Computer wird repariert. | A estrutura correta é sujeito + **wird** + Partizip II. |
| 3.2 | Die Rechnung wird bezahlt. | **Die Rechnung** é singular; por isso, **wird bezahlt**. |
| 3.3 | Die Briefe werden geschrieben. | No plural, a forma correta é **werden**. |
| 3.4 | Wird das Paket heute geliefert? | Em pergunta sim/não, o verbo conjugado vem em primeiro lugar. |
| 4.1 | Eine Brücke wird gebaut. | O objeto **eine Brücke** vira sujeito da passiva; o agente pode ser omitido. |
| 4.2 | Das Fenster wird geöffnet. | **das Fenster** vira sujeito e usa **wird geöffnet**. |
| 4.3 | Die Einladungen werden von der Sekretärin geschrieben. | O plural exige **werden**; a agente vem com **von**. |
| 4.4 | Die Rechnung wird online bezahlt. | **man** desaparece na passiva; o foco fica na conta. |
| 4.5 | Das Auto wird vom Mechaniker repariert. | **vom** é contração de **von dem** e marca o agente. |
| 5.1 | durch | Uma app é meio ou instrumento; por isso, usa-se **durch**. |
| 5.2 | von | O médico é agente direto; por isso, usa-se **von**. |
| 5.3 | durch | O código é o meio que abre a porta. |
| 5.4 | von | A diretora é agente da decisão, então usa-se **von**. |
| 6.1 | Das Haus wird gebaut. | Para ação em processo, use **werden**, não **sein**. |
| 6.2 | Die Dokumente werden geprüft. | **Dokumente** está no plural, então o verbo é **werden**. |
| 6.3 | Der Brief wird geschrieben. | A passiva exige Partizip II, não infinitivo. |
| 6.4 | Der Artikel wird gelesen. | O objeto da ativa vira sujeito no nominativo: **der Artikel**. |
| 7.1 | Das Fenster wird geputzt. | **das Fenster** é singular; usa **wird + geputzt**. |
| 7.2 | Die Dokumente werden geprüft. | Plural exige **werden**; **prüfen** vira **geprüft**. |
| 7.3 | Die Entscheidung wird von der Managerin getroffen. | A gerente é agente direta, então use **von**. |
| 7.4 | Der Text wird durch eine App gesendet. | A app é meio técnico; por isso, use **durch**. |
