# Respostas: Caso genitivo: posse

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | des | **Mann** é masculino; no genitivo usa **des** e aparece como **Mannes**. |
| 1.2 | der | **Tasche** é feminino; no genitivo usa **der**. |
| 1.3 | des | **Haus** é neutro; no genitivo usa **des** e vira **Hauses**. |
| 1.4 | der | **Kinder** está no plural; o genitivo plural usa **der**. |
| 1.5 | der | **Chefin** é feminino; por isso o genitivo é **der Chefin**. |
| 2.1 | meines | **Sohn** é masculino; o possessivo “meu” no genitivo masculino é **meines**. |
| 2.2 | meiner | **Schwester** é feminino; o genitivo feminino de “meu/minha” é **meiner**. |
| 2.3 | seines | **Vater** é masculino; “dele” no genitivo masculino é **seines**. |
| 2.4 | ihrer | **Kollegin** é feminino; “dela” no genitivo feminino é **ihrer**. |
| 2.5 | unserer | **Freunde** está no plural; “nossos” no genitivo plural é **unserer**. |
| 3.1 | das Auto meines Bruders | Para **der Bruder**, usa-se **meines** e o substantivo recebe **-s**. |
| 3.2 | die Tür des Hauses | **Haus** é neutro; o genitivo é **des Hauses**. |
| 3.3 | die Namen der Kinder | No plural, o artigo genitivo é **der** e **Kinder** não muda aqui. |
| 3.4 | die Meinung der Lehrerin | **Lehrerin** é feminino; o genitivo é **der Lehrerin**. |
| 4.1 | der Schlüssel des Vaters | **von dem Vater** pode virar **des Vaters** em registro formal. |
| 4.2 | die Idee meiner Kollegin | **meiner Kollegin** é genitivo feminino com possessivo. |
| 4.3 | das Zimmer des Kindes | **Kind** é neutro; usa **des** e final **-es**. |
| 4.4 | die Telefonnummer unserer Freunde | **Freunde** é plural; o possessivo no genitivo plural é **unserer**. |
| 4.5 | der Preis des Fahrrads | **Fahrrad** é neutro; usa **des** e final **-s**. |
| 5.1 | Der Name des Mannes ist Paul. | Masculino singular no genitivo exige **des** e normalmente final **-es**. |
| 5.2 | Die Tür des Hauses ist offen. | **Haus** precisa da forma genitiva **Hauses**. |
| 5.3 | Das Handy meines Bruders liegt hier. | **Bruder** é masculino; use **meines Bruders**, não **meiner Bruder**. |
| 5.4 | Die Freundin der Schwester kommt heute. | **Schwester** é feminino; o genitivo correto é **der Schwester**. |
| 6.1 | Der Name des Lehrers ist Müller. | **Lehrer** é masculino; genitivo: **des Lehrers**. |
| 6.2 | Die Tasche meiner Mutter ist rot. | **Mutter** é feminino; “minha” no genitivo é **meiner**. |
| 6.3 | Wegen des Regens fahren wir mit dem Bus. | Em registro formal, **wegen** combina com genitivo: **des Regens**. |
| 6.4 | Trotz des Problems ist er ruhig. | **trotz** pode usar genitivo: **des Problems**. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Wo beginnt Julia ihr Praktikum? | Sie beginnt ihr Praktikum in einer kleinen Firma. |
| 2. Was findet Julia am Anfang nicht? | Sie findet den Raum des Chefs nicht. |
| 3. Warum notiert Julia die Telefonnummer der Sekretärin? | Sie braucht später einen Termin. |
| 4. Was fehlt in der E-Mail am Nachmittag? | Der Anhang der Nachricht fehlt. |
| 5. Wessen Laptop braucht Julia kurz? | Sie braucht den Laptop ihres Kollegen. |
| 6. Wie bereitet Julia die Folien trotz des Problems vor? | Sie bereitet sie ruhig vor. |
| 7. Mit wem spricht Julia nach der Arbeit? | Sie spricht mit ihrem Bruder im Café. |
| 8. Qual expressão da história significa “por causa do novo projeto”? | **Wegen des neuen Projekts** significa “por causa do novo projeto”. |
| Prática 1 | **des**; **Büro** é neutro e usa **des** no genitivo. |
| Prática 2 | **der**; **Sekretärin** é feminino e usa **der** no genitivo. |
| Prática 3 | **der Laptop ihres Kollegen**; o possessivo no genitivo masculino é **ihres**. |
| Prática 4 | **der erste Tag des Praktikums**; **Praktikum** é neutro e vira **des Praktikums**. |
| Prática 5 | **trotz des Problems**; é a expressão com **trotz + genitivo**. |
| Prática 6 | Resposta modelo: **Wegen des Wetters bleibe ich zu Hause.** Usa **wegen + genitivo**. |
