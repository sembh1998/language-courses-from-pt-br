| 3.3 | esperançoso | `-voll` indica cheio de algo; `hoffnungsvoll` é “cheio de esperança”. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Warum sucht Mara eine neue Arbeit? | Porque ela quer mais liberdade e uma melhor possibilidade de se desenvolver. |
| 2. Welche Wörter im Text haben den Suffix `-ung`? | `Bewerbung`, `Entscheidung`, `Wirkung` e `Veränderung`. |
| 3. Was sagt Lena über Maras Bewerbung? | Ela diz que a candidatura é legível, mas ainda soa um pouco sem esperança. |
| 4. Wie verändert Mara den Text? | Ela muda algumas frases para que o texto pareça mais amigável e esperançoso. |
| 5. Was entdeckt Mara im Gespräch? | Ela descobre que a vaga combina com ela. |
| 6. Complete com uma palavra da história: `Eine kleine ___ kann eine große Wirkung haben.` | `Veränderung`, porque a frase fala de uma mudança pequena. |
| 7. Reescreva com o oposto: `Der Text klingt hoffnungslos.` Use `hoffnungsvoll`. | `Der Text klingt hoffnungsvoll.` O sufixo `-voll` indica cheio de esperança. |
| 8. Qual é o artigo de `Möglichkeit` e por quê? | `die Möglichkeit`, porque palavras com `-keit` são geralmente femininas. |
| Prática 1 | `Freundschaft`; ela termina em `-schaft`. |
| Prática 2 | `verändert` ou `Veränderung`; ambas trazem o prefixo `ver-`. |
| Prática 3 | `die Hoffnung`; `hoffnungsvoll` vem de `Hoffnung` + `-voll`. |
| Prática 4 | `les`; `lesbar` significa legível. |
| Prática 5 | Resposta possível: `Im Urlaub entdecke ich neue Cafés.` Deve usar `entdecken` corretamente. |
| Prática 6 | `hoffnungslos` significa sem esperança; `hoffnungsvoll` significa esperançoso ou cheio de esperança. |
