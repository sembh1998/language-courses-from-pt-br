# Respostas: Orações relativas no nominativo

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | der | **Arzt** é masculino e o relativo é sujeito de **arbeitet**. |
| 1.2 | die | **Studentin** é feminino e o relativo é sujeito de **lernt**. |
| 1.3 | das | **Baby** é neutro e o relativo é sujeito de **schläft**. |
| 1.4 | die | **Bücher** está no plural e o relativo é sujeito de **liegen**. |
| 1.5 | der | **Bus** é masculino e o relativo é sujeito de **kommt**. |
| 1.6 | das | **Hotel** é neutro e o relativo é sujeito de **liegt**. |
| 2.1 | Der Nachbar, der Musik hört, ist jung. | Para masculino singular no nominativo, usa-se **der**. |
| 2.2 | Die Frau, die in Leipzig wohnt, ist Ärztin. | O verbo **wohnt** deve ficar no fim da oração relativa. |
| 2.3 | Das Mädchen, das Geige spielt, ist talentiert. | **Mädchen** é neutro em alemão, por isso o relativo é **das**. |
| 2.4 | Die Kunden, die früh kommen, bekommen Kaffee. | No plural nominativo, o pronome relativo é **die**. |
| 2.5 | Ich sehe den Hund, der im Hof bellt. | **der** é sujeito de **bellt**, então está no nominativo. |
| 2.6 | Der Film, der heute läuft, ist spannend. | A oração relativa deve ser separada por vírgulas. |
| 3.1 | Der Lehrer, der Mathe unterrichtet, ist streng. | **der** retoma o masculino **Lehrer** e o verbo da relativa vai ao fim. |
| 3.2 | Die Katze, die auf dem Sofa schläft, ist schwarz. | **die** retoma **Katze** e é sujeito de **schläft**. |
| 3.3 | Das Restaurant, das um zwölf Uhr öffnet, ist beliebt. | **Restaurant** é neutro, então o relativo nominativo é **das**. |
| 3.4 | Die Kinder, die seit zwei Stunden spielen, sind müde. | Plural usa **die** e o verbo **spielen** fica no fim da relativa. |
| 3.5 | Der Computer, der im Arbeitszimmer steht, ist neu. | **Computer** é masculino e **der** é sujeito de **steht**. |
| 4.1 | die, porque Chefin é feminino e o relativo é sujeito. | A forma feminina nominativa do pronome relativo é **die**. |
| 4.2 | das, porque Fahrrad é neutro e o relativo é sujeito. | **Fahrrad** é neutro, por isso o relativo é **das**. |
| 4.3 | der, porque Zug é masculino e o relativo é sujeito. | **Zug** é masculino, então o relativo nominativo é **der**. |
| 4.4 | die, porque Freunde está no plural e o relativo é sujeito. | Para plural nominativo, usa-se **die**. |
| 4.5 | die, porque Lampe é feminino e o relativo é sujeito. | **Lampe** é feminino, então o relativo é **die**. |
| 5.1 | Der Mann, der hier wohnt, ist Journalist. | **der Mann** é masculino e **der** é sujeito de **wohnt**. |
| 5.2 | Die Frau, die mir hilft, ist meine Nachbarin. | **die** retoma **Frau** e funciona como sujeito de **hilft**. |
| 5.3 | Das Kind, das im Zimmer liest, ist leise. | **Kind** é neutro, então use **das**. |
| 5.4 | Die Kollegen, die mit mir arbeiten, sind sympathisch. | **Kollegen** está no plural, então use **die**. |
| 5.5 | Das Café, das in der Nähe vom Bahnhof liegt, ist klein. | **Café** é neutro e o relativo é sujeito de **liegt**. |
| 6.1 | Der Mann, der im Garten arbeitet, ist mein Onkel. | **Mann** é masculino; **die** estava incorreto. |
| 6.2 | Die Lehrerin, die die Aufgabe erklärt, ist geduldig. | Na oração relativa, o verbo **erklärt** deve ir para o fim. |
| 6.3 | Das Haus, das am Fluss steht, ist alt. | **Haus** é neutro, então o relativo correto é **das**. |
| 6.4 | Die Spieler, die heute trainieren, sind fit. | Faltavam as vírgulas que separam a oração relativa. |
| 6.5 | Ich kenne einen Musiker, der jeden Abend spielt. | Embora apareça **einen Musiker**, o relativo é sujeito de **spielt**, então é **der**. |
