# História: Revisão geral B1

Nível: B1

## História em alemão

### Szene 1: Vor der Prüfung

Mara bereitet sich auf eine B1-Prüfung vor, die nächste Woche stattfindet. Sie wiederholt die Grammatik, die ihr noch schwerfällt. Besonders die Relativsätze, bei denen man auf den Kasus achten muss, findet sie kompliziert. Ihr Freund Jonas, der die Prüfung schon bestanden hat, gibt ihr einen Tipp: „Schreib jeden Tag kurze Sätze, aber kontrolliere danach die Verbposition.“

### Szene 2: Im Kurs

Im Deutschkurs wird heute ein Modelltest geschrieben. Die Aufgaben werden von der Lehrerin erklärt, bevor die Zeit beginnt. Obwohl Mara nervös ist, bleibt sie ruhig, weil sie gut vorbereitet ist. Wenn sie mehr Zeit hätte, würde sie jeden Text zweimal lesen, aber im Test muss sie schnell entscheiden. Trotzdem markiert sie wichtige Wörter, damit sie die Fragen besser versteht.

### Szene 3: Nach dem Test

Nach dem schriftlichen Teil spricht Mara mit Jonas über die mündliche Prüfung. Sie möchte ihre Meinung klar begründen und passende Konnektoren benutzen. Jonas sagt: „Wenn ich an deiner Stelle wäre, würde ich zuerst eine kurze Struktur planen.“ Am Abend schreibt Mara eine Liste mit Redemitteln, die sie morgen üben möchte. Sie ist müde, trotzdem ist sie zufrieden, denn sie merkt, dass sie sicherer geworden ist.

## Notas de vocabulário

- sich auf eine Prüfung vorbereiten = preparar-se para uma prova
- stattfinden = acontecer; ocorrer
- schwerfallen = ser difícil para alguém
- auf den Kasus achten = prestar atenção ao caso gramatical
- einen Modelltest schreiben = fazer um simulado
- die Aufgabe wird erklärt = a tarefa é explicada
- obwohl = embora; apesar de que
- trotzdem = mesmo assim; apesar disso
- die Meinung begründen = justificar a opinião
- an deiner Stelle = no seu lugar

5. Wozu markiert Mara wichtige Wörter?
6. Was möchte Mara in der mündlichen Prüfung klar machen?
7. Qual conselho Jonas dá para a prova oral?
