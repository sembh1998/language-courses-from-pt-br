# Respostas: Revisão geral B1

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | der | `der Nachbar` é masculino e sujeito de `hat geholfen`, então o relativo fica no nominativo masculino. |
| 1.2 | das | `das Buch` é neutro; na relativa, é objeto de `empfohlen hast`, mas neutro nominativo e acusativo têm a mesma forma `das`. |
| 1.3 | der | A preposição `mit` exige dativo; `die Frau` no dativo feminino vira `der`. |
| 1.4 | die | `die Kinder` é plural e objeto de `gesehen haben`; no acusativo plural o relativo é `die`. |
| 1.5 | der | `der Mann` é o sujeito de `steht`, então o pronome relativo é nominativo masculino `der`. |
| 1.6 | denen | O professor explicou a tarefa aos estudantes; plural no dativo usa `denen`. |
| 2.1 | hätte; würde | Para hipótese irreal, usa-se `wenn + hätte` e `würde + infinitivo`. |
| 2.2 | Könnten | `Könnten Sie...?` é a forma educada em Konjunktiv II. |
| 2.3 | wäre | `wäre` é o Konjunktiv II de `sein`. |
| 2.4 | würde | Para conselho, `An deiner Stelle würde ich...` é uma estrutura muito comum. |
| 2.5 | hätten | `hätten gern` expressa desejo educado. |
| 3.1 | werden von den Mitarbeitern geprüft | Na passiva presente, o objeto vira sujeito e a forma é `werden + Partizip II`. |
| 3.2 | wird um acht Uhr geöffnet | Com `man`, a passiva foca no processo; `das Büro` é neutro singular, então `wird`. |
| 3.3 | wurde gestern von der Firma bezahlt | O verbo original está no passado simples; por isso a passiva usa `wurde + Partizip II`. |
| 3.4 | werden am Nachmittag beantwortet | `die E-Mails` está no plural, então a forma de `werden` é `werden`. |
| 3.5 | wird von der Lehrerin korrigiert | `der Test` é masculino singular; a passiva presente usa `wird korrigiert`. |
| 4.1 | deshalb | Há relação de consequência; depois de `deshalb`, o verbo vem em segunda posição. |
| 4.2 | Obwohl | `obwohl` introduz contraste e manda o verbo conjugado para o fim da subordinada. |
| 4.3 | weil | `weil` introduz motivo: a pessoa fica em casa porque está com dor de cabeça. |
| 4.4 | trotzdem | Há contraste: o curso é caro, mas a pessoa quer se inscrever mesmo assim. |
| 4.5 | dass | Depois de `Ich glaube`, `dass` introduz o conteúdo da opinião. |
| 4.6 | weil | A segunda oração apresenta a razão para a formação de Lisa. |
| 5.1 | aufstehen muss | Com `weil`, o verbo conjugado vai para o fim; em estrutura modal, o modal fica no fim. |
| 5.2 | haben | Depois de `deshalb`, o verbo conjugado fica em posição 2: `deshalb haben wir...`. |
| 5.3 | hat | Em subordinada com `obwohl`, o verbo conjugado aparece no fim da oração. |
| 5.4 | hat | Em pergunta indireta com `ob`, o verbo conjugado fica no fim. |
| 5.5 | ist | Depois de `trotzdem`, o verbo vem em posição 2: `trotzdem ist...`. |
| 6.1 | Könnten Sie mir bitte mehr Informationen über den Kurs schicken? | A resposta usa `Könnten Sie...?`, adequado para pedido formal e educado. |
| 6.2 | An deiner Stelle würde ich jeden Tag ein bisschen Deutsch hören. | `An deiner Stelle würde ich...` é uma fórmula clara para dar conselho. |
| 6.3 | Ich lerne Deutsch, weil ich in Deutschland arbeiten möchte. | `weil` introduz a razão e coloca o verbo conjugado no fim. |
| 6.4 | Die Anmeldung wird online gemacht. | A frase usa passiva presente com `wird + Partizip II`. |
| 6.5 | Das ist die Person, die mir geholfen hat. | A oração relativa acrescenta informação sobre `die Person`; `die` é nominativo feminino. |
