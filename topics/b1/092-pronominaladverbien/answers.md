# Respostas: Pronominaladverbien

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | darauf | **warten auf** retoma uma coisa: **die Bestätigung**. Como **auf** começa com vogal, usa-se **darauf**. |
| 1.2 | daran | **denken an** retoma o compromisso; **an** pede o composto **daran**. |
| 1.3 | damit | O programa é uma ferramenta/coisa usada para trabalhar; com **mit**, usa-se **damit**. |
| 1.4 | dafür | O dinheiro é para o projeto; com **für**, o composto é **dafür**. |
| 1.5 | darauf | **sich freuen auf** retoma o fim de semana; por isso **darauf**. |
| 1.6 | dafür | **sich interessieren für** retoma o curso como tema; por isso **dafür**. |
| 2.1 | Worauf wartest du? | A resposta usa **auf das Ergebnis**, então a pergunta com **auf** é **worauf**. |
| 2.2 | Woran denkst du? | O verbo **denken an** pede **woran** na pergunta. |
| 2.3 | Womit bezahlst du? | A resposta fala do meio de pagamento, com **mit**; a pergunta é **womit**. |
| 2.4 | Wofür brauchst du das Geld? | A resposta indica finalidade com **für die Miete**; a pergunta é **wofür**. |
| 2.5 | Woran erinnerst du dich? | **sich erinnern an** pede **woran** em perguntas sobre coisas/eventos. |
| 2.6 | Wofür interessierst du dich? | **sich interessieren für** pede **wofür** na pergunta. |
| 3.1 | Ich freue mich darauf. | A Einladung é uma coisa; **sich freuen auf** vira **darauf**. |
| 3.2 | Ich denke oft daran. | **an die Prüfung** é retomado por **daran**. |
| 3.3 | Das Team plant die Aufgaben damit. | O Online-Tool é um instrumento; com **mit**, usa-se **damit**. |
| 3.4 | Viele Leute engagieren sich dafür. | **sich engagieren für** retoma a ajuda/projeto com **dafür**. |
| 3.5 | Wir müssen uns daran gewöhnen. | **sich gewöhnen an** retoma **die Regeln** com **daran**. |
| 3.6 | Ich warte darauf. | **warten auf** retoma **den Vertrag** com **darauf**. |
| 4.1 | Ich warte auf Maria. Ich warte auf sie. | Maria é pessoa; use **auf sie**, não **darauf**. |
| 4.2 | Woran denkst du? Ich denke an meinen Urlaub. | O verbo é **denken an**, então a pergunta correta é **woran**. |
| 4.3 | Ich spreche mit meinem Kollegen. Ich spreche mit ihm. | Colega é pessoa; use **mit ihm**, não **damit**. |
| 4.4 | Worauf freust du dich? Ich freue mich auf den Besuch. | **sich freuen auf** pede **worauf** na pergunta. |
| 4.5 | Ich interessiere mich für den Kurs. Ich interessiere mich dafür. | **sich interessieren für** usa **dafür**, não **daran**. |
| 5.1 | Ich warte darauf. | A frase retoma uma coisa esperada com **warten auf**. |
| 5.2 | Woran denkst du? | “Em que” com **denken an** vira **woran**. |
| 5.3 | Ich arbeite jeden Tag damit. | “Com isso” como ferramenta ou meio é **damit**. |
| 5.4 | Wofür brauchst du das? | “Para que” com **für** vira **wofür**. |
| 5.5 | Ich interessiere mich dafür. | **sich interessieren für** retoma o tema com **dafür**. |
| 5.6 | Womit schreibst du? | “Com o que” como instrumento é **womit**. |
| 6.1 | Ich warte auf eine Nachricht. Ich warte darauf. | Resposta modelo; precisa retomar algo com **warten auf** e **darauf**. |
| 6.2 | Ich denke oft an meine Pläne. Ich denke oft daran. | Resposta modelo; **denken an** retoma o tema com **daran**. |
| 6.3 | Ich lerne mit einer App. Ich lerne damit. | Resposta modelo; **damit** retoma o meio usado para aprender. |
| 6.4 | Ich spare für eine Reise. Ich spare dafür. | Resposta modelo; **dafür** retoma a finalidade da economia. |

## Gabarito do teste

| ID | Resposta | Explicação | Pontos |
|---|---|---|---|
| 1 | Worauf freust du dich? | **sich freuen auf** pede **worauf** em perguntas sobre coisas ou eventos. | 1 |
| 2 | Wir suchen danach. | **suchen nach** retoma **eine Lösung** com **danach**. | 1 |
| 3 | Ich spreche mit ihr. | Para pessoa, usa-se **mit ihr**, não o composto **damit**. | 1 |
| 4 | Womit fährst du ins Büro? | Meio de transporte ou instrumento com **mit** vira **womit**. | 1 |
| 5 | Sie ist dafür. | A pergunta com **wofür** recebe resposta com **dafür**. | 1 |
| 6 | Ich denke daran. | Antes de vogal em **an**, aparece **r**: **daran**. | 1 |
| 7 | daran | **sich erinnern an** retoma o filme com **daran**. | 1 |
| 8 | damit | O dicionário é meio/ferramenta; com **mit**, use **damit**. | 1 |
| 9 | darüber | **diskutieren über** retoma o plano com **darüber**. | 1 |
| 10 | dafür | **kämpfen für** retoma melhores horários de trabalho com **dafür**. | 1 |
| 11 | Ich warte darauf. | **warten auf** retoma **die Entscheidung** com **darauf**. | 1 |
| 12 | Womit arbeitest du? | A pergunta retoma **mit welchem Gerät** com **womit**. | 1 |
| 13 | Ich denke an meinen Bruder. Ich denke an ihn. | Irmão é pessoa; use **an ihn**, não **daran**. | 1 |
| 14 | Wofür interessierst du dich? | O verbo **sich interessieren für** exige **wofür**. | 1 |
| 15 | Ich bin dafür. | “A favor disso” ou “por isso” é **dafür**. | 1 |
| 16 | Woran arbeitest du? | **arbeiten an** pergunta pelo projeto/tema com **woran**. | 1 |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1 | Jonas plant ein kleines Projekt für seinen Deutschkurs. |
| 2 | Er muss daran denken, alle Materialien mitzubringen. |
| 3 | Mit dem Laptop, dem Plakat und den bunten Stiften kann die Gruppe die Idee besser erklären; sie kann damit besser erklären. |
| 4 | Er wartet auf eine E-Mail von seinem Lehrer; er wartet darauf. |
| 5 | Er braucht das große Plakat für die Präsentation; es ist dafür praktisch. |
| 6 | Porque Ben é uma pessoa; para pessoas, Jonas deve dizer **mit dir**, não **damit**. |
| 7 | daran |
| 8 | an sie |
| Prática 1 | darauf |
| Prática 2 | Wofür |
| Prática 3 | damit |
| Prática 4 | Jonas spricht mit Ben. Jonas spricht mit ihm. |
| Prática 5 | Exemplo: **Ich denke oft an meine Familie. Ich denke oft an sie.** Se for uma coisa, pode ser **Ich denke oft daran.** |
| Prática 6 | Exemplo: **Ich lerne Deutsch für meine Arbeit. Ich lerne dafür.** |
