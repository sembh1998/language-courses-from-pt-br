# Respostas: Infinitivo com zu

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | zu | Em **ruhig zu bleiben**, o **zu** vem antes do infinitivo. |
| 1.2 | zu | A estrutura é **schön, dich zu sehen**. |
| 1.3 | zu | Depois de **keine Zeit haben**, usa-se infinitivo com **zu**. |
| 1.4 | aufzustehen | **aufstehen** é separável; o **zu** entra entre prefixo e verbo. |
| 1.5 | einzukaufen | **einkaufen** vira **einzukaufen** com **zu**. |
| 1.6 | mitzukommen | **mitkommen** é separável; por isso fica **mitzukommen**. |
| 2.1 | Ich versuche, früh zu schlafen. | **zu** fica imediatamente antes de **schlafen**. |
| 2.2 | Es ist schwer, aufzustehen. | Em verbos separáveis, **zu** entra no meio: **aufzustehen**. |
| 2.3 | Ich muss heute arbeiten. | Verbos modais como **müssen** usam infinitivo sem **zu**. |
| 2.4 | Es ist wichtig, Wasser zu trinken. | Após **es ist wichtig**, usa-se infinitivo com **zu**. |
| 2.5 | Ich habe keine Zeit, heute zu kochen. | O advérbio pode vir antes de **zu kochen**, mas o **zu** fica junto do verbo. |
| 2.6 | Er vergisst, die Tür abzuschließen. | **abschließen** é separável; a forma correta é **abzuschließen**. |
| 3.1 | Ich versuche, kochen zu lernen. | O verbo principal é **versuche**; a ação desejada fica como infinitivo com **zu**. |
| 3.2 | Es ist wichtig, pünktlich zu kommen. | Avaliações com **es ist wichtig** pedem o grupo com **zu**. |
| 3.3 | Wir planen, früher loszufahren. | **losfahren** é separável; por isso **loszufahren**. |
| 3.4 | Mara vergisst nicht, ihre Mutter anzurufen. | **anrufen** é separável; o **zu** entra como **anzurufen**. |
| 3.5 | Ich habe Zeit, dir zu helfen. | Depois de **Zeit haben**, o complemento vem com **zu**. |
| 4.1 | Ich versuche, Deutsch zu lernen. | **zu** deve ficar antes de **lernen**, não antes de **Deutsch**. |
| 4.2 | Es ist gesund, viel zu schlafen. | O advérbio **viel** fica antes do grupo **zu schlafen**. |
| 4.3 | Sie hat Lust, mitzukommen. | **mitkommen** é separável; a forma com **zu** é **mitzukommen**. |
| 4.4 | Wir haben vor, morgen einzukaufen. | **einkaufen** vira **einzukaufen**, não **zu einkaufen**. |
| 4.5 | Ich muss zu Hause bleiben. | Com **müssen**, o infinitivo vem sem **zu**. |
| 5.1 | Ich versuche, pünktlich anzukommen. | **ankommen** é separável; usa-se **anzukommen**. |
| 5.2 | Es ist schwer, ohne Pause zu arbeiten. | A avaliação **es ist schwer** combina com infinitivo com **zu**. |
| 5.3 | Wir haben Lust, einen Film zu sehen. | **Lust haben** geralmente introduz uma ação com **zu**. |
| 5.4 | Sie hat vor, früher aufzustehen. | **aufstehen** recebe **zu** no meio: **aufzustehen**. |
| 5.5 | Hast du Zeit, mir zu helfen? | **Zeit haben** pede o infinitivo **zu helfen**. |
| 6.1 | Ich versuche, mehr Deutsch zu lernen. | Resposta modelo; o essencial é usar **versuchen, ... zu ...**. |
| 6.2 | Es ist wichtig, genug zu schlafen. | Resposta modelo; **es ist wichtig** exige infinitivo com **zu**. |
| 6.3 | Ich habe Lust, Freunde zu treffen. | Resposta modelo; **Lust haben** combina com **zu treffen**. |
| 6.4 | Ich habe vor, früh aufzustehen. | Resposta modelo; **vorhaben** usa uma ação com **zu**. |
