# Respostas: Verbos com preposições fixas

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | auf | `warten` exige `auf` quando significa esperar por alguém ou algo. |
| 1.2 | von | `träumen` combina com `von` + dativo. |
| 1.3 | an | `denken an` significa pensar em ou lembrar de. |
| 1.4 | über | `sprechen über` é falar sobre um tema. |
| 1.5 | auf | `sich freuen auf` indica expectativa por algo futuro. |
| 1.6 | über | `sich freuen über` indica alegria por algo recebido ou presente. |
| 2.1 | Ich warte auf den Arzt. | A combinação correta é `warten auf` + acusativo. |
| 2.2 | Mira träumt von einem Haus. | `von` pede dativo; por isso `einem Haus`. |
| 2.3 | Wir sprechen über die Reise. | Para assunto de conversa, use `sprechen über`. |
| 2.4 | Ich freue mich auf den Urlaub nächsten Monat. | As férias são futuras, então use `sich freuen auf`. |
| 2.5 | Woran denkst du? | Para coisa com `denken an`, a pergunta é `Woran?`. |
| 2.6 | Tom freut sich über das Geschenk von gestern. | O presente já foi recebido; por isso `über`. |
| 3.1 | den | `auf` com `warten` pede acusativo: `den Bus`. |
| 3.2 | einer | `von` sempre pede dativo: `einer neuen Wohnung`. |
| 3.3 | den | `über` com `sprechen` pede acusativo: `den Chef`. |
| 3.4 | deinen | `an` com `denken` pede acusativo: `deinen Reisepass`. |
| 3.5 | das | `auf` com `sich freuen` pede acusativo; neutro fica `das`. |
| 3.6 | die | `über` com `sich freuen` pede acusativo; feminino fica `die`. |
| 4.1 | Ich warte auf den Zug. | Em alemão não se usa `für` com `warten`; usa-se `auf`. |
| 4.2 | Sie träumt von einem ruhigen Leben. | `träumen von` exige `von` e dativo. |
| 4.3 | Wir sprechen über das Problem. | `über` aqui pede acusativo; `das Problem`. |
| 4.4 | Ich freue mich auf die Ferien. | `sich freuen` é reflexivo; com `ich`, use `mich`. |
| 4.5 | Er denkt an seine Eltern. | A expressão correta é `denken an`. |
| 5.1 | Ich warte auf die E-Mail. | `esperar pelo e-mail` vira `auf die E-Mail warten`. |
| 5.2 | Sie träumt von einem ruhigen Leben. | `von` pede dativo: `einem ruhigen Leben`. |
| 5.3 | Wir sprechen über das neue Projekt. | O assunto da conversa vem com `über` + acusativo. |
| 5.4 | Denkst du an die Prüfung? | `pensar na prova` é `an die Prüfung denken`. |
| 5.5 | Ich freue mich auf den Besuch morgen. | A visita é futura; use `auf`. |
| 6.1 | Ich warte auf eine Nachricht. | Resposta-modelo; mantém `warten auf` + acusativo. |
| 6.2 | Ich träume von einem interessanten Job. | Resposta-modelo; `von` + dativo. |
| 6.3 | Ich denke an meinen Pass. | Resposta-modelo; `an` + acusativo. |
| 6.4 | Ich spreche gern über Musik. | Resposta-modelo; `über` introduz o tema. |
| 6.5 | Ich freue mich auf meinen Urlaub. | Resposta-modelo; usa pronome reflexivo e `auf` para futuro. |
