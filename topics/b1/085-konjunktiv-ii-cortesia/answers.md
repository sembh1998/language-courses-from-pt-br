# Respostas: Konjunktiv II: Cortesia

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | hätte | Em pedidos com substantivo, **Ich hätte gern ...** é a fórmula natural. |
| 1.2 | Könnten | Com **Sie**, a forma formal de **können** no Konjunktiv II é **könnten**. |
| 1.3 | wäre | **Das wäre ...** expressa “isso seria ...” de modo educado. |
| 1.4 | würde | **Ich würde gern + infinitivo** expressa “eu gostaria de fazer algo”. |
| 1.5 | Könntest | Com **du**, a forma informal é **könntest**. |
| 1.6 | Hätten | **Hätten Sie ...?** pergunta formalmente se a pessoa teria algo/tempo. |
| 2.1 | Ich hätte gern ein Wasser. | Para pedir bebida, **hätte gern** soa educado; as outras opções são bruscas. |
| 2.2 | Könnten Sie mir bitte helfen? | **Könnten Sie** é formal e educado para atendimento. |
| 2.3 | Könntest du bitte das Fenster schließen? | Com amigo, usa-se **du** e a forma **könntest**. |
| 2.4 | Ich würde gern einen Termin vereinbaren. | **würde gern + infinitivo** é adequado para marcar horário por telefone. |
| 2.5 | Könnten Sie das bitte wiederholen? | A frase combina **Sie**, **könnten** e **bitte** corretamente. |
| 2.6 | Das wäre sehr freundlich. | **wäre** é a forma correta de **sein** para “seria”. |
| 3.1 | Ich hätte gern einen Kaffee. | Troca **ich will** por uma fórmula de pedido educado. |
| 3.2 | Könnten Sie mir bitte helfen? | A ordem direta vira pergunta formal com **könnten**. |
| 3.3 | Könntest du mir deine Telefonnummer geben? | Para **du**, use **könntest** e infinitivo no final. |
| 3.4 | Ich würde gern einen Termin vereinbaren. | Expressa a intenção de marcar horário de forma educada. |
| 3.5 | Würden Sie bitte hier unterschreiben? | **Würden Sie bitte + infinitivo** suaviza a instrução. |
| 3.6 | Könnten Sie das bitte noch einmal sagen? | Pedido formal de repetição com **könnten Sie**. |
| 4.1 | Ich würde gern einen Salat bestellen. | Com a ação **bestellen**, use **würde gern + infinitivo** no final. |
| 4.2 | Könnten Sie mir helfen? | **Sie** exige **könnten**, não **könntest**. |
| 4.3 | Das wäre sehr nett. | Para “isso seria”, use **wäre**, não **würde** sozinho. |
| 4.4 | Ich hätte gern einen Kaffee. | Para pedir café, **hätte gern** é mais natural que **würde gern** sem verbo. |
| 4.5 | Würden Sie bitte die Tür öffnen? | Com **würden**, o infinitivo fica no final e não se repete **Sie**. |
| 5.1 | Ich hätte gern eine Auskunft. | **Auskunft** é feminino; a expressão fixa é **Ich hätte gern ...**. |
| 5.2 | Könnten Sie das bitte wiederholen? | Pedido formal com **Sie** e **könnten**. |
| 5.3 | Könntest du mich morgen anrufen? | Informal com **du**; **anrufen** fica no infinitivo final. |
| 5.4 | Ich würde gern einen Tisch reservieren. | Para a ação de reservar, usa-se **würde gern + infinitivo**. |
| 5.5 | Das wäre sehr nett. | Fórmula comum para “isso seria muito gentil”. |
| 6.1 | Ich hätte gern zwei Brötchen. | Em padaria, **Ich hätte gern ...** é a forma padrão de pedido. |
| 6.2 | Ich würde gern einen Termin vereinbaren. | **einen Termin vereinbaren** significa marcar um horário. |
| 6.3 | Könnten Sie meine Frage beantworten? | Com professor em contexto formal, **Könnten Sie ...?** é adequado. |
| 6.4 | Könntest du bitte einen Moment warten? | Com amiga, use **du** e **könntest**. |
| 6.5 | Ich wäre Ihnen für eine Antwort dankbar. | Fórmula formal com **wäre** para expressar agradecimento antecipado. |
