# Respostas: Introdução ao Perfekt

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | habe | Com `ich`, auxiliar `haben` vira `habe`. |
| 1.2 | hast | Com `du`, auxiliar `haben` vira `hast`. |
| 1.3 | hat | Com `sie`, auxiliar `haben` vira `hat`. |
| 1.4 | haben | Com `wir`, use `haben`. |
| 1.5 | bin | `gehen` indica movimento e usa `sein`; com `ich`, `bin`. |
| 2.1 | Ich habe Deutsch gelernt. | `haben` + particípio no fim. |
| 2.2 | Sie ist früh gekommen. | `kommen` costuma usar `sein` no Perfekt. |
| 2.3 | Wir haben Musik gehört. | O particípio `gehört` fica no final. |
| 3.1 | gelernt | Particípio regular de `lernen`. |
| 3.2 | gekauft | Particípio regular de `kaufen`. |
| 3.3 | gemacht | Particípio de `machen`. |
| 3.4 | gegangen | Particípio de `gehen`. |
| 3.5 | gekommen | Particípio de `kommen`. |
| 4.1 | Ich habe Kaffee getrunken. | `trinken` usa `haben`; particípio `getrunken`. |
| 4.2 | Du hast Deutsch gelernt. | Com `du`, use `hast`. |
| 4.3 | Er ist nach Hause gegangen. | Movimento com `gehen` usa `sein`. |
| 4.4 | Wir haben Musik gehört. | `hören` usa `haben`. |
| 5.1 | Ich habe Milch gekauft. | `kaufen` usa `haben`. |
| 5.2 | Wir sind nach Berlin gefahren. | `fahren` com deslocamento usa `sein`. |
| 5.3 | Sie hat Sport gemacht. | `machen` usa `haben`. |
| 5.4 | Du hast gestern gelernt. | `du` usa `hast`; `gelernt` no fim. |
| 6.1 | getrunken | Particípio de `trinken`. |
| 6.2 | gespielt | Particípio regular de `spielen`. |
| 6.3 | gegangen | Particípio de `gehen`. |
| 7.1 | Ich habe Deutsch gelernt. | Resposta-modelo correta no Perfekt. |
| 7.2 | Ich bin nach Hause gegangen. | Resposta-modelo com `sein` e movimento. |
