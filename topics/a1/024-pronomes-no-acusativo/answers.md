# Respostas: Pronomes no acusativo

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | sie | Maria é feminino: sie. |
| 1.2 | ihn | Tom é masculino: ihn. |
| 1.3 | uns | Para nós no acusativo usa uns. |
| 1.4 | es | Buch é neutro: es. |
| 2.1 | dich | Você informal como objeto é dich. |
| 2.2 | ihn | Paul é masculino. |
| 2.3 | Sie | Forma formal no acusativo é Sie. |
| 3.1 | Ich kaufe ihn. | Stift é masculino: ihn. |
| 3.2 | Wir lesen sie. | Zeitung é feminino: sie. |
| 3.3 | Er findet es. | Handy é neutro: es. |
| 4.1 | Ich sehe dich. | Depois de sehen, objeto: dich. |
| 4.2 | Sie ruft ihn. | Ele como objeto é ihn. |
| 4.3 | Der Lehrer fragt uns. | Nós como objeto é uns. |
| 5.1 | Ich sehe dich. | te/você como objeto: dich. |
| 5.2 | Wir kennen sie. | a/ela como objeto: sie. |
| 5.3 | Sie ruft uns. | nos como objeto: uns. |
