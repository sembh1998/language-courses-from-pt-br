# Artigos indefinidos ein, eine

Nível: A1

## Objetivo

Usar **ein, eine, ein** no nominativo e distinguir artigo definido de indefinido.

## Explicação

O artigo indefinido em alemão equivale a “um/uma”. No nominativo singular, ele acompanha o gênero do substantivo.

| Gênero | Artigo definido | Artigo indefinido | Exemplo |
|---|---|---|---|
| masculino | der | ein | ein Mann |
| feminino | die | eine | eine Frau |
| neutro | das | ein | ein Kind |

Use artigo definido quando a pessoa ou coisa já é conhecida: **der Mann** = o homem. Use artigo indefinido para apresentar algo pela primeira vez: **ein Mann** = um homem.

Não há artigo indefinido plural simples como “uns/umas” em alemão: **Kinder** = crianças.

## Exemplos

| Alemão | Tradução | Observação |
|---|---|---|
| ein Mann | um homem | masculino |
| eine Frau | uma mulher | feminino |
| ein Kind | uma criança | neutro |
| Das ist ein Buch. | Isto é um livro. | neutro |
| Das ist eine Lampe. | Isto é uma lâmpada. | feminino |
| Ein Lehrer ist hier. | Um professor está aqui. | masculino |
| Der Lehrer ist nett. | O professor é simpático. | definido, conhecido |

## Erros comuns

- Usar **eine** com masculino ou neutro: correto é **ein Mann**, **ein Buch**.
- Procurar um plural de **ein**. Em A1, normalmente diga apenas **Bücher** para “livros”.
- Misturar definido e indefinido: **der** = o, **ein** = um.

## Resumo

No nominativo: **ein** para masculino e neutro, **eine** para feminino. Definido: **der/die/das**. Indefinido: **ein/eine/ein**.
