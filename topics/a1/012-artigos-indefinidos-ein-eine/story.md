# História: Artigos indefinidos ein, eine

Nível: A1

## História em alemão

Das ist ein Kursraum. Im Kursraum ist ein Tisch. Auf dem Tisch ist ein Buch. Neben dem Tisch ist eine Lampe.

Eine Frau kommt in den Raum. Sie hat eine Frage. Ein Lehrer ist auch da. Er sagt: Das ist eine gute Frage.

## Tradução linha por linha

| Alemão | Tradução |
|---|---|
| Das ist ein Kursraum. | Isto é uma sala de curso. |
| Im Kursraum ist ein Tisch. | Na sala de curso há uma mesa. |
| Auf dem Tisch ist ein Buch. | Sobre a mesa há um livro. |
| Neben dem Tisch ist eine Lampe. | Ao lado da mesa há uma lâmpada. |
| Eine Frau kommt in den Raum. | Uma mulher entra na sala. |
| Sie hat eine Frage. | Ela tem uma pergunta. |
| Ein Lehrer ist auch da. | Um professor também está lá. |
| Er sagt: Das ist eine gute Frage. | Ele diz: Essa é uma boa pergunta. |

## Notas de vocabulário

- der Kursraum = a sala de curso
- auf dem Tisch = sobre a mesa
- neben = ao lado de
- auch da = também lá/presente

## Perguntas de compreensão

1. O que há sobre a mesa?
2. Quem tem uma pergunta?
3. Qual frase usa eine com Frage?
