# Respostas: Verbos com mudança de vogal

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | liest | **lesen** muda **e → ie** com **du**. |
| 1.2 | gibt | **geben** muda **e → i** com **er**. |
| 1.3 | nimmt | **nehmen** muda para **nimmt** com **sie** singular. |
| 1.4 | isst | **essen** muda para **isst** com **das Kind** = **es**. |
| 1.5 | fährst | **fahren** muda **a → ä** com **du**. |
| 1.6 | spricht | **sprechen** muda **e → i** com **er**. |
| 2.1 | Du liest die E-Mail. | Com **du**, **lesen** fica **liest**. |
| 2.2 | Er fährt nach Köln. | Com **er**, **fahren** recebe Umlaut: **fährt**. |
| 2.3 | Sie nimmt den Kaffee. | **sie** singular usa **nimmt**, não **nehmen**. |
| 2.4 | Ich gebe dir Tee. | Com **ich**, não há mudança de vogal: **gebe**. |
| 2.5 | Das Baby schläft. | **das Baby** equivale a **es**; **schlafen** vira **schläft**. |
| 2.6 | Du sprichst langsam. | Com **du**, **sprechen** fica **sprichst**. |
| 3.1 | Du liest den Text. | Ao trocar para **du**, **lesen** muda para **liest**. |
| 3.2 | Du fährst nach Berlin. | Ao trocar para **du**, **fahren** muda para **fährst**. |
| 3.3 | Du gibst Max ein Buch. | Ao trocar para **du**, **geben** muda para **gibst**. |
| 3.4 | Du sprichst Deutsch. | Ao trocar para **du**, **sprechen** muda para **sprichst**. |
| 3.5 | Du nimmst den Apfel. | Ao trocar para **du**, **nehmen** muda para **nimmst**. |
| 4.1 | e → i | **geben** vira **gibst** com **du**. |
| 4.2 | e → ie | **lesen** vira **liest** com **du**. |
| 4.3 | a → ä | **fahren** vira **fährst** com **du**. |
| 4.4 | e → ie | **sehen** vira **sieht** com **er**. |
| 4.5 | a → ä | **schlafen** vira **schläft** com **sie** singular. |
| 4.6 | e → i | **sprechen** vira **sprichst** com **du**. |
| 5.1 | esse | Com **ich**, **essen** fica **esse**, sem mudança de vogal especial. |
| 5.2 | fahren | Com **wir**, usa-se a forma igual ao infinitivo. |
| 5.3 | lest | Com **ihr**, **lesen** fica **lest**, sem **ie**. |
| 5.4 | sprechen | **Sie** plural usa **sprechen**, sem mudança de vogal. |
| 5.5 | trägt | Com **er**, **tragen** muda **a → ä**. |
| 5.6 | siehst | Com **du**, **sehen** muda **e → ie**: **siehst**. |
| 6.1 | Du liest ein Buch. | “Você” corresponde a **du**; **lesen** vira **liest**. |
| 6.2 | Er nimmt den Bus. | “Ele” corresponde a **er**; **nehmen** vira **nimmt**. |
| 6.3 | Sie schläft. | “Ela” corresponde a **sie** singular; **schlafen** vira **schläft**. |
| 6.4 | Du sprichst Deutsch. | “Você fala” é **du sprichst**. |
| 6.5 | Er fährt nach Hause. | “Ele vai de veículo/para casa” usa **er fährt nach Hause**. |
| 6.6 | Sie sieht den Lehrer. | “Ela vê” é **sie sieht**; **der Lehrer** no acusativo fica **den Lehrer**. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. O que Mia lê em casa? | Ela lê uma mensagem. A frase alemã é **Sie liest eine Nachricht**. |
| 2. Quem come pão e toma chá? | Tom come pão e toma chá: **Tom isst Brot und trinkt Tee**. |
| 3. Complete em alemão: Tom, du ___ sehr laut! | **sprichst**. Com **du**, **sprechen** vira **sprichst**. |
| 4. Como Mia vai para a escola? | Ela vai de ônibus: **mit dem Bus**. |
| 5. O que Tom pega de manhã? | Ele pega seu livro: **Tom nimmt sein Buch**. |
| 6. Complete em alemão: Der Hund ___. | **schläft**. **Der Hund** equivale a **er**. |
| 7. Quem lê um texto na escola? | O professor lê um texto: **der Lehrer liest einen Text**. |
| 8. Complete com a forma correta: Tom ___ dem Lehrer das Buch. | **gibt**. Tom equivale a **er**; **geben** vira **gibt**. |
| Prática 1 | **Du liest eine Nachricht.** Com **du**, **lesen** fica **liest**. |
| Prática 2 | **Er fährt zur Schule.** Com **er**, **fahren** fica **fährt**. |
| Prática 3 | **Sie sieht den Hund.** Com **sie** singular, **sehen** fica **sieht**. |
| Prática 4 | **nimmt**. Com Tom = **er**, **nehmen** fica **nimmt**. |
| Prática 5 | **liest**. **Der Lehrer** equivale a **er**; **lesen** fica **liest**. |
| Prática 6 | **spricht**. Mia equivale a **sie** singular; **sprechen** fica **spricht**. |
