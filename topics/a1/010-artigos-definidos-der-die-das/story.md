# História: Artigos definidos der, die, das

Nível: A1

## História em alemão

Der Lehrer ist im Kurs. Die Frau heißt Anna. Das Kind heißt Leo. Der Tisch ist klein. Die Lampe ist neu. Das Buch ist gut.

Die Schüler lernen Deutsch. Die Bücher sind auf dem Tisch. Anna sagt: Der Kurs ist interessant. Leo sagt: Das Wort ist neu.

## Tradução linha por linha

| Alemão | Tradução |
|---|---|
| Der Lehrer ist im Kurs. | O professor está no curso. |
| Die Frau heißt Anna. | A mulher se chama Anna. |
| Das Kind heißt Leo. | A criança se chama Leo. |
| Der Tisch ist klein. | A mesa é pequena. |
| Die Lampe ist neu. | A lâmpada é nova. |
| Das Buch ist gut. | O livro é bom. |
| Die Schüler lernen Deutsch. | Os alunos aprendem alemão. |
| Die Bücher sind auf dem Tisch. | Os livros estão sobre a mesa. |
| Anna sagt: Der Kurs ist interessant. | Anna diz: O curso é interessante. |
| Leo sagt: Das Wort ist neu. | Leo diz: A palavra é nova. |

## Notas de vocabulário

- der Kurs = o curso
- das Wort = a palavra
- die Schüler = os alunos
- neu = novo

## Perguntas de compreensão

1. Qual artigo aparece com Lehrer?
2. Qual artigo aparece com Lampe?
3. Qual artigo é usado no plural die Bücher?
