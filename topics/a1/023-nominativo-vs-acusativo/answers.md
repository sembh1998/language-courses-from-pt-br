# Respostas: Nominativo vs Acusativo

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | nominativo | É quem faz a ação, portanto está no nominativo. |
| 1.2 | acusativo | É o objeto direto comprado; masculino no acusativo usa den. |
| 1.3 | nominativo | É o sujeito da frase. |
| 1.4 | acusativo | Recebe a ação de brauchen. |
| 2.1 | Der | Mann é sujeito, então nominativo: der. |
| 2.2 | den | Mann é objeto direto masculino, então acusativo: den. |
| 2.3 | Der | Hund faz a ação; nominativo. |
| 2.4 | den | Arzt recebe a ação de chamar; acusativo. |
| 3.1 | Der Mann sieht die Frau. | O sujeito masculino é der Mann; Frau não muda no acusativo. |
| 3.2 | Ich sehe den Schüler. | Schüler é masculino e objeto direto: den. |
| 3.3 | Die Frau sieht den Lehrer. | Frau é sujeito; Lehrer é objeto masculino: den. |
| 4.1 | Der Junge sieht den Hund. | Hund é objeto masculino, então der vira den. |
| 4.2 | Der Vater kocht. | Vater é sujeito; precisa de der. |
| 4.3 | Ich finde den Schlüssel. | Schlüssel é objeto masculino, então den. |
| 5.1 | Der Freund sieht das Auto. | Freund é sujeito masculino; Auto é neutro e não muda. |
| 5.2 | Ich kaufe das Brot. | Brot é neutro; no acusativo continua das. |
| 5.3 | Die Frau ruft den Arzt. | Arzt é masculino e objeto direto. |
| 6.1 | Der Junge sieht den Mann. | Ao trocar papéis, o novo sujeito fica der e o objeto masculino fica den. |
| 6.2 | Der Schüler hört die Lehrerin. | Schüler vira sujeito masculino: der; Lehrerin como objeto feminino continua die. |
