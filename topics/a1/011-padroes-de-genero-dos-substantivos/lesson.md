# Padrões de gênero dos substantivos

Nível: A1

## Objetivo

Reconhecer padrões frequentes que ajudam a prever **der, die, das** em substantivos alemães.

## Explicação

O gênero alemão precisa ser memorizado com cada substantivo, mas alguns padrões ajudam muito. Estes padrões não explicam tudo, porém são úteis em A1.

| Padrão | Artigo comum | Exemplos |
|---|---|---|
| dias da semana | der | der Montag, der Freitag |
| meses | der | der Januar, der Mai |
| estações | der | der Sommer, der Winter |
| final -ung | die | die Wohnung, die Zeitung |
| final -heit | die | die Freiheit, die Krankheit |
| final -keit | die | die Möglichkeit, die Pünktlichkeit |
| final -chen | das | das Mädchen, das Brötchen |
| final -lein | das | das Büchlein |
| infinitivo como substantivo | das | das Lernen, das Essen |

Use esses padrões como ajuda, mas confirme palavras novas em dicionário ou material confiável.

## Exemplos

| Alemão | Tradução | Observação |
|---|---|---|
| der Montag | a segunda-feira | dia = der |
| der Januar | janeiro | mês = der |
| der Sommer | o verão | estação = der |
| die Wohnung | o apartamento | final -ung = die |
| die Krankheit | a doença | final -heit = die |
| die Möglichkeit | a possibilidade | final -keit = die |
| das Mädchen | a menina | final -chen = das |
| das Lernen | o aprendizado | infinitivo substantivado = das |

## Erros comuns

- Achar que o gênero segue sempre o português. **die Wohnung** é “o apartamento”.
- Esquecer que **-chen** é neutro: **das Mädchen**.
- Tratar padrões como regra absoluta para todas as palavras. Eles ajudam, mas não substituem memorizar.

## Resumo

Dias, meses e estações geralmente usam **der**. Finais **-ung, -heit, -keit** geralmente usam **die**. Finais **-chen, -lein** e infinitivos substantivados usam **das**.
