# Acusativo: objeto direto

Nível: A1

## Objetivo

Entender quando usar o acusativo para o objeto direto e praticar a mudança mais importante no A1: **der → den** e **ein → einen** para substantivos masculinos.

## Explicação

Em alemão, o **acusativo** aparece com frequência no **objeto direto**, isto é, aquilo ou a pessoa que recebe a ação do verbo.

Pense em duas perguntas simples:

- Quem faz a ação? Normalmente é o sujeito.
- Quem ou o que recebe a ação? Normalmente é o objeto direto no acusativo.

Exemplo:

**Ich sehe den Mann.**

- **Ich** faz a ação.
- **den Mann** recebe a ação: eu vejo **o homem**.

No A1, a mudança principal é no **masculino singular**:

| Nominativo | Acusativo | Exemplo |
|---|---|---|
| der Mann | den Mann | Ich sehe den Mann. |
| ein Hund | einen Hund | Sie hat einen Hund. |

Feminino, neutro e plural não mudam no artigo definido/indefinido básico:

| Gênero/número | Nominativo | Acusativo | Exemplo |
|---|---|---|---|
| Masculino | der / ein | den / einen | Ich kaufe einen Kaffee. |
| Feminino | die / eine | die / eine | Ich kaufe eine Tasche. |
| Neutro | das / ein | das / ein | Ich lese ein Buch. |
| Plural | die | die | Ich sehe die Kinder. |

Alguns verbos comuns que frequentemente têm objeto direto:

- **sehen** = ver
- **haben** = ter
- **kaufen** = comprar
- **lesen** = ler
- **essen** = comer
- **trinken** = beber
- **machen** = fazer

## Exemplos

| Alemão | Tradução | Observação |
|---|---|---|
| Ich sehe den Lehrer. | Eu vejo o professor. | Masculino: der Lehrer → den Lehrer. |
| Sie kauft einen Apfel. | Ela compra uma maçã. | Masculino: ein Apfel → einen Apfel. |
| Er liest das Buch. | Ele lê o livro. | Neutro não muda: das. |
| Wir haben eine Tasche. | Nós temos uma bolsa. | Feminino não muda: eine. |
| Du trinkst den Kaffee. | Você bebe o café. | Masculino: der Kaffee → den Kaffee. |
| Ich esse einen Kuchen. | Eu como um bolo. | Masculino: ein Kuchen → einen Kuchen. |
| Maria sieht die Kinder. | Maria vê as crianças. | Plural com die. |
| Tom macht die Aufgabe. | Tom faz a tarefa. | Feminino com die. |

## Erros comuns

- Usar **der** para objeto direto masculino: **Ich sehe der Mann** está errado. O correto é **Ich sehe den Mann**.
- Usar **ein** para objeto direto masculino: **Sie kauft ein Apfel** está errado. O correto é **Sie kauft einen Apfel**.
- Mudar todos os artigos no acusativo. No básico do A1, a mudança visível é só no masculino singular: **die**, **das** e plural **die** ficam iguais.
- Tentar traduzir palavra por palavra do português. Em português dizemos “vejo o homem”; em alemão, o caso do artigo mostra a função: **den Mann**.

## Prática guiada

Use este caminho mental:

1. Identifique o verbo.
2. Pergunte: quem ou o que recebe a ação?
3. Veja o gênero do substantivo.
4. Se for masculino singular, use **den** ou **einen**.

| Passo | Frase | Decisão |
|---|---|---|
| 1 | Ich kaufe ___ Stift. | Comprar o quê? **der Stift** é masculino. |
| 2 | Ich kaufe den Stift. | Com artigo definido: der → den. |
| 3 | Ich kaufe einen Stift. | Com artigo indefinido: ein → einen. |

Mini-checagem:

- **Ich habe ___ Hund.** Como **der Hund** é masculino e é objeto de **haben**, a resposta é **einen**.
- **Ich lese ___ Buch.** Como **das Buch** é neutro, a resposta é **ein** ou **das**, conforme o sentido.
- **Ich sehe ___ Frau.** Como **die Frau** é feminino, a resposta pode ser **die** ou **eine**, sem mudança especial.

## Resumo

O acusativo marca o objeto direto. Para A1, memorize a regra central: no masculino singular, **der vira den** e **ein vira einen** quando o substantivo recebe a ação. Feminino, neutro e plural mantêm os artigos básicos iguais.
