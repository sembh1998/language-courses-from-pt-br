## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | kann | Com ich: kann. |
| 1.2 | können | Com wir: können. |

| 1.4 | können | Com sie plural: können. |

| 2.2 | du kannst | A forma de du é kannst. |
| 2.3 | Wir können kochen. | Wir usa können e infinitivo final. |

| 3.2 | Du kannst kommen. | Com du: kannst. |
| 3.3 | Wir können dir helfen. | O infinitivo helfen fica no final. |
| 4.1 | Ich kann Deutsch lesen. | Modal kann + infinitivo final. |
| 4.2 | Ihr könnt heute kommen. | Com ihr: könnt. |

| 5.3 | Er kann gut kochen. | Capacidade: kann + kochen. |
