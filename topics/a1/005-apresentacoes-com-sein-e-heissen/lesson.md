# Apresentações com sein e heißen

Nível: A1

## Objetivo

Aprender a se apresentar, perguntar o nome de outra pessoa e usar formas básicas de `sein` e `heißen` em pequenos diálogos.

## Explicação

Em alemão, duas estruturas aparecem o tempo todo quando você conhece alguém: `Ich heiße...` e `Ich bin...`.

Use `heißen` quando quiser dizer ou perguntar o nome:

- **Ich heiße Ana.** = Eu me chamo Ana. / Meu nome é Ana.
- **Wie heißt du?** = Como você se chama?

Use `sein` quando quiser dizer quem a pessoa é, como ela é, ou identificar alguém:

- **Ich bin Ana.** = Eu sou Ana.
- **Du bist Paul.** = Você é Paul.
- **Er ist Lukas.** = Ele é Lukas.
- **Sie ist Lena.** = Ela é Lena.

Para A1, você precisa dominar principalmente estas formas:

| Pronome | sein | heißen |
|---|---|---|
| ich | bin | heiße |
| du | bist | heißt |
| er/sie | ist | heißt |
| Sie | sind | heißen |

`du` é informal: use com amigos, colegas, crianças e pessoas em contexto casual. `Sie` é formal: use com desconhecidos, clientes, professores ou em situações mais formais.

Uma diferença importante para falantes de português: em alemão, não diga `Ich bin heiße Ana`. Escolha uma estrutura: `Ich heiße Ana` ou `Ich bin Ana`.

## Exemplos

| Alemão | Tradução | Observação |
|---|---|---|
| Ich heiße Ana. | Eu me chamo Ana. / Meu nome é Ana. | Forma principal para dizer o nome. |
| Ich bin Ana. | Eu sou Ana. | Também serve para se apresentar. |
| Wie heißt du? | Como você se chama? | Pergunta informal sobre nome. |
| Ich heiße Bruno. Und du? | Eu me chamo Bruno. E você? | Resposta com pergunta de volta. |
| Wer bist du? | Quem é você? | Pergunta informal sobre identidade. |
| Du bist Clara. | Você é Clara. | `du` + `bist`. |
| Er heißt Lukas. | Ele se chama Lukas. | `er` + `heißt`. |
| Sie heißt Lena. | Ela se chama Lena. | `sie` + `heißt`. |
| Er ist mein Freund. | Ele é meu amigo. | Identidade/relação com `sein`. |
| Sie ist meine Freundin. | Ela é minha amiga. | Identidade/relação com `sein`. |
| Das ist Maria. | Esta é a Maria. | Apresentando outra pessoa. |
| Wie heißen Sie? | Como o senhor/a senhora se chama? | Pergunta formal sobre nome. |
| Wer sind Sie? | Quem é o senhor/a senhora? | Pergunta formal sobre identidade. |
| Ich bin aus Brasilien. | Eu sou do Brasil. | Origem simples com `sein`. |

## Erros comuns

- Dizer **Ich bin heiße Ana**. O correto é **Ich heiße Ana** ou **Ich bin Ana**.
- Usar **Wie bist du?** para perguntar o nome. Para nome, use **Wie heißt du?**.
- Esquecer o `t` em **du heißt**. A forma correta é **du heißt**, não **du heiße**.
- Trocar **du bist** por **du ist**. Com `du`, use **bist**.
- Trocar **er/sie ist** por **er/sie bist**. Com `er` e `sie`, use **ist**.
- Confundir **sie** e **Sie**: **sie ist** = ela é; **Sie sind** = o senhor/a senhora é.
- Usar pergunta formal com resposta informal sem perceber: **Wie heißen Sie?** combina com **Ich heiße...**, mas o tratamento é formal.

## Resumo

Use **Ich heiße...** para dizer seu nome e **Wie heißt du?** para perguntar o nome informalmente. Use **Ich bin...**, **du bist...** e **er/sie ist...** para dizer quem alguém é. Em situação formal, use **Wie heißen Sie?** e **Wer sind Sie?**. Para A1, memorize as combinações como blocos: **ich bin**, **du bist**, **er/sie ist**, **ich heiße**, **du heißt**, **er/sie heißt**.
