# Respostas: Nominativo: sujeito da frase

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Der | `Mann` é masculino e sujeito; no nominativo usa `der`. |
| 1.2 | Die | `Frau` é feminino e sujeito; no nominativo usa `die`. |
| 1.3 | Das | `Kind` é neutro e sujeito; no nominativo usa `das`. |
| 1.4 | Die | Plural definido no nominativo usa `die`. |
| 1.5 | Der | `Hund` é masculino e faz a ação de dormir. |
| 1.6 | Das | `Haus` é neutro e aparece como sujeito. |
| 2.1 | Der Mann arbeitet. | Sujeito masculino no nominativo pede `der`, não `den` ou `dem`. |
| 2.2 | Die Frau liest. | `Frau` é feminino; o sujeito no nominativo usa `die`. |
| 2.3 | Das Kind malt. | `Kind` é neutro; o artigo correto no nominativo é `das`. |
| 2.4 | Die Autos stehen dort. | Plural no nominativo usa `die`. |
| 2.5 | Ein Hund bellt. | Com artigo indefinido masculino como sujeito, usa-se `ein`. |
| 2.6 | Eine Lehrerin spricht. | `Lehrerin` é feminino; no nominativo indefinido usa `eine`. |
| 3.1 | Die Katze | É quem faz a ação de dormir. |
| 3.2 | Wir | O pronome `wir` é o sujeito da frase. |
| 3.3 | Das Auto | É o elemento descrito como novo. |
| 3.4 | Ein Lehrer | É quem vem hoje; portanto é o sujeito. |
| 3.5 | Sie | O pronome `sie` está antes do verbo e é o sujeito. |
| 3.6 | Der Tisch | É o elemento descrito como grande. |
| 4.1 | Ein | `Mann` é masculino; no nominativo indefinido usa `ein`. |
| 4.2 | Eine | `Frau` é feminino; no nominativo indefinido usa `eine`. |
| 4.3 | Ein | `Kind` é neutro; no nominativo indefinido usa `ein`. |
| 4.4 | Sie | Maria é feminino singular; o pronome sujeito é `sie`. |
| 4.5 | Er | Paul é masculino singular; o pronome sujeito é `er`. |
| 4.6 | Sie | Ana e Leo formam plural; o pronome sujeito é `sie`. |
| 5.1 | Er kommt. | `der Mann` pode ser substituído por `er`. |
| 5.2 | Sie lernt. | `die Frau` pode ser substituída por `sie`. |
| 5.3 | Es spielt. | `das Kind` é neutro; o pronome é `es`. |
| 5.4 | Sie singen. | `die Kinder` é plural; o pronome é `sie`. |
| 5.5 | Wir wohnen hier. | `Lukas und ich` inclui “eu”; o pronome correto é `wir`. |
| 6.1 | Der Mann kommt. | `Mann` é masculino e sujeito; use `der`. |
| 6.2 | Die Frau lernt Deutsch. | `Frau` é feminino e sujeito; use `die`. |
| 6.3 | Das Kind spielt. | `Kind` é neutro e sujeito; use `das`. |
| 6.4 | Ein Hund schläft. | `Hund` é masculino; com “um”, use `ein`. |
| 6.5 | Eine Lehrerin ist hier. | `Lehrerin` é feminino; com “uma”, use `eine`. |
| 6.6 | Die Kinder singen. | Plural definido no nominativo usa `die`. |
| 7.1 | Der Mann kommt. | `den` não é forma de sujeito masculino; o nominativo é `der`. |
| 7.2 | Die Frau lernt. | `Frau` precisa de `die` no nominativo. |
| 7.3 | Das Kind spielt. | `Kind` é neutro; use `das`. |
| 7.4 | Ein Lehrer wohnt hier. | `Lehrer` masculino no nominativo indefinido usa `ein`. |
| 7.5 | Eine Katze schläft. | `Katze` é feminino; no indefinido nominativo usa `eine`. |
