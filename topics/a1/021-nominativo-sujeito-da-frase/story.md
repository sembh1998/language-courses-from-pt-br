# História: Nominativo: sujeito da frase

Nível: A1

## História em alemão

### Cena 1

Der Mann kommt in den Park.
Die Frau sitzt auf der Bank.
Das Kind spielt mit einem Ball.
Der Hund läuft schnell.
Die Katze schläft im Garten.

### Cena 2

Ein Lehrer kommt auch.
Eine Lehrerin sagt hallo.
Ein Mädchen malt ein Haus.
Die Kinder lachen.
Wir lernen Deutsch im Park.

### Cena 3

Ich bin Ana.
Du bist Tom.
Er ist Lehrer.
Sie ist Lehrerin.
Es ist ein kleines Kind.
Sie sind Freunde.

## Tradução linha por linha

| Alemão | Tradução |
|---|---|
| Der Mann kommt in den Park. | O homem vem ao parque. |
| Die Frau sitzt auf der Bank. | A mulher está sentada no banco. |
| Das Kind spielt mit einem Ball. | A criança brinca com uma bola. |
| Der Hund läuft schnell. | O cachorro corre rápido. |
| Die Katze schläft im Garten. | O gato dorme no jardim. |
| Ein Lehrer kommt auch. | Um professor também vem. |
| Eine Lehrerin sagt hallo. | Uma professora diz olá. |
| Ein Mädchen malt ein Haus. | Uma menina pinta uma casa. |
| Die Kinder lachen. | As crianças riem. |
| Wir lernen Deutsch im Park. | Nós aprendemos alemão no parque. |
| Ich bin Ana. | Eu sou Ana. |
| Du bist Tom. | Você é Tom. |
| Er ist Lehrer. | Ele é professor. |
| Sie ist Lehrerin. | Ela é professora. |
| Es ist ein kleines Kind. | É uma criança pequena. |
| Sie sind Freunde. | Eles/elas são amigos(as). |

## Notas de vocabulário

- `der Mann` = o homem; masculino no nominativo
- `die Frau` = a mulher; feminino no nominativo
- `das Kind` = a criança; neutro no nominativo
- `der Hund` = o cachorro; masculino no nominativo
- `die Katze` = o gato/a gata; palavra feminina em alemão
- `ein Lehrer` = um professor; masculino no nominativo
- `eine Lehrerin` = uma professora; feminino no nominativo
- `die Kinder` = as crianças; plural no nominativo
- `wir` = nós; pronome sujeito
- `sie` = ela ou eles/elas; veja o verbo para entender

## Perguntas de compreensão

1. Quem vem ao parque na primeira frase?
2. Qual é o sujeito em `Die Frau sitzt auf der Bank`?
3. Complete: `___ Kind spielt mit einem Ball.`
4. Qual frase tem sujeito plural: `Der Hund läuft schnell` ou `Die Kinder lachen`?
5. Em `Wir lernen Deutsch im Park`, qual é o sujeito?
6. `Er ist Lehrer` significa que o sujeito é masculino ou feminino?
7. Complete com artigo: `___ Lehrerin sagt hallo.`
8. Reescreva com pronome: `Die Katze schläft.`

## Prática com a história

1. Circule mentalmente o sujeito: `Ein Mädchen malt ein Haus.`
2. Complete: `___ Mann kommt in den Park.`
3. Complete: `___ Kinder lachen.`
4. Troque por pronome: `Der Hund läuft schnell.`
5. Troque por pronome: `Das Kind spielt.`
6. Traduza: “Nós aprendemos alemão.”
