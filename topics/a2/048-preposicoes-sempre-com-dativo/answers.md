# Respostas: Preposições sempre com dativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | dem | **mit** sempre pede dativo; **der Zug** vira **dem Zug**. |
| 1.2 | der | **bei** pede dativo; no feminino, **die Tante** vira **der Tante**. |
| 1.3 | den | No plural dativo, o artigo é **den**; **Kinder** já tem plural próprio. |
| 1.4 | meinem | **von** pede dativo; masculino com possessivo fica **meinem Bruder**. |
| 1.5 | der | **nach** no sentido de “depois de” pede dativo; **die Arbeit** vira **der Arbeit**. |
| 2.1 | Ich fahre mit dem Auto. | **mit** exige dativo; **das Auto** vira **dem Auto**. |
| 2.2 | Sie lernt seit einem Monat Deutsch. | **seit** pede dativo; **ein Monat** vira **einem Monat**. |
| 2.3 | Die Post ist gegenüber dem Bahnhof. | **gegenüber** pede dativo; **der Bahnhof** vira **dem Bahnhof**. |
| 2.4 | Er ist bei seinem Vater. | **bei** pede dativo; masculino com possessivo fica **seinem Vater**. |
| 2.5 | Wir kommen aus der Stadt. | **aus** pede dativo; **die Stadt** vira **der Stadt**. |
| 2.6 | Maria geht zum Arzt. | **zu** pede dativo, e **zu dem Arzt** contrai para **zum Arzt**. |
| 3.1 | nach | A expressão fixa é **nach Hause**, “para casa”. |
| 3.2 | von | **von** indica origem/autoria: a carta é “da” irmã. |
| 3.3 | mit | **mit** é usado para meio de transporte e pede dativo. |
| 3.4 | aus | **aus** indica origem: vir de Portugal. |
| 3.5 | bei | **bei** pode indicar estar na casa de alguém. |
| 3.6 | seit | **seit** expressa duração até agora: há três meses. |
| 4.1 | zum Supermarkt | **zu + dem** contrai para **zum**. |
| 4.2 | zur Schule | **zu + der** contrai para **zur**. |
| 4.3 | beim Arzt | **bei + dem** contrai para **beim**. |
| 4.4 | vom Lehrer | **von + dem** contrai para **vom**. |
| 4.5 | zu den Eltern | Plural dativo usa **den**; não há contração comum aqui. |
| 5.1 | Ich fahre mit der U-Bahn. | **mit** pede dativo; **die U-Bahn** vira **der U-Bahn**. |
| 5.2 | Sie geht zur Apotheke. | **zu der Apotheke** contrai para **zur Apotheke**. |
| 5.3 | Wir sind bei den Großeltern. | **bei** pede dativo; plural usa **den**. |
| 5.4 | Er kommt aus der Schule. | **aus** indica saída/origem de dentro de um lugar e pede dativo. |
| 5.5 | Das Café ist gegenüber dem Hotel. | **gegenüber** pede dativo; **das Hotel** vira **dem Hotel**. |
| 6.1 | Ich fahre mit dem Bus. | **der Bus** vira **dem Bus** depois de **mit**. |
| 6.2 | Sie wohnt seit einem Jahr in Hamburg. | **seit** pede dativo; **ein Jahr** vira **einem Jahr**. |
| 6.3 | Wir gehen zur Schule. | **zu der Schule** normalmente contrai para **zur Schule**. |
| 6.4 | Das Buch ist von der Lehrerin. | **von** pede dativo; feminino fica **der Lehrerin**. |
| 6.5 | Der Laden ist gegenüber dem Park. | **gegenüber** pede dativo; **der Park** vira **dem Park**. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1 | Lara wohnt seit einem Jahr in Köln. |
| 2 | Ela vai de bonde: **mit der Straßenbahn**. |
| 3 | O ponto fica em frente à padaria: **gegenüber dem Bäcker**. |
| 4 | A mensagem vem do chefe dela: **von ihrem Chef**. |
| 5 | Ela vai até sua amiga Nina: **zu ihrer Freundin Nina**. |
| 6 | Nina aprende alemão há seis meses: **seit sechs Monaten**. |
| 7 | **bei**; a frase é **Die beiden sitzen bei Nina in der Küche**. |
| 8 | **der**; a frase é **Der Park liegt gegenüber der Schule**. |
| Prática 1 | **Lara geht zur Arbeit.** **zu der** contrai para **zur**. |
| Prática 2 | **dem**; **nach dem Frühstück** usa dativo. |
| Prática 3 | **ihrem**; **von** pede dativo com possessivo masculino. |
| Prática 4 | **aus**; **aus Brasilien** indica origem. |
| Prática 5 | Exemplo: **Ich gehe mit dem Hund spazieren.** Usa **mit** + dativo. |
| Prática 6 | Exemplo: **Ich lerne seit einem Monat Deutsch.** Usa **seit** + dativo. |
