# Respostas: Textos práticos simples

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Liebe Lisa | **Liebe** é uma saudação informal adequada para uma amiga. |
| 1.2 | Hast du am Samstag Zeit? | A frase pergunta corretamente se a pessoa tem tempo no sábado. |
| 1.3 | Viele Grüße | **Viele Grüße** é uma despedida informal comum. |
| 1.4 | Die Party beginnt um 19 Uhr. | Para horário exato, usa-se **um**; o verbo concorda com **die Party**. |
| 2.1 | lade | O verbo separável é **einladen**: **ich lade ... ein**. |
| 2.2 | am | Para dias da semana, usa-se **am**. |
| 2.3 | antworten | **antworten** completa o pedido de resposta. |
| 2.4 | Viele | A despedida fixa é **Viele Grüße**. |
| 2.5 | Zuerst | **Zuerst** organiza a primeira ação de uma rotina. |
| 3.1 | Ich lade dich ins Kino ein. | O verbo separável fica com **lade** na posição verbal e **ein** no final. |
| 3.2 | Der Kurs beginnt um 8 Uhr. | A frase informa horário com **um**. |
| 3.3 | Ich komme heute später. | Mensagem curta e direta para avisar atraso. |
| 3.4 | Bitte antworte mir. | Forma simples e educada de pedir resposta. |
| 4.1 | Die Party ist um 18 Uhr bei mir. | A ordem coloca sujeito, verbo e informações práticas de horário/lugar. |
| 4.2 | Am Sonntag habe ich meine Oma besucht. | Relato passado usa Perfekt: **habe ... besucht**. |
| 4.3 | Dann fahre ich zur Arbeit. | Com **dann** no início, o verbo fica em segunda posição. |
| 4.4 | Kannst du am Freitag kommen? | Pergunta com modal: **Kannst du ... kommen?** |
| 5.1 | Lieber Tom, | **Lieber** é usado para um destinatário masculino em saudação informal. |
| 5.2 | Ich lerne jeden Tag Deutsch. | Rotina frequente é expressa no presente. |
| 5.3 | Am Wochenende habe ich Freunde getroffen. | Ação concluída no passado usa Perfekt. |
| 5.4 | Möchtest du mitkommen? | Pergunta curta e natural para convidar alguém a ir junto. |
| 6.1 | Hallo, möchtest du am Sonntag um 15 Uhr Kaffee trinken? Bitte antworte mir. Viele Grüße | O texto inclui convite, dia, horário, pedido de resposta e despedida. |
| 6.2 | Ich stehe um 7 Uhr auf. Dann frühstücke ich. Danach fahre ich zur Arbeit. | As frases descrevem rotina com presente e conectores simples. |
| 6.3 | Am Samstag habe ich lange geschlafen. Dann habe ich eingekauft. Am Sonntag habe ich meine Familie besucht. | O relato usa Perfekt para ações concluídas no fim de semana. |
