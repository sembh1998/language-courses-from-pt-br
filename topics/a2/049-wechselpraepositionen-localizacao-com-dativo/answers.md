# Respostas: Wechselpräpositionen: Localização com dativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | dem | **der Tisch** vira **dem Tisch** no dativo após **wo?**. |
| 1.2 | der | **die Schule** vira **der Schule** no dativo. |
| 1.3 | der | **die Wand** vira **der Wand** no dativo. |
| 1.4 | dem | **das Bett** vira **dem Bett** no dativo. |
| 1.5 | dem | **der Stuhl** vira **dem Stuhl** no dativo. |
| 1.6 | den | No plural dativo, usa-se **den** e muitas vezes o substantivo recebe **-n**. |
| 2.1 | Das Heft liegt auf dem Tisch. | A pergunta **wo?** pede localização com dativo: **dem Tisch**. |
| 2.2 | Maria ist in der Küche. | Para estar em um lugar, usamos **in + dativo**: **der Küche**. |
| 2.3 | Die Uhr hängt an der Wand. | Localização na parede usa **an + dativo**: **an der Wand**. |
| 2.4 | Der Rucksack steht neben dem Sofa. | **das Sofa** vira **dem Sofa** no dativo. |
| 2.5 | Die Schuhe sind unter dem Bett. | **unter** com localização usa dativo: **dem Bett**. |
| 2.6 | Paul wartet vor der Tür. | **die Tür** vira **der Tür** no dativo. |
| 3.1 | im Zimmer | **im** é a contração comum de **in dem**. |
| 3.2 | am Fenster | **am** é a contração comum de **an dem**. |
| 3.3 | im Büro | **in dem Büro** normalmente vira **im Büro**. |
| 3.4 | am Tisch | **an dem Tisch** pode ser contraído para **am Tisch**. |
| 4.1 | na escola | **in der Schule** indica localização dentro/na escola. |
| 4.2 | sobre a mesa | **auf dem Tisch** é localização sobre uma superfície. |
| 4.3 | na parede | **an der Wand** indica algo junto ou pendurado na parede. |
| 4.4 | atrás da casa | **hinter dem Haus** indica posição atrás de algo. |
| 4.5 | entre as cadeiras | **zwischen den Stühlen** usa dativo plural. |
| 5.1 | an | Quadros ficam pendurados **an der Wand**. |
| 5.2 | auf | Uma garrafa em cima da mesa fica **auf dem Tisch**. |
| 5.3 | unter | Dormir embaixo da cama/cadeira usa **unter**. |
| 5.4 | hinter | Um jardim atrás da casa fica **hinter dem Haus**. |
| 5.5 | zwischen | Para posição entre dois elementos, use **zwischen**. |
| 5.6 | vor | Esperar em frente à porta é **vor der Tür warten**. |
| 6.1 | Das Handy liegt auf dem Tisch. | **auf dem Tisch** expressa localização sobre a mesa. |
| 6.2 | Sie ist in der Schule. | **die Schule** muda para **der Schule** no dativo. |
| 6.3 | Der Rucksack steht neben dem Bett. | **das Bett** vira **dem Bett** com **neben** em localização. |
| 6.4 | Das Bild hängt an der Wand. | Objetos na parede normalmente usam **hängen an + dativo**. |
| 6.5 | Der Schlüssel liegt zwischen den Büchern. | Entre livros exige **zwischen den Büchern** no dativo plural. |
| 7.1 | Es ist auf dem Tisch. | A resposta indica onde o livro está, então usa dativo. |
| 7.2 | Sie ist unter dem Stuhl. | **die Tasche** é substituída por **sie**; o lugar usa **unter dem Stuhl**. |
| 7.3 | Er hängt an der Wand. | **der Kalender** é **er**; na parede usa **an der Wand**. |
| 7.4 | Er ist im Büro. | **im Büro** é a forma comum de **in dem Büro**. |

| Wo stehen ihre Schuhe? | Sie stehen vor dem Schrank. |
| Wo sitzt Mia in der Schule? | Sie sitzt neben ihrer Freundin Lara. |

| Prática 4 | im Kühlschrank; **im** = **in dem**. |
| Prática 5 | Der Zettel liegt neben der Tasse. |
| Prática 6 | Die Uhr hängt an der Wand. |
