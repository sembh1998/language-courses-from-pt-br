# Respostas: Declinação após artigo definido

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | freundliche | Com `der` no nominativo masculino, o adjetivo recebe `-e`. |
| 1.2 | kleine | Com `die` no feminino nominativo, o adjetivo recebe `-e`. |
| 1.3 | neue | Com `das` no neutro nominativo, o adjetivo recebe `-e`. |
| 1.4 | alten | No plural com artigo definido `die`, o adjetivo recebe `-en`. |
| 1.5 | warme | Com `der` no nominativo masculino, a forma é `warme`. |
| 2.1 | den großen | `sehen` pede acusativo; masculino acusativo usa `den` e adjetivo `-en`. |
| 2.2 | Die neue | `Tasche` é feminino singular; nominativo com `die` usa adjetivo `-e`. |
| 2.3 | das alte | `Museum` é neutro e objeto direto; neutro acusativo mantém `das` e `-e`. |
| 2.4 | Die kleinen | Plural com artigo definido usa `die` e adjetivo `-en`. |
| 2.5 | den schwarzen | `Mantel` é masculino e objeto direto; por isso `den` + `schwarzen`. |
| 3.1 | den guten Lehrer | No acusativo masculino, `der gute Lehrer` vira `den guten Lehrer`. |
| 3.2 | die nette Frau | Feminino acusativo tem a mesma forma do nominativo: `die nette Frau`. |
| 3.3 | das kleine Kind | Neutro acusativo tem a mesma forma do nominativo: `das kleine Kind`. |
| 3.4 | die neuen Nachbarn | Plural com artigo definido usa adjetivo `-en`. |
| 4.1 | dem alten Freund | `mit` pede dativo; masculino dativo usa `dem` + `-en`. |
| 4.2 | dem roten Haus | `vor` com localização pede dativo; neutro dativo usa `dem` + `-en`. |
| 4.3 | der neuen Kollegin | `helfen` pede dativo; feminino dativo usa `der` + `-en`. |
| 4.4 | den kleinen Geschäften | `zu` pede dativo; plural dativo usa `den` + adjetivo `-en` e substantivo com `-n` quando possível. |
| 5.1 | Der gute Mann kommt aus Berlin. | No nominativo masculino com `der`, o adjetivo recebe `-e`, não `-en`. |
| 5.2 | Ich trinke den warmen Kaffee. | `Kaffee` é masculino acusativo: `den` + adjetivo `-en`. |
| 5.3 | Die schönen Blumen sind teuer. | Plural com `die` exige adjetivo `-en`: `schönen`. |
| 5.4 | Mit der netten Ärztin spreche ich heute. | `mit` pede dativo; feminino dativo usa `der` + `-en`. |
| 6.1 | der gute Freund | `Freund` é masculino; nominativo com artigo definido usa `der` + `gute`. |
| 6.2 | die nette Lehrerin | `Lehrerin` é feminino; nominativo com `die` usa `nette`. |
| 6.3 | die kleinen Kinder | Plural com `die` usa adjetivo `-en`: `kleinen`. |
| 6.4 | Ich kaufe das frische Brot. | `Brot` é neutro acusativo; depois de `das`, o adjetivo fica `frische`. |
| 6.5 | Wir sehen das alte Auto. | `Auto` é neutro acusativo; a forma é `das alte Auto`. |

## Perguntas da história

|---|---|
| 1. Wohin geht Lena am Samstag? | Lena geht auf den kleinen Markt. |
| 2. Was kauft Lena für ihre Mutter? | Sie kauft das warme Brot und die schönen Blumen. |
| 3. Wen trifft Lena vor dem alten Haus? | Sie trifft den netten Nachbarn. |
| 4. Womit hilft Lena dem Nachbarn? | Sie hilft ihm mit den schweren Taschen. |
| 5. Wo sitzt die Familie am Abend? | Die Familie sitzt in der hellen Küche. |
