# Respostas: Preposições de Tempo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | seit | `seit` indica uma situação que começou no passado e continua: aprender alemão desde janeiro. |
| 1.2 | um | `um` é usado para hora exata: `um 19 Uhr`. |
| 1.3 | am | Dias da semana usam `am`: `am Samstag`. |
| 1.4 | Im | Estações do ano usam `im`: `im Winter`. |
| 1.5 | ab | `ab` significa a partir de um ponto no tempo. |
| 1.6 | bis | `bis` marca o limite final: até amanhã. |
| 2.1 | seit drei Monaten | A situação continua até agora, por isso usamos `seit`. |
| 2.2 | vor drei Monaten | Para evento terminado no passado, usamos `vor`. |
| 2.3 | in drei Monaten | Para futuro com sentido de “daqui a”, usamos `in`. |
| 2.4 | um neun Uhr | Hora exata pede `um`. |
| 2.5 | im April | Meses usam `im`. |
| 2.6 | bis Sonntag | `bis` significa até. |
| 3.1 | Ich arbeite von acht bis zwölf Uhr. | `von ... bis` expressa intervalo de tempo. |
| 3.2 | Der Kurs beginnt am Montag. | Para dia da semana, use `am`. |
| 3.3 | Sie kommt in zehn Minuten. | `in` indica futuro: daqui a dez minutos. |
| 3.4 | Wir bleiben bis Freitag. | `bis` marca até quando algo continua. |
| 4.1 | einem | `seit` com `Jahr` no dativo singular fica `seit einem Jahr`. |
| 4.2 | einer | `Woche` é feminino; no dativo singular: `einer Woche`. |
| 4.3 | einer | `in einer Stunde` usa dativo feminino singular. |
| 4.4 | einem | `Monat` é masculino; com `vor` no dativo: `vor einem Monat`. |
| 5.1 | Der Unterricht beginnt in einer Stunde. | Para futuro, “daqui a uma hora”, usamos `in`. |
| 5.2 | Wir treffen uns am Dienstag. | Dia da semana pede `am`, não `im` ou `um`. |
| 5.3 | Sie hat im Oktober Geburtstag. | Mês pede `im`. |
| 5.4 | Ich arbeite von acht bis vier. | A estrutura de intervalo é `von ... bis`. |
| 6.1 | Ich lerne seit einem Jahr Deutsch. | Resposta-modelo: usa `seit` para uma atividade que continua. |
| 6.2 | Ich treffe Freunde am Wochenende. | Resposta-modelo: `am Wochenende` é expressão comum para fim de semana. |
| 6.3 | Ich stehe um sieben Uhr auf. | Resposta-modelo: `um` marca o horário exato. |
| 6.4 | Ich bleibe bis morgen zu Hause. | Resposta-modelo: `bis` indica o limite final. |
