# Respostas: Direção com acusativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | in die | **die Schule** é feminino; com direção e acusativo, fica **in die Schule**. |
| 1.2 | auf den | **der Tisch** é masculino; no acusativo, **der** vira **den**. |
| 1.3 | an die | **die Wand** é feminino; direção para a parede usa acusativo feminino **die**. |
| 1.4 | unter das | **das Bett** é neutro; no acusativo, **das** não muda. |
| 1.5 | neben das | **das Sofa** é neutro; com direção, usa-se **neben das Sofa**. |
| 1.6 | zwischen die | **die Kinder** é plural; acusativo plural com artigo definido fica **die**. |
| 2.1 | den | A pergunta implícita é **wohin?**; **der Park** vira **den Park**. |
| 2.2 | die | **die Küche** é feminino; no acusativo o artigo continua **die**. |
| 2.3 | das | **das Fenster** é neutro; no acusativo o artigo continua **das**. |
| 2.4 | die | **die Tür** é feminino; direção para a porta usa **an die Tür**. |
| 2.5 | den | **der Schreibtisch** é masculino; no acusativo fica **den Schreibtisch**. |
| 2.6 | den | **der Schrank** é masculino; com direção, usa **hinter den Schrank**. |
| 3.1 | in die Schule | **Schule** é feminino e a expressão mostra direção. |
| 3.2 | auf den Tisch | **Tisch** é masculino; por isso **den** no acusativo. |
| 3.3 | an die Wand | Para pendurar na parede, usa-se **an** com acusativo. |
| 3.4 | neben das Regal | **Regal** é neutro; acusativo neutro fica **das**. |
| 3.5 | zwischen die Stühle | Plural definido no acusativo mantém **die**. |
| 4.1 | sim | **Wohin** pergunta para onde, então indica direção. |
| 4.2 | não | **Wo** pergunta onde, não para onde. |
| 4.3 | sim | **Wohin fahren** pergunta o destino do movimento. |
| 4.4 | não | **Wo sitzt** pergunta localização parada. |
| 4.5 | sim | **Wohin hängst du** pergunta para onde algo será pendurado. |
| 5.1 | Ich gehe in die Schule. | **Schule** é feminino; direção usa **in die**. |
| 5.2 | Stell die Tasse auf den Tisch. | **stellen** indica colocar; **der Tisch** vira **den Tisch**. |
| 5.3 | Sie hängt den Spiegel an die Wand. | **Wand** é feminino e recebe movimento com **an die**. |
| 5.4 | Wir fahren in die Stadt. | **Stadt** é feminino; destino com **in die**. |
| 5.5 | Setz dich zwischen die Stühle. | **Stühle** é plural; acusativo plural definido fica **die**. |
| 6.1 | unter den Tisch | **der Tisch** é masculino; direção para debaixo da mesa usa **den**. |
| 6.2 | neben die Tür | **Tür** é feminino; direção ao lado da porta usa **die**. |
| 6.3 | vor das Fenster | **Fenster** é neutro; acusativo neutro fica **das**. |
| 6.4 | ins Kino | **ins** é a contração de **in das** para direção ao cinema. |
| 6.5 | an die Wand | **Wand** é feminino; para pendurar na parede usa **an die**. |

| 8 | den | **Schreibtisch** é masculino; com mudança de posição, fica **an den Schreibtisch**. | 1 |
| 9 | das | **Haus** é neutro; no acusativo definido continua **das**. | 1 |
| 10 | die | **Tasche** é feminino; no acusativo definido continua **die**. | 1 |
