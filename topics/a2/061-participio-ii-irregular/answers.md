# Respostas: Particípio II irregular

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | gesehen | O particípio II irregular de **sehen** é **gesehen**. |
| 1.2 | geschrieben | O particípio II irregular de **schreiben** é **geschrieben**. |
| 1.3 | gegessen | O particípio II irregular de **essen** é **gegessen**. |
| 1.4 | getrunken | O particípio II irregular de **trinken** é **getrunken**. |
| 1.5 | gefahren | O particípio II irregular de **fahren** é **gefahren**. |
| 2.1 | gesehen | A forma correta é **gesehen**; as outras formas não existem no padrão. |
| 2.2 | gegessen | **Essen** muda para **gegessen** no Perfekt. |
| 2.3 | getrunken | **Trinken** tem particípio irregular **getrunken**. |
| 2.4 | geschrieben | **Schreiben** forma **geschrieben**, não **geschreibt**. |
| 2.5 | gefahren | **Fahren** forma **gefahren**. |
| 3.1 | habe | **Essen** normalmente usa **haben** no Perfekt. |
| 3.2 | sind | **Fahren** com deslocamento usa **sein**. |
| 3.3 | hat | **Schreiben** usa **haben**; com **er**, a forma é **hat**. |
| 3.4 | habt | **Trinken** usa **haben**; com **ihr**, a forma é **habt**. |
| 3.5 | ist | **Fahren** para uma cidade indica deslocamento; com **Anna**, use **ist**. |
| 4.1 | Ich habe den Bus gesehen. | Presente com **sehen** vira Perfekt com **habe ... gesehen**. |
| 4.2 | Du hast einen Brief geschrieben. | Com **du**, o auxiliar é **hast** e o particípio é **geschrieben**. |
| 4.3 | Wir haben Suppe gegessen. | Com **wir**, use **haben** e **gegessen**. |
| 4.4 | Ihr habt Tee getrunken. | Com **ihr**, use **habt** e **getrunken**. |
| 4.5 | Sie ist nach Frankfurt gefahren. | Movimento para Frankfurt pede **ist ... gefahren**. |
| 5.1 | Wir haben im Park gegessen. | **Gegessen** usa **haben** e a forma do particípio está correta. |
| 5.2 | Ich bin nach Hause gefahren. | Para deslocamento com **fahren**, use **sein**. |
| 5.3 | Er hat ein Buch geschrieben. | **Schreiben** usa **haben** e o particípio é **geschrieben**. |
| 5.4 | Hast du den Lehrer gesehen? | A pergunta usa **hast** e o particípio correto **gesehen**. |
| 6.1 | Ich habe Reis gegessen. | Resposta-modelo com **habe ... gegessen**. |
| 6.2 | Ich habe Wasser getrunken. | Resposta-modelo com **habe ... getrunken**. |
| 6.3 | Ich bin nach Hamburg gefahren. | Deslocamento para Hamburg usa **bin ... gefahren**. |
| 6.4 | Ich habe eine Nachricht geschrieben. | **Schreiben** no Perfekt usa **habe ... geschrieben**. |
