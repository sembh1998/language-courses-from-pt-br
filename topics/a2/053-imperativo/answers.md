# Respostas: Imperativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Komm! | Com **du**, usamos a forma curta do verbo sem o pronome. |
| 1.2 | Wartet! | Com **ihr**, a forma é como no presente de **ihr**, sem o pronome. |
| 1.3 | Machen Sie! | Na forma formal, o verbo vem com **Sie** depois. |
| 1.4 | Geht! | **ihr geht** vira **Geht!** no imperativo. |
| 1.5 | Hör! | Para **du**, **hören** fica **Hör!** em pedido direto. |
| 1.6 | Seien Sie! | **sein** é irregular; a forma formal é **Seien Sie!**. |
| 2.1 | Sei | **sein** com **du** vira **Sei!**. |
| 2.2 | Lies | **lesen** tem imperativo irregular com **du**: **Lies!**. |
| 2.3 | Sprechen Sie | Pedido formal exige verbo + **Sie**. |
| 2.4 | Wartet | Para **ihr**, usa-se **wartet** sem o pronome. |
| 2.5 | Mach | Para **du**, **machen** fica **Mach!**. |
| 2.6 | Seid | **sein** com **ihr** vira **Seid!**. |
| 3.1 | Komm um zehn Uhr! | A frase com **du kommst** vira imperativo **Komm!**. |
| 3.2 | Macht die Hausaufgabe! | A forma de **ihr macht** é usada sem **ihr**. |
| 3.3 | Warten Sie einen Moment! | A forma formal mantém **Sie** depois do verbo. |
| 3.4 | Geh nach Hause! | **du gehst** vira **Geh!** no imperativo. |
| 3.5 | Seid pünktlich! | Para várias pessoas informais, **sein** vira **Seid!**. |
| 3.6 | Helfen Sie mir! | Com tratamento formal, use **Helfen Sie!**. |
| 4.1 | Warten Sie bitte einen Moment! | Para pessoa desconhecida, a forma formal com **bitte** é adequada. |
| 4.2 | Kommt bitte herein! | Para dois amigos, use **ihr**: **Kommt!**. |
| 4.3 | Wiederholen Sie bitte! | Com professora, o tratamento formal **Sie** é mais adequado. |
| 4.4 | Trink bitte Wasser! | Para uma amiga, use **du**: **Trink!**. |
| 5.1 | Komm bitte! | No imperativo com **du**, o pronome **du** não aparece. |
| 5.2 | Kommen Sie bitte hier! | A forma formal precisa de **Sie** após o verbo. |
| 5.3 | Kommen Sie bitte ins Büro! | Com **Sie**, use **Kommen Sie**, não **Kommt Sie**. |
| 5.4 | Macht bitte die Bücher auf! | Com **ihr**, o pronome não aparece no imperativo. |
| 5.5 | Sei bitte ruhig! | **sein** com **du** tem a forma irregular **Sei!**. |
| 6.1 | Komm bitte morgen! | Pedido informal singular: **Komm** + **bitte**. |
| 6.2 | Hört bitte zu! | Para colegas, **ihr** usa **Hört**; **zuhören** é separável. |
| 6.3 | Unterschreiben Sie bitte hier! | Pedido formal: verbo + **Sie** + **bitte**. |
| 6.4 | Sprich bitte langsam! | **sprechen** com **du** tem imperativo irregular **Sprich!**. |

## Perguntas da história

|---|---|

| 2 | Ana deve ler a primeira frase. |
| 3 | Ela usa **Hört bitte gut zu!**, a forma de **ihr**. |
| 4 | Porque ela fala diretamente com Pedro; para uma pessoa informal, usa-se **du**: **Sei!**. |

| 6 | Ele diz: **Warten Sie bitte noch zwei Minuten.** |
