# Respostas: Superlativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | am langsamsten | Uso predicativo: sem substantivo depois, use **am ...sten**. |
| 1.2 | am leichtesten | **leicht** recebe **-esten** na forma predicativa. |
| 1.3 | am besten | **gut** tem superlativo irregular: **am besten**. |
| 1.4 | am heißesten | **heiß** usa **-esten** para facilitar a pronúncia. |
| 2.1 | am kürzesten | A pergunta pede forma predicativa; por isso **am kürzesten**. |
| 2.2 | das beste | Antes de **Café** neutro, use **das beste**. |
| 2.3 | am liebsten | Para preferência, a forma correta é **am liebsten**. |
| 2.4 | den billigsten | **Preis** é masculino acusativo após **suchen**, então **den billigsten**. |
| 3.1 | der beste | **Film** é masculino; com artigo definido no nominativo: **der beste**. |
| 3.2 | die schönste | **Wohnung** é feminino; use **die schönste**. |
| 3.3 | das kleinste | **Zimmer** é neutro; use **das kleinste**. |
| 3.4 | die leichteste | **Tasche** é feminino; use **die leichteste**. |
| 4.1 | Dieses Restaurant ist am besten. | Sem substantivo depois do adjetivo, use **am besten**. |
| 4.2 | Das ist das beste Restaurant. | Antes de **Restaurant**, use a forma atributiva **das beste**. |
| 4.3 | Ich trinke am liebsten Tee. | Preferência pessoal é expressa com **am liebsten**. |
| 4.4 | Wir nehmen den schnellsten Zug. | **Zug** é masculino acusativo após **nehmen**, então **den schnellsten**. |
| 5.1 | Das ist der schönste Park. | A frase passa para uso atributivo antes de **Park**. |
| 5.2 | Das ist die beste Idee. | **Idee** é feminino; por isso **die beste**. |
| 5.3 | Das ist das größte Zimmer. | **Zimmer** é neutro; **groß** vira **größte**. |
| 5.4 | Das ist der billigste Laden. | **Laden** é masculino; use **der billigste**. |
| 6.1 | Dieses Hotel ist am besten. | Para predicativo, não use **das besten**; use **am besten**. |
| 6.2 | Das ist der beste Plan. | Antes de **Plan**, use artigo + superlativo atributivo. |
| 6.3 | Ich fahre am liebsten mit dem Zug. | A forma correta de preferência é **am liebsten**, não **am gernsten**. |
| 6.4 | Das ist die heißeste Suppe. | **Suppe** é feminino e o adjetivo vem antes do substantivo. |

## Gabarito do teste

| ID | Resposta | Explicação | Pontos |
|---|---|---|---|
| 1 | am bequemsten | O adjetivo está no fim da frase; uso predicativo com **am ...sten**. | 1 |
| 2 | Das ist das beste Buch. | Antes de **Buch** neutro, a forma é **das beste**. | 1 |
| 3 | am meisten | Para “dormir mais”, use a forma predicativa **am meisten**. | 1 |
| 4 | am deutlichsten | Sem substantivo depois, use **am deutlichsten**. | 1 |
| 5 | die älteste | **Straße** é feminino; **alt** recebe Umlaut: **älteste**. | 1 |
| 6 | Die nächste Station ist dort. | **nächste** é a forma de **nah** usada para “próxima”. | 1 |
| 7 | Ich höre am liebsten Musik. | Para dizer do que mais gosta, use **am liebsten**. | 1 |
| 8 | Der rote Pullover ist am wärmsten. | A frase é predicativa; não use artigo antes do superlativo. | 1 |
| 9 | Das ist die teuerste Tasche. | **Tasche** é feminino; forma atributiva: **die teuerste**. | 1 |
| 10 | das frischeste | **Brot** é neutro; após **kaufen**, use **das frischeste Brot**. | 1 |
| 11 | am schnellsten | **schnell** no superlativo predicativo vira **am schnellsten**. | 1 |
| 12 | Das ist das größte Problem. | **Problem** é neutro; **groß** vira **größte**. | 1 |
| 13 | Ich trinke am liebsten Kaffee. | O correto é **am liebsten**, não **am liebersten**. | 1 |
| 14 | die kürzeste Frage | **Frage** é feminino; use **die kürzeste** antes do substantivo. | 1 |
