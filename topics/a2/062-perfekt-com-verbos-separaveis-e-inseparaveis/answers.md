# Respostas: Perfekt com verbos separáveis e inseparáveis

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | aufgestanden | `aufstehen` é separável; o `ge-` fica entre `auf-` e o radical. |
| 1.2 | eingekauft | `einkaufen` é separável: `ein` + `ge` + `kauft`. |
| 1.3 | besucht | `besuchen` tem prefixo inseparável `be-`, então não recebe `ge-`. |
| 1.4 | bekommen | `bekommen` não recebe `ge-` e mantém a mesma forma no Particípio II. |
| 1.5 | verstanden | `verstehen` tem prefixo inseparável `ver-`, então não recebe `ge-`. |
| 1.6 | angerufen | `anrufen` é separável: `an` + `ge` + `rufen`. |
| 2.1 | eingekauft | A forma correta para verbo separável é `ein` + `ge` + `kauft`. |
| 2.2 | besucht | Com `be-`, não se usa `ge-` no Particípio II. |
| 2.3 | verstanden | Com `ver-`, a forma correta é `verstanden`, sem `ge-`. |
| 2.4 | aufgeräumt | `aufräumen` é separável e recebe `ge-` depois de `auf-`. |
| 2.5 | bekommen | `bekommen` é inseparável e não vira `gebekommen`. |
| 2.6 | mitgebracht | `mitbringen` é separável: `mit` + `ge` + `bracht`. |
| 3.1 | separável | `aufstehen` separa o prefixo `auf-` em formas simples e no Particípio II. |
| 3.2 | inseparável | `besuchen` começa com `be-`, prefixo inseparável. |
| 3.3 | separável | `einkaufen` é separável; no Perfekt fica `eingekauft`. |
| 3.4 | inseparável | `verstehen` começa com `ver-`, prefixo inseparável. |
| 3.5 | separável | `anrufen` é separável; no Perfekt fica `angerufen`. |
| 3.6 | inseparável | `bekommen` começa com `be-`, prefixo inseparável. |
| 4.1 | bin aufgestanden | `aufstehen` normalmente usa `sein` no Perfekt e o Particípio II é `aufgestanden`. |
| 4.2 | haben eingekauft | `Meine Eltern` é plural, então o auxiliar é `haben`; o Particípio II é `eingekauft`. |
| 4.3 | hat besucht | `Sara` usa `hat`; `besucht` não recebe `ge-`. |
| 4.4 | haben bekommen | `Wir` usa `haben`; `bekommen` é inseparável. |
| 4.5 | habt verstanden | `Ihr` usa `habt`; `verstanden` vem de `verstehen`. |
| 4.6 | hat angerufen | `Ben` usa `hat`; `anrufen` vira `angerufen`. |
| 5.1 | aufgestanden | `geaufstanden` está errado porque `ge-` deve vir depois de `auf-`. |
| 5.2 | eingekauft | `geeinkauft` está errado; o padrão é `ein` + `ge` + `kauft`. |
| 5.3 | besucht | `gebesucht` está errado porque `be-` é inseparável. |
| 5.4 | bekommen | `gebekommen` está errado porque `bekommen` não recebe `ge-`. |
| 5.5 | verstanden | `geverstanden` está errado porque `ver-` é inseparável. |
| 5.6 | angerufen | `geanrufen` está errado; em verbos separáveis, `ge-` entra após o prefixo. |
| 6.1 | Ich bin früh aufgestanden. | Para “levantar-se”, usa-se `sein` e `aufgestanden`. |
| 6.2 | Wir haben heute eingekauft. | `einkaufen` usa `haben` e vira `eingekauft`. |
| 6.3 | Sie hat ihre Tante besucht. | `besuchen` é inseparável e fica `besucht`. |
| 6.4 | Du hast einen Brief bekommen. | `bekommen` usa `haben` e não recebe `ge-`. |
| 6.5 | Ich habe die Frage verstanden. | `verstehen` vira `verstanden`, sem `ge-`. |
| 6.6 | Er hat seinen Vater angerufen. | `anrufen` vira `angerufen`; `Vater` fica acusativo masculino. |
