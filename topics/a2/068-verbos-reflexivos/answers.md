# Respostas: Verbos Reflexivos

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | mich | Com **ich**, o pronome reflexivo é **mich**. |
| 1.2 | dich | Com **du**, o pronome reflexivo é **dich**. |
| 1.3 | sich | Com **Lisa** na terceira pessoa singular, usamos **sich**. |
| 1.4 | uns | Com **wir**, o pronome reflexivo é **uns**. |
| 1.5 | euch | Com **ihr**, o pronome reflexivo é **euch**. |
| 2.1 | Ich interessiere mich für Bücher. | **Ich** pede **mich**; o verbo usa **für**. |
| 2.2 | Ihr wascht euch. | **Ihr** pede **euch** como pronome reflexivo. |
| 2.3 | Wir freuen uns. | **Wir** pede **uns**; **sich** não combina aqui. |
| 3.1 | auf | **sich freuen auf** indica algo esperado no futuro. |
| 3.2 | über | **sich freuen über** indica alegria com algo recebido ou existente. |
| 3.3 | für | **sich interessieren** é usado com **für**. |
| 3.4 | auf | As férias são futuras; por isso usamos **auf**. |
| 4.1 | Ich wasche mich am Morgen. | A ação volta para **ich**, então usamos **mich**. |
| 4.2 | Sie freut sich über die Nachricht. | Para terceira pessoa, usamos **sich**; com notícia recebida, **über**. |
| 4.3 | Wir interessieren uns für Musik. | **Wir** pede **uns** e o verbo pede **für**. |
| 4.4 | Interessierst du dich für Kino? | Em pergunta com **du**, o pronome é **dich**. |
| 5.1 | Du wäschst dich. | Ao mudar para **du**, o verbo vira **wäschst** e o pronome vira **dich**. |
| 5.2 | Wir freuen uns auf Freitag. | Ao mudar para **wir**, usamos **freuen** e **uns**. |
| 5.3 | Er interessiert sich für Politik. | Na terceira pessoa singular, usamos **interessiert** e **sich**. |
| 5.4 | Ihr freut euch über den Kaffee. | Com **ihr**, usamos **freut** e **euch**. |
| 6.1 | Ich freue mich auf den Abend. | **Ich** não combina com **sich**; o correto é **mich**. |
| 6.2 | Wir interessieren uns für Sprachen. | **Wir** pede **uns**, não **sich**. |
| 6.3 | Du wäschst dich nach der Arbeit. | A pessoa **du** pede o reflexivo **dich**. |
| 6.4 | Anna freut sich über die Blumen. | Com algo recebido/presente, usa-se **freuen über**, não **für**. |
