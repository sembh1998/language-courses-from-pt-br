# Flashcards Prompt

Generate `flashcards.yaml` for a Spanish learning topic for Brazilian Portuguese speakers.

If flashcards are useful, create concise cards with `front` in Spanish, `back` in Brazilian Portuguese, `example` in Spanish, and `example_translation` in Brazilian Portuguese. If flashcards are not useful, use `cards: []` and a clear `note` explaining why.

Keep `front` and `example` clean Spanish text with no Markdown or parenthetical Portuguese.
