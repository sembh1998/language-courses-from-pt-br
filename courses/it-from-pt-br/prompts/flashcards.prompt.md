# Flashcards Prompt

Generate `flashcards.yaml` for an Italian learning topic for Brazilian Portuguese speakers.

If flashcards are useful, create concise cards with `front` in Italian, `back` in Brazilian Portuguese, `example` in Italian, and `example_translation` in Brazilian Portuguese. If flashcards are not useful, use `cards: []` and a clear `note` explaining why.

Keep `front` and `example` clean Italian text with no Markdown or parenthetical Portuguese.
