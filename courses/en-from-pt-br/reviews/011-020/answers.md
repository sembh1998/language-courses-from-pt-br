# Revisão 011-020: gabarito

## Gabarito do teste

| Questão | Resposta | Explicação |
|---|---|---|
| 1 | There are two new messages in the team channel. | Usa `there are` porque `two new messages` está no plural. |
| 2 | Where is the file? | `Where` pergunta localização; a ordem correta é `Where is...`. |
| 3 | Hi Ana, can you check the ticket? | É uma mensagem curta de chat com saudação, pedido simples e contexto de trabalho. |
| 4 | The build is running now. | `is running` indica presente contínuo, uma ação acontecendo agora. |
| 5 | We can't deploy today. | `can't` expressa impossibilidade ou falta de permissão no contexto. |
| 6 | is | Usa `there is` com `a private repository`, que está no singular. |
| 7 | at | Para posições específicas na tela, `at the top of the page` é a forma natural. |
| 8 | simple | `simple tool` descreve uma ferramenta fácil de entender; outro adjetivo A1 adequado pode ser aceito pelo professor. |
| 9 | Who | `Who` pergunta a pessoa responsável por atualizar o arquivo. |
| 10 | reviewing | Depois de `I am`, o presente contínuo usa verbo com `-ing`: `reviewing`. |
| 11 | There are three files in the docs folder. | `There are` combina com `three files`, e `in the docs folder` indica localização em pasta. |
| 12 | The terminal is next to the browser. | `next to` significa `ao lado de`; a frase mantém ordem simples sujeito + verbo + lugar. |
| 13 | Can you join the meeting now? | `Can you... ?` faz um pedido/pergunta de habilidade ou disponibilidade. |
| 14 | The pipeline is running now. | `is running now` descreve o status atual do pipeline. |
| 15 | The path is in the config file. | `path` e `config file` são vocabulário do bloco; `in` indica que o caminho está dentro do arquivo. |
| 16 | This is a safe browser. | A ordem básica é `This is a` + adjetivo + substantivo. |
| 17 | Are there any bugs today? | Pergunta com plural usa `Are there`; em perguntas, `any` é natural com `bugs`. |
| 18 | The files are under the root folder. | `files` está no plural, então o verbo correto é `are`. |
| 19 | I am updating the board. | Resposta-modelo no presente contínuo; outras respostas com `I am` + verbo `-ing` podem ser aceitas se fizerem sentido. |
| 20 | Hi team, I am available at ten. Thanks. | A mensagem usa saudação, disponibilidade simples e encerramento curto, adequado para chat profissional. |
