# Revisão 031-040: respostas

## Gabarito do teste

| ID | Resposta | Observação |
| --- | --- | --- |
| 1 | has | The checkout service é singular, então usa has. Em perguntas e negativas com auxiliar, voltaria para have. |
| 2 | Marina's branch is ready. | Para posse com nome próprio, usamos 's: Marina's branch significa a branch da Marina. |
| 3 | because | Because introduz a causa: o preview está vazio porque o arquivo de dados está ausente. |
| 4 | The report is quite clear. | Quite vem antes do adjetivo clear para indicar intensidade moderada. |
| 5 | easier | Easy forma o comparativo como easier; a estrutura completa é easier than. |
| 6 | a product manager | Product manager conversa com usuários/stakeholders e ajuda a organizar prioridades do produto. |
| 7 | A sessão expirou. | Expired indica que algo passou do prazo ou deixou de ser válido; em software, normalmente é preciso entrar de novo. |
| 8 | I am a backend developer. | Uma bio A1 usa I am + artigo + cargo; a ordem das outras opções não é natural. |
| 9 | Can you help me with the VPN? | Can you help me with...? é uma forma simples e educada de pedir ajuda com um problema específico. |
| 10 | have | We exige have, não has. A frase fala que o time possui um servidor de staging. |
| 11 | designer's | Designer's notes significa as anotações do designer; o 's marca posse. |
| 12 | quickly | Quickly descreve como o upload roda; é advérbio de modo. |
| 13 | than | Comparativos como worse precisam de than para comparar duas coisas. |
| 14 | The API has an error, but the app works. | API é singular, por isso has; but conecta contraste entre erro na API e funcionamento do app. |
| 15 | I work with JavaScript and I like tests. | Work with apresenta stack/ferramenta, e and soma duas informações da bio. |
| 16 | Does the platform have a status page? | Depois de does, o verbo principal fica na forma base: have, não has. |
| 17 | The build is too slow. | Too vem antes do adjetivo e indica excesso: lenta demais. |
| 18 | O ID do workspace é inválido; é preciso verificar ou corrigir esse ID. | Invalid indica valor errado ou formato incorreto; a ação provável é revisar o ID usado. |
| 19 | Resposta possível: Hello, I am new here. Can you help me with setup? | A resposta deve se apresentar e pedir ajuda de forma simples; variações A1 corretas são aceitáveis. |
| 20 | Resposta possível: The new automation is easier than the manual process. | A produção deve usar easier than para comparar a automação nova com o processo manual; variações A1 corretas são aceitáveis. |
