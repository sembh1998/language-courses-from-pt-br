# Flashcards Prompt

Generate `flashcards.yaml` for an English learning topic for Brazilian Portuguese speakers.

If flashcards are useful, create concise cards with `front` in English, `back` in Brazilian Portuguese, `example` in English, and `example_translation` in Brazilian Portuguese. If flashcards are not useful, use `cards: []` and a clear `note` explaining why.

Keep `front` and `example` clean English text with no Markdown or parenthetical Portuguese.
