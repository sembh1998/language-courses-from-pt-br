# Respostas: This, that, these, those na interface

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | this file | `This` combina com singular perto. |
| 1.2 | these lines | `These` combina com plural perto. |
| 1.3 | those logs | `Those` combina com plural longe. |
| 2.1 | This | `Button` é singular e a frase aponta para algo perto. |
| 2.2 | Those | `Files` é plural; `those` indica longe. |
| 2.3 | These | `Tests` é plural; `these` indica perto. |
| 2.4 | That | `Bug` é singular; `that` indica longe ou já mencionado. |
| 3.1 | This file is open. | `This file` é a forma para “este arquivo”. |
| 3.2 | Those lines are old. | `Those` combina com plural e usa `are`. |
| 3.3 | These logs are useful. | `These logs` é plural perto. |
| 4.1 | These lines are important. | No plural perto, trocamos `this line is` por `these lines are`. |
| 4.2 | That button is blue. | No singular longe, usamos `that button is`. |
| 4.3 | These logs are useful. | `Logs` é plural, então não combina com `this`. |
| 5.1 | This button is important. | Resposta modelo; deve usar `this button` corretamente. |
| 5.2 | Those logs are from yesterday. | Resposta modelo; deve usar `those logs` corretamente. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | This file is new. | `File` é singular e perto. |
| 2 | Those | `Errors` é plural; `those` combina com plural. |
| 3 | that button | `That` combina com singular longe. |
| 4 | These | `Lines` é plural perto/selecionado. |
| 5 | That bug is urgent. | “Aquele bug” corresponde a `that bug`. |
| 6 | These tests are slow. | Plural perto usa `these` e `are`. |
| 7 | Those files are old. | A ordem correta é demonstrativo, substantivo, verbo e adjetivo. |
| 8 | Click this button. | Resposta modelo; a instrução deve usar `this button`. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
