# História: Countable e uncountable em contexto técnico

Nível: A1

## História em inglês

Marta opens a folder.

There are many files in the folder.
There is some code in one file.

Marta sees an issue on the board.
She needs information about the issue.

The team has data in the database.
The software is simple and useful.

Marta says, "I need some information and one file."
Her teammate sends the file.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Marta opens a folder. | Marta abre uma pasta. |
| There are many files in the folder. | Há muitos arquivos na pasta. |
| There is some code in one file. | Há algum código em um arquivo. |
| Marta sees an issue on the board. | Marta vê uma issue no quadro. |
| She needs information about the issue. | Ela precisa de informação sobre a issue. |
| The team has data in the database. | A equipe tem dados no banco de dados. |
| The software is simple and useful. | O software é simples e útil. |
| Marta says, "I need some information and one file." | Marta diz: "Eu preciso de alguma informação e um arquivo." |
| Her teammate sends the file. | O colega de equipe dela envia o arquivo. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| many files | muitos arquivos | `Files` é contável no plural. |
| some code | algum código | `Code` é incontável no uso geral. |
| an issue | uma issue | `Issue` é contável. |
| information | informação | Não use `informations`. |
| software | software | Não use `softwares` neste contexto. |

## Perguntas de compreensão

1. What does Marta open?
2. Are there many files in the folder?
3. What does Marta need about the issue?
4. Where does the team have data?
5. Is the software simple?
