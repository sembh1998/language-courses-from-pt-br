# Respostas: Countable e uncountable em contexto técnico

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | file | `File` é contável: `a file`, `two files`. |
| 1.2 | some | `Information` é incontável; `some information` é natural. |
| 1.3 | much data | `Data` é tratado aqui como incontável, então combina com `much`. |
| 2.1 | many | `Files` é plural contável. |
| 2.2 | much | `Code` é incontável no uso geral. |
| 2.3 | some | `Some` pode ser usado com `data`. |
| 2.4 | an | `Issue` começa com som de vogal e é contável: `an issue`. |
| 3.1 | I have two files. | `Files` recebe plural porque há quantidade específica. |
| 3.2 | We need information. | Em inglês, `information` não recebe `s`. |
| 3.3 | This software is new. | `Software` não usa `a` neste uso geral. |
| 4.1 | I need much information. | `Much` combina com incontáveis como `information`. |
| 4.2 | This is software. | Removemos `a` antes de `software`. |
| 4.3 | files | `File` é contável e o plural regular é `files`. |
| 5.1 | We have many issues. | Resposta modelo; deve usar `many issues` corretamente. |
| 5.2 | I need some code. | Resposta modelo; deve usar `some code` sem plural. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | I have three files. | `Files` é contável e aceita número. |
| 2 | many | `Issues` é contável no plural. |
| 3 | some software | `Software` fica sem plural nesse uso. |
| 4 | much | `Time` é incontável nessa frase. |
| 5 | The information is in the file. | `Information` fica no singular gramatical em inglês. |
| 6 | There is much code in this branch. | `Code` é incontável; por isso usamos `there is` e `much`. |
| 7 | We need some data. | Ordem básica: sujeito, verbo e complemento. |
| 8 | There are two buttons on the page. | Resposta modelo; precisa ter uma palavra técnica contável no plural. |
