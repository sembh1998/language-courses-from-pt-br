# Countable e uncountable em contexto técnico

Nível: A1

## Objetivo

Neste tópico, você vai diferenciar substantivos contáveis e incontáveis em inglês dentro de situações técnicas. O foco é evitar erros comuns como `informations` e `softwares` e usar `much`, `many`, `some` e `a/an` com segurança.

## Explicação

Substantivos contáveis são coisas que você pode contar em unidades: `a file`, `two issues`, `three buttons`. Substantivos incontáveis normalmente não recebem plural e não usam `a/an`: `some code`, `information`, `data`, `software`.

Use `many` com contáveis no plural:

- `many files`
- `many issues`
- `many lines`

Use `much` com incontáveis:

- `much information`
- `much data`
- `much software`

Em frases positivas simples, `some` funciona bem com os dois grupos:

- `I have some files.`
- `I need some information.`

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| I opened a file. | Eu abri um arquivo. | `File` é contável; usamos `a`. |
| We have three issues. | Nós temos três issues/problemas. | `Issue` é contável e pode ir ao plural. |
| I need some code. | Eu preciso de algum código. | `Code` é incontável neste uso geral. |
| This information is important. | Esta informação é importante. | Em inglês, `information` não vai ao plural. |
| The data is old. | Os dados estão antigos. | Em A1, trate `data` como incontável no contexto técnico. |
| This software is free. | Este software é gratuito. | Não diga `a software` ou `softwares`. |
| How many files are in the folder? | Quantos arquivos há na pasta? | `Many` combina com contáveis no plural. |
| How much data do we need? | De quantos dados nós precisamos? | `Much` combina com incontáveis. |

## Erros comuns

1. Usar plural com palavras incontáveis.

Forma correta: `I need information.`

Evite: `I need informations.`

2. Usar `a` antes de `software` no sentido geral.

Forma correta: `This is useful software.`

Evite: `This is a useful software.`

3. Trocar `many` e `much`.

Forma correta: `many files`, `much data`.

Evite: `much files`, `many data`.

4. Confundir português com inglês.

Em português, dizemos “informações” e “softwares”. Em inglês técnico básico, prefira `information` e `software` sem `s`.

## Prática guiada

Classifique mentalmente:

- `file`: contável.
- `issue`: contável.
- `code`: incontável.
- `information`: incontável.
- `software`: incontável.

Complete:

- `How ___ files are open?` -> `many`
- `How ___ data do we have?` -> `much`
- `I need ___ information.` -> `some`

## Resumo

Use `a/an` e números com palavras contáveis como `file` e `issue`. Use `some`, `much` e a forma sem plural com palavras incontáveis como `code`, `information`, `data` e `software`.
