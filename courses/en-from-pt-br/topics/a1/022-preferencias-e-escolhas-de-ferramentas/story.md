# História: Likes, preferences e escolhas de ferramentas

Nível: A1

## História em inglês

Iara is choosing tools for a small project.

She says, "I like Python."

Her teammate says, "I prefer JavaScript for this app."

Iara uses Linux and VS Code.

For her, VS Code is easier.

The team chooses a simple framework.

Iara says, "This framework is better for beginners."

The project starts well.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Iara is choosing tools for a small project. | Iara está escolhendo ferramentas para um projeto pequeno. |
| She says, "I like Python." | Ela diz: "Eu gosto de Python." |
| Her teammate says, "I prefer JavaScript for this app." | O colega dela diz: "Eu prefiro JavaScript para este aplicativo." |
| Iara uses Linux and VS Code. | Iara usa Linux e VS Code. |
| For her, VS Code is easier. | Para ela, VS Code é mais fácil. |
| The team chooses a simple framework. | A equipe escolhe um framework simples. |
| Iara says, "This framework is better for beginners." | Iara diz: "Este framework é melhor para iniciantes." |
| The project starts well. | O projeto começa bem. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| `choosing` | escolhendo | Ação de decidir entre opções. |
| `I like` | eu gosto de | Não use `of`. |
| `I prefer` | eu prefiro | Preferência entre opções. |
| `easier` | mais fácil | Comparação simples. |
| `better` | melhor | Comparação simples. |

## Perguntas de compreensão

1. Para que Iara está escolhendo ferramentas?
2. De qual linguagem Iara gosta?
3. Qual linguagem o colega prefere?
4. Quais ferramentas Iara usa?
5. Para quem o framework é melhor?
