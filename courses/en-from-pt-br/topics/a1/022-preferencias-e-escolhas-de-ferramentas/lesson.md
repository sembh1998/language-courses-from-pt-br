# Likes, preferences e escolhas de ferramentas

Nível: A1

## Objetivo

Falar de gostos e preferências sobre linguagem, editor, sistema operacional e framework com frases simples.

## Explicação

Use `I like` para dizer que você gosta de algo: `I like Python`. Use `I prefer` para escolher uma opção: `I prefer VS Code`. Use `I use` para dizer o que você usa: `I use Linux`.

Para comparações muito simples, use `better` e `easier`: `This editor is easier`, `Linux is better for me`. A frase `for me` deixa claro que é sua opinião.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| I like Python. | Eu gosto de Python. | Gosto geral. |
| I prefer VS Code. | Eu prefiro VS Code. | Escolha entre opções. |
| I use Linux at work. | Eu uso Linux no trabalho. | Ferramenta ou sistema usado. |
| This framework is easier. | Este framework é mais fácil. | Comparação simples. |
| For me, this editor is better. | Para mim, este editor é melhor. | Opinião pessoal. |

## Erros comuns

Evite `I like of Python`. Em inglês, diga `I like Python` sem `of`.

Evite `I prefer more Linux`. Diga `I prefer Linux`.

Evite tratar opinião como fato absoluto: `For me` ajuda a suavizar.

## Prática guiada

Monte frases com `I like`, `I prefer` e `I use`. Depois acrescente uma opinião simples com `better` ou `easier`.

## Resumo

Com poucas estruturas, você consegue falar de preferências técnicas: `I like`, `I prefer`, `I use`, `better` e `easier`.
