# Gabarito: Likes, preferences e escolhas de ferramentas

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | I like Python. | Em inglês, `like` não usa `of`. |
| 1.2 | I prefer Linux. | `Prefer` expressa preferência. |
| 1.3 | This editor is easier. | Use `is` antes de `easier`. |
| 2.1 | like | `I like` significa eu gosto de. |
| 2.2 | use | `Use` fala da ferramenta usada. |
| 2.3 | easier | `Easier` significa mais fácil. |
| 2.4 | better | `Better` significa melhor. |
| 3.1 | I use VS Code. | `Use` indica uso. |
| 3.2 | For me, this framework is easier. | `For me` mostra opinião pessoal. |
| 3.3 | I prefer this operating system. | `Operating system` é sistema operacional. |
| 4.1 | I like Java. | Remova `of` depois de `like`. |
| 4.2 | I prefer this editor. | `Prefer` não precisa de `more`. |
| 5.1 | I like Python. | Resposta-modelo com linguagem. |
| 5.2 | I use Git. | Resposta-modelo com ferramenta. |
