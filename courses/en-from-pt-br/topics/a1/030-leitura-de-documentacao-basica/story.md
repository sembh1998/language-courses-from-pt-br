# História: Leitura de documentação básica

Nível: A1

## História em inglês

Bianca opens the project documentation.
The first title is Overview.
She reads the requirements.
The project needs Node and Git.
Then she finds Installation.
There is a short list of steps.
She sees a warning before the command.
The note says the command is slow.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Bianca opens the project documentation. | Bianca abre a documentação do projeto. |
| The first title is Overview. | O primeiro título é Visão geral. |
| She reads the requirements. | Ela lê os requisitos. |
| The project needs Node and Git. | O projeto precisa de Node e Git. |
| Then she finds Installation. | Depois ela encontra Instalação. |
| There is a short list of steps. | Há uma lista curta de passos. |
| She sees a warning before the command. | Ela vê um aviso antes do comando. |
| The note says the command is slow. | A nota diz que o comando é lento. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| documentation | documentação | Texto que explica um projeto ou ferramenta. |
| needs | precisa | Verbo simples para necessidade. |
| slow | lento | Descreve velocidade baixa. |
