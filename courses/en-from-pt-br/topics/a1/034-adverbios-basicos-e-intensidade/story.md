# História: Adverbs básicos e intensidade

Nível: A1

## História em inglês

The team's app is very slow today.
The API is really fast, but the page is too slow.
Ana checks the setup.
The setup is quite simple.
Rafa runs the test quickly.
The script starts slowly.
Ana reads the command carefully and runs it correctly.
Now the app works.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| The team's app is very slow today. | O app da equipe está muito lento hoje. |
| The API is really fast, but the page is too slow. | A API é realmente rápida, mas a página está lenta demais. |
| Ana checks the setup. | Ana verifica o setup. |
| The setup is quite simple. | O setup é bastante simples. |
| Rafa runs the test quickly. | Rafa roda o teste rapidamente. |
| The script starts slowly. | O script inicia lentamente. |
| Ana reads the command carefully and runs it correctly. | Ana lê o comando com cuidado e o roda corretamente. |
| Now the app works. | Agora o app funciona. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| today | hoje | Marca tempo presente. |
| carefully | cuidadosamente | Advérbio extra para modo. |
| works | funciona | Verbo comum para sistemas. |

## Perguntas de compreensão

1. How is the team's app today?
2. Is the API fast?
3. Is the setup hard or simple?
4. How does Rafa run the test?
5. How does Ana run the command?
