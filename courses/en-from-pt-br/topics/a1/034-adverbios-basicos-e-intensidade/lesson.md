# Adverbs básicos e intensidade

Nível: A1

## Objetivo

Usar advérbios básicos para falar de intensidade, velocidade e forma correta de execução em contextos de testes, comandos e problemas.

## Explicação

Advérbios dão mais informação sobre adjetivos ou verbos.

Intensidade antes de adjetivos:

- `very slow` = muito lento
- `really fast` = realmente rápido / muito rápido
- `quite simple` = bastante simples
- `too slow` = lento demais

Modo geralmente depois do verbo ou objeto:

- `The test runs quickly.` = O teste roda rapidamente.
- `The script starts slowly.` = O script inicia lentamente.
- `Run the command correctly.` = Rode o comando corretamente.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| The app is very slow. | O app está muito lento. | `Very` intensifica `slow`. |
| The setup is quite simple. | O setup é bastante simples. | `Quite` é intensidade moderada. |
| The test runs quickly. | O teste roda rapidamente. | `Quickly` descreve o verbo. |
| Run the command correctly. | Rode o comando corretamente. | `Correctly` indica modo. |

## Erros comuns

- Não use `too` para elogio neutro. `Too slow` significa lento demais, algo negativo.
- Coloque `very`, `really`, `quite` e `too` antes do adjetivo: `very fast`, não `fast very`.
- Para verbos, use formas como `quickly`, `slowly`, `correctly`.

## Prática guiada

Compare:

- `The test is slow.` -> `The test is very slow.`
- `The command runs.` -> `The command runs quickly.`
- `The setup is simple.` -> `The setup is quite simple.`

## Resumo

Use `very`, `really`, `quite` e `too` antes de adjetivos. Use `quickly`, `slowly` e `correctly` para explicar como uma ação acontece.
