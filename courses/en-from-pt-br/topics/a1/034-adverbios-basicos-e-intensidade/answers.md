# Respostas: Adverbs básicos e intensidade

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | lento demais | `Too slow` indica excesso negativo. |
| 1.2 | quickly | `Quickly` significa rapidamente. |
| 1.3 | very slow | Advérbios de intensidade vêm antes do adjetivo. |
| 2.1 | really | `Really fast` intensifica rápido. |
| 2.2 | slowly | `Slowly` descreve como o script inicia. |
| 2.3 | correctly | `Correctly` indica modo correto de execução. |
| 3.1 | The app is very slow. | `Very` vem antes de `slow`. |
| 3.2 | The test runs quickly. | `Quickly` descreve o verbo `runs`. |
| 3.3 | The setup is quite simple. | `Quite` significa bastante. |
| 4.1 | The server is very fast. | `Very` fica antes do adjetivo `fast`. |
| 4.2 | The command runs correctly. | `Correctly` indica como o comando roda. |
| 4.3 | The build is too slow. | `Too` antes de `slow` indica excesso. |
| 5.1 | The test runs quickly. | Resposta modelo; deve usar `quickly` ou `slowly` com uma ação. |
