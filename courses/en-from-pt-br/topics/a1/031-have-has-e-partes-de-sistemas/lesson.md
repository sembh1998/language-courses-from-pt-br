# Have/has e partes de sistemas

Nível: A1

## Objetivo

Usar `have` e `has` para falar de recursos, partes de sistemas, componentes e problemas simples.

## Explicação

`Have` significa ter. Use `have` com `I`, `you`, `we` e `they`. Use `has` com `he`, `she`, `it` e nomes singulares.

- `We have two services.` = Nós temos dois serviços.
- `The app has a login page.` = O app tem uma página de login.
- `It has a bug.` = Ele/isso tem um bug.

Para negativas, use `do not have` ou `does not have`.

- `We do not have a test server.`
- `The app does not have a dark mode.`

Para perguntas, use `Do` ou `Does` no começo.

- `Do we have a backup?`
- `Does the app have a login page?`

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| The app has a login page. | O app tem uma página de login. | `The app` é singular, então use `has`. |
| We have two services. | Nós temos dois serviços. | Com `we`, use `have`. |
| Does the API have an error? | A API tem um erro? | Pergunta com `does`. |
| They do not have access. | Eles não têm acesso. | Negativa com `do not have`. |

## Erros comuns

- Não use `has` com `we`: o correto é `We have`, não `We has`.
- Em pergunta com `does`, o verbo volta para `have`: `Does it have...?`, não `Does it has...?`.
- Em negativa com `does not`, use `have`: `It does not have...`.

## Prática guiada

Transforme mentalmente:

- Afirmação: `The system has a cache.`
- Pergunta: `Does the system have a cache?`
- Negativa: `The system does not have a cache.`

## Resumo

Use `have` com plurais e `I/you/we/they`; use `has` com singular e `he/she/it`. Em perguntas e negativas com `does`, use sempre `have`.
