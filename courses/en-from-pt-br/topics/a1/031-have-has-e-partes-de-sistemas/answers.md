# Respostas: Have/has e partes de sistemas

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | The app has a login page. | `The app` é singular, então usamos `has`. |
| 1.2 | have | Com `we`, usamos `have`. |
| 1.3 | Does it have a cache? | Depois de `does`, o verbo fica como `have`. |
| 2.1 | have | Com `they`, usamos `have`. |
| 2.2 | has | `The server` é singular, então usa `has`. |
| 2.3 | have | Em pergunta com `does`, usamos `have`. |
| 3.1 | We have a backup. | Com `we`, a forma correta é `have`. |
| 3.2 | The component has a problem. | `The component` é singular, então usa `has`. |
| 3.3 | The system does not have a cache. | Negativa com `does not` usa `have`. |
| 4.1 | Does the app have a new feature? | Pergunta com sujeito singular usa `does` + `have`. |
| 4.2 | We do not have a test server. | Com `we`, a negativa usa `do not have`. |
| 4.3 | They have access. | Com `they`, trocamos `has` por `have`. |
| 5.1 | The app has a login page. | Resposta modelo; deve usar `has` com um sujeito singular. |
