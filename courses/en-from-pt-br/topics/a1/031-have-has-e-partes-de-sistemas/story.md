# História: Have/has e partes de sistemas

Nível: A1

## História em inglês

The team has a small app.
The app has a login page.
It has two services.
One service has a problem.
Ana asks, "Does the API have logs?"
Rui says, "Yes, it has logs."
The team has a backup.
Now the problem is clear.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| The team has a small app. | A equipe tem um app pequeno. |
| The app has a login page. | O app tem uma página de login. |
| It has two services. | Ele tem dois serviços. |
| One service has a problem. | Um serviço tem um problema. |
| Ana asks, "Does the API have logs?" | Ana pergunta: "A API tem logs?" |
| Rui says, "Yes, it has logs." | Rui diz: "Sim, ela tem logs." |
| The team has a backup. | A equipe tem um backup. |
| Now the problem is clear. | Agora o problema está claro. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| logs | registros | Arquivos ou linhas que mostram eventos. |
| clear | claro | Aqui significa fácil de entender. |
| asks | pergunta | Verbo usado para fazer uma pergunta. |

## Perguntas de compreensão

1. What does the team have?
2. What does the app have?
3. How many services does it have?
4. Does the API have logs?
5. Does the team have a backup?
