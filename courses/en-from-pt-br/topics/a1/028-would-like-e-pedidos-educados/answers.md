# Respostas: Modal would like e pedidos educados

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Could you help me? | `Could you...` é uma forma educada de pedir. |
| 1.2 | I would like to ask a question. | `Would like to` significa “gostaria de”. |
| 1.3 | Can I ask a question? | A frase pede permissão para perguntar. |
| 2.1 | to | Com verbo, usamos `would like to`. |
| 2.2 | Could | `Could you...` inicia pedido educado. |
| 2.3 | Can | `Can I...` pede permissão. |
| 2.4 | like | `I'd like` é contração de `I would like`. |
| 3.1 | Could you check this file? | Pedido educado com `could`. |
| 3.2 | I would like to understand this error. | `Would like to understand` expressa intenção educada. |
| 3.3 | Can I ask a question? | Frase natural para entrar em uma conversa. |
| 4.1 | Could you help me with this ticket? | A ordem transforma comando em pedido educado. |
| 4.2 | I would like to ask about the bug. | `I want to` pode virar `I would like to`. |
| 4.3 | I would like to ask a question. | Faltava `to` antes do verbo `ask`. |
| 5.1 | Could you help me with this error? | Resposta modelo; deve pedir ajuda de forma educada. |
| 5.2 | I would like to join the meeting. | Resposta modelo; deve usar `would like to`. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | Could you help me, please? | `Could` com `please` soa educado. |
| 2 | to | A estrutura é `would like to` antes de verbo. |
| 3 | Could you send the link? | Pedido educado e correto para enviar link. |
| 4 | Can | `Can I ask...` pede permissão para perguntar. |
| 5 | I would like to ask about the ticket. | Tradução direta e natural. |
| 6 | Could you check this line? | `Could you` transforma comando em pedido. |
| 7 | I would like to ask a question. | Ordem correta da estrutura com `would like to`. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | Mina is in a team call. | A primeira frase informa o contexto. |
