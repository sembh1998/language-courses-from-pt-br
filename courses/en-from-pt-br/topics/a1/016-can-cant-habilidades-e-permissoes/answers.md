# Respostas: Can/can't para habilidades e permissões

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | I can fix it. | Depois de `can`, o verbo fica na forma básica: `fix`. |
| 1.2 | Can you join? | Em perguntas, `can` vem antes do sujeito. |
| 1.3 | We can't deploy now. | `Can't` indica que não é possível ou permitido. |
| 2.1 | access | Depois de `can`, use `access`, sem `s`. |
| 2.2 | edit | Depois de `can't`, use o verbo base. |
| 2.3 | help | `Can you help me?` é pedido comum de ajuda. |
| 3.1 | I can test the app. | `Can` expressa capacidade ou possibilidade. |
| 3.2 | We can't deploy today. | `Can't deploy` expressa impossibilidade. |
| 3.3 | Can you help me? | Pergunta com `Can you...` para pedir ajuda. |
| 4.1 | She can access the system. | Não use `accesses` depois de `can`. |
| 4.2 | We can't delete files. | `Can't` já é negativo; não use `don't can't`. |
| 5.1 | I can fix bugs. | Resposta modelo; deve usar `can` com verbo base. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | He can open the file. | Depois de `can`, usamos `open`, sem `to` e sem `s`. |
| 2 | Can you help me? | Ordem correta da pergunta: `Can + sujeito + verbo`. |
| 3 | The guest can't edit it. | `Can't edit` mostra falta de permissão. |
| 4 | test | Depois de `can`, o verbo fica na forma básica. |
| 5 | She can access the report. | `Can access` expressa permissão ou capacidade. |
| 6 | I can't open the dashboard. | `Can't open` expressa impossibilidade de abrir. |
| 7 | Can you join the meeting? | Não use `to` depois de `can`. |
| 8 | I can't delete this file. | Resposta modelo com `can't delete`. |
