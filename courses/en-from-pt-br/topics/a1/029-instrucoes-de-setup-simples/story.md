# História: Instruções de setup simples

Nível: A1

## História em inglês

Rafa starts a new project with Ana.
Ana writes four simple steps.
First, install Node.
Then, open the terminal.
Next, clone the repository.
Finally, run the command.
Rafa follows the steps.
The app starts.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Rafa starts a new project with Ana. | Rafa começa um novo projeto com Ana. |
| Ana writes four simple steps. | Ana escreve quatro passos simples. |
| First, install Node. | Primeiro, instale o Node. |
| Then, open the terminal. | Depois, abra o terminal. |
| Next, clone the repository. | Em seguida, clone o repositório. |
| Finally, run the command. | Finalmente, rode o comando. |
| Rafa follows the steps. | Rafa segue os passos. |
| The app starts. | O app inicia. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| steps | passos | Instruções em ordem. |
| follows | segue | Aqui significa seguir instruções. |
| starts | inicia | Pode ser começar ou iniciar. |

## Perguntas de compreensão
