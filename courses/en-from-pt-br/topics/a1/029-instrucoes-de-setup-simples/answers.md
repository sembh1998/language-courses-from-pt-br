# Respostas: Instruções de setup simples

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Abra o terminal | `Open` significa abrir, e `terminal` é terminal. |
| 1.2 | clone the repository | `Clone` é o verbo usado para clonar um repositório. |
| 1.3 | finally | `Finally` marca o último passo de uma sequência. |
| 2.1 | install | A instrução correta é `Install Node`. |
| 2.2 | open | Com terminal, a ação natural é `open`. |
| 2.3 | run | Em contexto técnico, `run` significa rodar/executar. |
| 3.1 | First, install Node. | `First` introduz o primeiro passo. |
| 3.2 | Next, clone the repository. | `Next` indica o próximo passo, e `repository` é repositório. |
| 3.3 | Finally, run the command. | `Finally` indica o fim da sequência. |
| 4.1 | First, install Node. Then, open the terminal. Finally, run the command. | A sequência usa primeiro, depois e finalmente. |
| 4.2 | Clone the repository. | No imperativo, o verbo vem no começo. |
| 5.1 | First, open the terminal. Then, run the command. | Resposta modelo; deve haver dois passos com `first` e `then`. |
