# História: Adjetivos básicos para tecnologia

Nível: A1

## História em inglês

Joao has a new tool at work. The tool is simple and fast. His old system is slow. Today, he has a hard bug and a complex report. The report is private, but the help page is public.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Joao has a new tool at work. | João tem uma ferramenta nova no trabalho. |
| The tool is simple and fast. | A ferramenta é simples e rápida. |
| His old system is slow. | O sistema antigo dele é lento. |
| Today, he has a hard bug and a complex report. | Hoje, ele tem um bug difícil e um relatório complexo. |
| The report is private, but the help page is public. | O relatório é privado, mas a página de ajuda é pública. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| new tool | ferramenta nova | Adjetivo antes do substantivo. |
| hard bug | bug difícil | `Hard` descreve dificuldade. |
| public | público | Oposto de `private`. |

## Perguntas de compreensão

1. What does Joao have at work?
2. Is the tool fast?
3. What is slow?
4. Is the report public or private?
