# Respostas: Adjetivos básicos para tecnologia

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | fast | `Fast` significa rápido. |
| 1.2 | private | `Private` significa privado. |
| 1.3 | a new tool | Em inglês, o adjetivo vem antes do substantivo. |
| 2.1 | easy | `Easy` significa fácil. |
| 2.2 | complex | `Complex problem` significa problema complexo. |
| 2.3 | safe | `Safe password` ou `password is safe` indica segurança. |
| 3.1 | a slow system | `Slow` vem antes de `system`. |
| 3.2 | a simple tool | A ordem natural é adjetivo + substantivo. |
| 3.3 | The file is public. | Quando o adjetivo vem depois de `is`, não há substantivo depois dele. |
| 4.1 | an old server | Use `an` antes de som de vogal em `old`. |
| 4.2 | private files | Adjetivos não recebem plural em inglês. |
| 5.1 | The app is slow. | Resposta modelo com `slow`. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | safe | `Safe` significa seguro. |
| 2 | a simple page | A ordem correta é adjetivo antes do substantivo. |
| 3 | hard | `Hard` é uma forma básica de dizer difícil. |
| 4 | old | `Old server` significa servidor antigo. |
| 5 | This is a complex problem. | `Complex` vem antes de `problem`. |
| 6 | The files are private. | Com plural, use `are`; o adjetivo não muda. |
| 7 | fast apps | Adjetivos em inglês não recebem plural. |
| 8 | This is a new dashboard. | Resposta modelo com `new dashboard`. |
