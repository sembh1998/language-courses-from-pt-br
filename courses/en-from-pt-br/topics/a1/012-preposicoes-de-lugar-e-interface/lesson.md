# Preposições de lugar e interface

Nível: A1

## Objetivo

Você vai localizar objetos, elementos de tela e arquivos usando preposições básicas: `in`, `on`, `at`, `under`, `next to` e `between`.

## Explicação

Preposições mostram onde algo está.

| Inglês | Tradução | Observação |
|---|---|---|
| in the folder | na pasta | Dentro de um espaço. |
| on the page | na página | Sobre uma superfície ou tela. |
| at the top | no topo | Ponto específico. |
| under the table | embaixo da mesa | Abaixo de algo. |
| next to the button | ao lado do botão | Perto, do lado. |
| between two icons | entre dois ícones | No meio de dois itens. |

Em tecnologia, `in` é comum para pastas, arquivos e sistemas. `On` é comum para páginas, telas e sites. `At` aparece em pontos da interface: `at the top`, `at the bottom`.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| The file is in the folder. | O arquivo está na pasta. | Local digital. |
| The button is on the page. | O botão está na página. | Elemento de interface. |
| The title is at the top. | O título está no topo. | Posição específica. |
| The cable is under the desk. | O cabo está embaixo da mesa. | Ambiente físico. |
| The icon is next to the search bar. | O ícone está ao lado da barra de busca. | Interface. |
| The menu is between the logo and the button. | O menu está entre o logo e o botão. | Dois pontos de referência. |

## Erros comuns

1. Usar `in` para página: prefira `on the page`.

2. Usar `on` para pasta: prefira `in the folder`.

3. Confundir `at the top` com `on the top`; para interface, `at the top` é mais natural.

## Prática guiada

Olhe para uma tela e diga três posições: `The button is on the page`, `The title is at the top`, `The file is in the folder`.

## Resumo

Use `in` para dentro, `on` para tela ou superfície, `at` para ponto específico, `under` para abaixo, `next to` para ao lado e `between` para entre dois itens.
