# História: Preposições de lugar e interface

Nível: A1

## História em inglês

Nina opens the project page. The logo is at the top. The search bar is next to the menu. A blue button is on the page. The help icon is between two links. The main file is in the folder.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Nina opens the project page. | Nina abre a página do projeto. |
| The logo is at the top. | O logo está no topo. |
| The search bar is next to the menu. | A barra de busca está ao lado do menu. |
| A blue button is on the page. | Um botão azul está na página. |
| The help icon is between two links. | O ícone de ajuda está entre dois links. |
| The main file is in the folder. | O arquivo principal está na pasta. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| project page | página do projeto | Usa `on` para elementos na página. |
| help icon | ícone de ajuda | Elemento de interface. |
| main file | arquivo principal | Usa `in the folder`. |

## Perguntas de compreensão

1. Where is the logo?
2. Where is the search bar?
3. What is on the page?
4. Where is the main file?
