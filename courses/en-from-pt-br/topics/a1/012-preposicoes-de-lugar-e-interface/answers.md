# Respostas: Preposições de lugar e interface

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | in | Arquivos ficam `in the folder`, dentro da pasta. |
| 1.2 | on | Elementos aparecem `on the page`, na página. |
| 1.3 | at | Para ponto específico da interface, usamos `at the top`. |
| 2.1 | under | `Under` significa embaixo de. |
| 2.2 | next | A expressão completa é `next to`. |
| 2.3 | between | `Between` significa entre dois itens. |
| 3.1 | The file is in the folder. | `In` indica dentro da pasta. |
| 3.2 | The button is on the page. | `On` é natural para página ou tela. |
| 3.3 | The menu is at the top. | `At the top` indica posição no topo. |
| 4.1 | The file is in the folder. | Pasta pede `in`, não `on`. |
| 4.2 | The button is on the page. | Página pede `on`, não `in`. |
| 5.1 | The icon is next to the button. | Resposta modelo com `next to`. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | The image is in the folder. | Dentro de pasta é `in the folder`. |
| 2 | The link is on the page. | Elemento em página é `on the page`. |
| 3 | between two fields | `Between` significa entre. |
| 4 | at | `At the bottom` é ponto específico da interface. |
| 5 | The button is next to the field. | `Next to` significa ao lado de. |
| 6 | The laptop is under the desk. | `Under` significa embaixo de. |
| 7 | The title is at the top. | Use `at the top`, não `in the top`. |
| 8 | The menu is between the logo and the button. | Resposta modelo; precisa usar `between` com dois pontos de referência. |
