# Gabarito: Revisão integrada A1 para software engineers

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | I am a software engineer. | `I am a` apresenta cargo corretamente. |
| 1.2 | Can you help me? | Essa é uma pergunta para pedir ajuda. |
| 1.3 | Check the logs. | Imperativo começa direto com o verbo. |
| 2.1 | am | `I am` indica estado ou identidade. |
| 2.2 | work | `I work with` descreve stack ou atividade. |
| 2.3 | help | `Can you help me?` é pedido básico de ajuda. |
| 3.1 | Run the command. | Imperativo usado em instrução técnica. |
| 3.2 | The build failed. | `Failed` indica que a build falhou. |
| 4.1 | I work with JavaScript. | Não use `am` antes de `work` no presente simples. |
| 4.2 | Can you help me? | Perguntas com `can` começam com `Can`. |
| 5.1 | I am a developer. I work with React. Can you help me? | Resposta modelo com apresentação, stack e pedido de ajuda. |

## Gabarito do teste
