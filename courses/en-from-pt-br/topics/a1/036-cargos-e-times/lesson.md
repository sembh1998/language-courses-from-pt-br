# Vocabulário de cargos e times

Nível: A1

## Objetivo

Neste tópico, você vai falar sobre cargos e pessoas de um time de tecnologia em inglês. O foco é explicar quem faz o quê com frases simples, como `A developer writes code` e `QA tests the app`.

## Explicação

Para explicar funções em um time, use frases curtas no presente simples:

`pessoa ou cargo + verbo + complemento`

Exemplos:

- `A developer writes code.`
- `A designer creates screens.`
- `Support helps customers.`

Com cargos no singular, use `a` ou `an` antes do cargo quando a frase fala de uma pessoa qualquer: `a developer`, `a designer`, `an engineer`. Quando você fala de uma área ou função como grupo, pode usar o nome sem artigo: `QA tests the app`, `DevOps manages servers`.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| A developer writes code. | Um desenvolvedor escreve código. | Frase simples com cargo e ação. |
| An engineer designs systems. | Um engenheiro projeta sistemas. | Use `an` antes de som de vogal. |
| A designer creates screens. | Um designer cria telas. | `creates` tem `s` por causa de `a designer`. |
| A product manager plans the roadmap. | Um gerente de produto planeja o roadmap. | Cargo comum em produto. |
| QA tests the app. | QA testa o aplicativo. | `QA` pode funcionar como área. |
| DevOps manages the deploy. | DevOps gerencia o deploy. | Frase simples sobre operação. |
| Support helps the customer. | O suporte ajuda o cliente. | `customer` é cliente. |
| A stakeholder gives feedback. | Um stakeholder dá feedback. | Pessoa interessada no projeto. |

## Erros comuns

1. Esquecer o artigo com cargo singular.

Forma natural: `She is a developer.`

Evite: `She is developer.`

2. Confundir `customer` com `client` em contexto de produto.

`Customer` é muito usado para cliente ou usuário pagante de um produto.

3. Esquecer o `s` no verbo com cargo singular.

Forma correta: `A designer creates screens.`

## Prática guiada

Monte frases com este padrão:

- `A developer writes code.`
- `A QA engineer tests the app.`
- `Support helps customers.`
- `A stakeholder gives feedback.`

## Resumo

Para falar de cargos e times em inglês A1, combine cargo + verbo + complemento. Use frases curtas para explicar responsabilidades: escrever código, testar, criar telas, ajudar clientes e dar feedback.
