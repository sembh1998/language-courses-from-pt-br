# Gabarito: Vocabulário de cargos e times

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | a developer | Desenvolvedores geralmente escrevem código. |
| 1.2 | a product manager | O product manager costuma planejar roadmap e prioridades. |
| 1.3 | a designer | Designers geralmente criam telas e interfaces. |
| 2.1 | developer | `A developer writes code` é a frase padrão. |
| 2.2 | customers | O suporte ajuda clientes. |
| 2.3 | tests | `QA` pode ser tratado como área que testa o app. |
| 3.1 | A designer creates screens. | Com cargo singular, use `a`; com `designer`, o verbo recebe `s`. |
| 3.2 | A stakeholder gives feedback. | `Gives` recebe `s` por causa do sujeito singular. |
| 4.1 | She is a developer. | Cargo singular depois de `is` precisa de artigo. |
| 4.2 | A developer writes code. | No singular, `write` vira `writes`. |
| 5.1 | DevOps manages the deploy. | Resposta modelo; deve explicar uma função de DevOps em frase simples. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | He is an engineer. | Use `an` antes de som de vogal em `engineer`. |
| 2 | QA | QA é associado a testes de funcionalidades. |
| 3 | customer | `Customer` significa cliente. |
| 4 | plans | Com `A product manager`, o verbo recebe `s`. |
| 5 | Support helps the customer. | A frase usa `support`, `helps` e `customer` corretamente. |
| 6 | A developer writes code. | Resposta modelo; deve ligar desenvolvedor a código. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | Maria is a developer. | A história apresenta o cargo dela. |
| 2 | Ben creates screens. | Ben é o designer. |
| 3 | Support helps customers. | Essa frase aparece diretamente na história. |
