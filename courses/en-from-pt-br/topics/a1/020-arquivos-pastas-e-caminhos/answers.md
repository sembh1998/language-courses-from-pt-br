# Gabarito: Arquivos, pastas e caminhos

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | The file is in the root folder. | `File` é arquivo e `root folder` é pasta raiz. |
| 1.2 | path | `Path` significa caminho. |
| 1.3 | Where is the file? | Pergunta de localização usa `Where is...?`. |
| 2.1 | in | O arquivo está dentro da pasta. |
| 2.2 | under | `Under` indica abaixo na hierarquia. |
| 2.3 | at | `At` combina com caminho específico. |
| 3.1 | Open the test folder. | Comando simples com `Open`. |
| 3.2 | The path is correct. | Frase com sujeito + `is` + adjetivo. |
| 3.3 | Update the config file. | `Config file` é arquivo de configuração. |
| 4.1 | The file is in the folder. | Faltava o verbo `is`. |
| 4.2 | Where is the path? | Em perguntas, `is` vem antes do sujeito. |
| 5.1 | The file is in src. | Resposta-modelo de localização. |
