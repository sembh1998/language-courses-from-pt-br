# Respostas: Pedir repetição, esclarecimento e ajuda

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Can you repeat? | Essa frase pede repetição de forma clara. |
| 1.2 | Can you speak slowly? | `Speak slowly` significa falar devagar. |
| 1.3 | What does it mean? | Essa estrutura pergunta significado. |
| 2.1 | repeat | `Can you repeat?` é um pedido de repetição. |
| 2.2 | slowly | `Slowly` é o advérbio para “devagar”. |
| 2.3 | understand | `I don't understand` significa “eu não entendo”. |
| 2.4 | help | `Can you help me?` pede ajuda. |
| 3.1 | Can you repeat? | Tradução direta e natural para reunião. |
| 3.2 | I don't understand this error. | `This error` significa “este erro”. |
| 3.3 | Can you write it in the chat? | Frase útil quando a fala não ficou clara. |
| 4.1 | What does deploy mean? | A ordem correta é `What does X mean?`. |
| 4.2 | Can you repeat, please? | `Can you... please?` deixa o pedido mais educado. |
| 4.3 | Can you help me with this error? | O pedido fica específico com `with this error`. |
| 5.1 | Can you speak slowly, please? | Resposta modelo; deve pedir fala mais devagar. |
| 5.2 | What does branch mean? | Resposta modelo; deve seguir `What does X mean?`. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | Sorry? | `Sorry?` pode pedir repetição de forma curta. |
| 2 | speak | A expressão correta é `speak slowly`. |
| 3 | What does merge mean? | Essa é a ordem correta para perguntar significado. |
| 4 | understand | Completa `I don't understand`. |
| 5 | Can you help me with this task? | `Help me with` é a combinação correta. |
| 6 | I need help with this error. | A frase especifica qual é o problema. |
| 7 | Can you write it in the chat? | A ordem natural começa com `Can you`. |
| 8 | One moment, please. | Resposta modelo; a frase dá tempo com educação. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | Joao is in an online meeting. | A primeira frase informa o contexto. |
| 2 | Yes, the manager speaks fast. | Esse é o motivo do pedido de João. |

| 5 | No, he does not understand the error. | Ele diz `I don't understand this error`. |
