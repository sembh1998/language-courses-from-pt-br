# Vocabulário de computador e ferramentas

Nível: A1

## Objetivo

Nomear ferramentas básicas de trabalho em inglês e reconhecer substantivos contáveis e incontáveis simples.

## Explicação

Use `a` ou `an` com um item contável singular: `a laptop`, `a monitor`, `an editor`. Use plural com `s`: `two monitors`, `three tickets`. Alguns nomes funcionam como ideia geral e normalmente não usam `a`: `software`, `code`, `information`.

Para dizer que algo existe ou está disponível, use frases simples: `I have a laptop`, `We use an IDE`, `The repository is private`.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| I use a laptop. | Eu uso um laptop. | `Laptop` é contável. |
| She has two monitors. | Ela tem dois monitores. | Plural com `s`. |
| We open the terminal. | Nós abrimos o terminal. | Ferramenta de linha de comando. |
| The repository is private. | O repositório é privado. | Substantivo contável singular. |
| This software is useful. | Este software é útil. | `Software` não usa `a` neste uso geral. |

## Erros comuns

Evite `a software` quando falar de software em geral. Diga `software` ou `a software tool`.

Evite esquecer o plural: `two monitor` deve ser `two monitors`.

Evite trocar `browser` por `browse`: `browser` é o navegador; `browse` é o verbo navegar.

## Prática guiada

Separe mentalmente: objetos contáveis (`laptop`, `monitor`, `ticket`) e ideias/materiais gerais (`software`, `code`, `information`).

## Resumo

Vocabulário de ferramentas permite falar do ambiente de trabalho. Use `a/an` com itens contáveis singulares e cuidado com palavras gerais como `software` e `code`.
