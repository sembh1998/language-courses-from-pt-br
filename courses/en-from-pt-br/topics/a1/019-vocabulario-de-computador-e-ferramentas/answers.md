# Gabarito: Vocabulário de computador e ferramentas

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | browser | `Browser` significa navegador. |
| 1.2 | She has two monitors. | Com `two`, use plural: `monitors`. |
| 1.3 | I use software. | `Software` em geral não usa `a`. |
| 2.1 | terminal | O terminal é usado para comandos. |
| 2.2 | board | O board contém tickets. |
| 2.3 | repository | Repositório pode ficar no GitHub. |
| 2.4 | editor | Escrevemos código em um editor. |
| 3.1 | I use a laptop. | `Laptop` é contável singular, então usa `a`. |
| 3.2 | The pipeline is green. | `Green` indica status positivo. |
| 3.3 | This ticket is urgent. | `This` indica este/esta. |
| 4.1 | two monitors | Plural regular com `s`. |
| 4.2 | three tickets | `Ticket` recebe `s` no plural. |
| 5.1 | I use a browser. | Resposta-modelo com ferramenta contável. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | an editor | Antes de som de vogal, use `an`. |
| 2 | board | `Board` é quadro de tarefas. |
| 3 | The code is clean. | `Code` é tratado como singular/incontável aqui. |
| 4 | pipeline | Pipeline pode estar rodando. |
| 5 | monitors | Com `two`, use plural. |
| 6 | The repository is private. | `Repository` significa repositório. |
| 7 | We use an IDE. | `IDE` começa com som de vogal, então use `an`. |
| 8 | I use software. | Retire `a` antes de `software` geral. |
| 9 | I use a laptop. | Resposta-modelo com uma ferramenta. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | She has two monitors. | A história diz `two monitors`. |
| 2 | She opens the browser and the editor. | Essas são as duas primeiras ferramentas. |
| 3 | The repository is private. | A história usa `private`. |
| 4 | She sees a new ticket. | O ticket está no board. |
| 5 | The pipeline is green. | Verde indica status bom. |
