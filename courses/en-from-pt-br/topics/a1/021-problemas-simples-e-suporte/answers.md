# Gabarito: Problemas simples e suporte

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | It doesn't work. | Com `it`, a negativa usa `doesn't`. |
| 1.2 | I have an error. | Use `an` antes de `error`. |
| 1.3 | Can you help me? | Pedido claro e educado. |
| 2.1 | slow | `Slow` significa lento/lenta. |
| 2.2 | reset | `Password reset` é redefinição de senha. |
| 2.3 | problem | `Login problem` é problema de login. |
| 3.1 | The page is slow. | Frase simples com `is`. |
| 3.2 | I need support. | `Need` significa precisar. |
| 3.3 | Can you reset my password? | Pergunta educada com `Can you`. |
| 4.1 | It doesn't work. | A forma `don't` não combina com `it`. |
| 4.2 | I need help with this error. | Use `with` para indicar o problema. |
| 5.1 | I have an error. Can you help me? | Resposta-modelo clara para suporte. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | The link doesn't open. | `The link` é singular, então a negativa usa `doesn't`. |
| 2 | I have a login problem. | Ordem natural: `a login problem`. |
| 3 | Can you reset my password? | Pedido claro com `Can you`. |
| 4 | message | `Error message` é mensagem de erro. |
| 5 | try | `Try again` significa tentar de novo. |
| 6 | The button doesn't work. | `Button` singular pode ser retomado com `doesn't`. |
| 7 | I need help with this error. | Use `with this error` para especificar. |
| 8 | I have an error. | Antes de `error`, use `an`. |
| 9 | Yes, I can. | Resposta curta correta para `Can you...?`. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | The login page is slow. | A história cita a página de login. |
| 2 | No, it doesn't work. | O botão não funciona. |
| 3 | She sends a message to support. | Bruna procura suporte. |
| 4 | Support asks her to try again. | A resposta é `Please try again`. |
| 5 | She needs a password reset. | Esse é o problema final. |
