# Problemas simples e suporte

Nível: A1

## Objetivo

Pedir ajuda em inglês com frases simples sobre erro, lentidão, login e redefinição de senha.

## Explicação

Para relatar problema, use frases diretas: `It doesn't work`, `I have an error`, `The page is slow`. Para pedir ajuda, use `Can you help me?` ou `I need help with...`.

Com `it`, a negativa no presente simples usa `doesn't`: `It doesn't work`. Para falar de um problema específico, use `with`: `I have a problem with login`.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| It doesn't work. | Não funciona. | Frase curta e útil. |
| I have an error. | Eu tenho um erro. | Use `an` antes de `error`. |
| The page is slow. | A página está lenta. | Relato claro de lentidão. |
| I have a login problem. | Eu tenho um problema de login. | Problema específico. |
| Can you reset my password? | Você pode redefinir minha senha? | Pedido educado. |

## Erros comuns

Evite `It don't work`. Com `it`, use `doesn't`.

Evite mensagens vagas como `Problem`. Diga o que acontece: `I have an error on the login page`.

## Prática guiada

Use este modelo: problema + lugar + pedido. Exemplo: `I have an error on the login page. Can you help me?`

## Resumo

Para suporte básico, seja claro e curto: diga o problema, onde acontece e peça ajuda com educação.
