# História: Problemas simples e suporte

Nível: A1

## História em inglês

Bruna opens the app in the morning.

The login page is slow.

She clicks the button, but it doesn't work.

Bruna sends a message to support.

She writes, "I have an error on the login page. Can you help me?"

Support answers, "Please try again."

Bruna tries again, but she needs a password reset.

Support resets the password, and the app works.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Bruna opens the app in the morning. | Bruna abre o aplicativo de manhã. |
| The login page is slow. | A página de login está lenta. |
| She clicks the button, but it doesn't work. | Ela clica no botão, mas ele não funciona. |
| Bruna sends a message to support. | Bruna envia uma mensagem para o suporte. |
| She writes, "I have an error on the login page. Can you help me?" | Ela escreve: "Eu tenho um erro na página de login. Você pode me ajudar?" |
| Support answers, "Please try again." | O suporte responde: "Por favor, tente de novo." |
| Bruna tries again, but she needs a password reset. | Bruna tenta de novo, mas precisa de uma redefinição de senha. |
| Support resets the password, and the app works. | O suporte redefine a senha, e o aplicativo funciona. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| `slow` | lento | Problema de desempenho. |
| `doesn't work` | não funciona | Com `it`, use `doesn't`. |
| `support` | suporte | Equipe ou pessoa que ajuda. |
| `try again` | tentar de novo | Pedido comum. |
| `password reset` | redefinição de senha | Ajuda comum de login. |

## Perguntas de compreensão

1. Qual página está lenta?
2. O botão funciona?
3. Para quem Bruna envia uma mensagem?
4. O que o suporte pede primeiro?
5. Do que Bruna precisa no final?
