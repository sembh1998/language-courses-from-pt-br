# Comparativos simples

Nível: A1

## Objetivo

Neste tópico, você vai comparar ferramentas, telas e soluções com frases curtas em inglês. O foco é usar comparativos simples como `faster`, `slower`, `easier`, `harder`, `better` e `worse`, sempre com `than` quando houver comparação direta.

## Explicação

Para comparar duas coisas em inglês, use esta estrutura:

`coisa 1 + is + comparativo + than + coisa 2`

Exemplos:

- `This app is faster than the old app.`
- `The new screen is easier than the old screen.`
- `This solution is better than that solution.`

Muitos adjetivos curtos recebem `-er`: `fast` vira `faster`, `slow` vira `slower`, `easy` vira `easier` e `hard` vira `harder`. Dois comparativos importantes são irregulares: `good` vira `better` e `bad` vira `worse`.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| This tool is faster than the old tool. | Esta ferramenta é mais rápida do que a ferramenta antiga. | `faster than` compara velocidade. |
| The login screen is easier than the settings screen. | A tela de login é mais fácil do que a tela de configurações. | `easier` troca `y` por `ier`. |
| This task is harder than the last task. | Esta tarefa é mais difícil do que a última tarefa. | `harder` pode falar de dificuldade. |
| The new fix is better than the old fix. | A nova correção é melhor do que a correção antiga. | `better` é irregular. |
| This bug is worse than the warning. | Este bug é pior do que o aviso. | `worse` é irregular. |
| The dashboard is slower than the report page. | O painel é mais lento do que a página de relatório. | `slower than` compara desempenho. |

## Erros comuns

1. Esquecer `than` em uma comparação direta.

Forma correta: `This tool is faster than that tool.`

Evite: `This tool is faster that tool.`

2. Usar `more` com comparativos curtos deste tópico.

Forma natural: `This screen is easier.`

Evite: `This screen is more easy.`

3. Regularizar `better` e `worse`.

Forma correta: `better`, `worse`.

Evite: `gooder`, `badder`.

## Prática guiada

Compare duas opções do seu trabalho:

- Uma ferramenta rápida: `This tool is faster than the old tool.`
- Uma tela difícil: `This screen is harder than the home screen.`
- Uma solução boa: `This solution is better than the first idea.`

## Resumo

Use `is + comparativo + than` para comparar duas coisas. Em tecnologia, esses comparativos ajudam a falar de velocidade, facilidade, dificuldade e qualidade de ferramentas, telas e soluções.
