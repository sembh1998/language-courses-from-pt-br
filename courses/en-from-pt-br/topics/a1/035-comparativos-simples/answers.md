# Gabarito: Comparativos simples

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | faster | `Faster` significa mais rápido. |
| 1.2 | better | `Good` tem comparativo irregular: `better`. |
| 1.3 | than | `Than` liga as duas partes da comparação. |
| 2.1 | faster | A frase compara a página com uma página antiga. |
| 2.2 | easier | `Easy` vira `easier` no comparativo. |
| 2.3 | worse | `Worse` significa pior. |
| 3.1 | This tool is slower than that tool. | A estrutura correta é `is + comparativo + than`. |
| 3.2 | The new solution is better than the old solution. | `Better than` expressa melhor do que. |
| 4.1 | This task is harder than the last task. | A frase compara a dificuldade das duas tarefas. |
| 4.2 | This screen is easier than the old screen. | Para `easy`, use `easier`, não `more easy`. |
| 5.1 | This tool is better than the old tool. | Resposta modelo; outra comparação correta com `better than` também serve. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | The new site is faster than the old site. | A frase usa `faster than` corretamente. |
| 2 | harder | `Hard` vira `harder` no comparativo. |
| 3 | worse | `Worse` é o comparativo de `bad`. |
| 4 | than | Comparações diretas pedem `than`. |
| 5 | This fix is better than the first fix. | `Better than` é a forma correta para melhor do que. |
| 6 | The settings screen is slower than the home screen. | Resposta modelo; deve comparar duas telas com `slower than`. |
