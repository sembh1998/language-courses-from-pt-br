# História: Comparativos simples

Nível: A1

## História em inglês

Ana tests two tools at work. The new tool is faster than the old tool. The setup is easier than the first setup. One screen is slower than the dashboard, but the new solution is better. Ana writes a short note: "This version is better than the old version."

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Ana tests two tools at work. | Ana testa duas ferramentas no trabalho. |
| The new tool is faster than the old tool. | A nova ferramenta é mais rápida do que a ferramenta antiga. |
| The setup is easier than the first setup. | A configuração é mais fácil do que a primeira configuração. |
| One screen is slower than the dashboard, but the new solution is better. | Uma tela é mais lenta do que o painel, mas a nova solução é melhor. |
| Ana writes a short note: "This version is better than the old version." | Ana escreve uma nota curta: "Esta versão é melhor do que a versão antiga." |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| tests | testa | Verbo no presente simples com `s` para `Ana`. |
| two tools | duas ferramentas | Vocabulário comum em tecnologia. |
| version | versão | Útil para produto e software. |
| better than | melhor do que | Comparativo irregular com `than`. |

## Perguntas de compreensão

1. Qual ferramenta é mais rápida?
2. O que é mais fácil?
3. A nova solução é melhor ou pior?
