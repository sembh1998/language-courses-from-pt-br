# História: Projeto A1: primeiro dia em um time internacional

Nível: A1

## História em inglês

Today is Bruno's first day on an international team. He writes, "Hi, I am Bruno. I am a software developer from Brazil." His teammate sends instructions: "Open the repository. Install the packages. Run the tests." Bruno opens the project, but the install fails. He sends an update: "I read the README. The install failed. Can you help me?"

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Today is Bruno's first day on an international team. | Hoje é o primeiro dia de Bruno em um time internacional. |
| He writes, "Hi, I am Bruno. I am a software developer from Brazil." | Ele escreve: "Oi, eu sou Bruno. Eu sou desenvolvedor de software do Brasil." |
| His teammate sends instructions: "Open the repository. Install the packages. Run the tests." | O colega dele envia instruções: "Abra o repositório. Instale os pacotes. Rode os testes." |
| Bruno opens the project, but the install fails. | Bruno abre o projeto, mas a instalação falha. |
| He sends an update: "I read the README. The install failed. Can you help me?" | Ele envia um update: "Eu li o README. A instalação falhou. Você pode me ajudar?" |

## Vocabulário da história

| Inglês | Tradução | Observação |
