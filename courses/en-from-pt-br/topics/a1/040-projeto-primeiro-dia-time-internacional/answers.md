# Gabarito: Projeto A1: primeiro dia em um time internacional

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Hi, I am new on the team. | É uma apresentação adequada para o primeiro dia. |
| 1.2 | Install the packages. | É uma instrução de setup em imperativo. |
| 1.3 | Can you help me? | Essa pergunta pede ajuda. |
| 2.1 | day | `First day` significa primeiro dia. |
| 2.2 | with | A expressão correta é `help with`. |
| 2.3 | failed | `The setup failed` indica que a configuração falhou. |
| 3.1 | Open the repository. | Tradução direta de instrução técnica. |
| 3.2 | I need help with the error. | Frase curta e clara para pedir ajuda. |
| 4.1 | O pacote não foi encontrado. | `Not found` significa não encontrado. |
| 4.2 | A chave de API é obrigatória. | `Required` indica algo obrigatório. |
| 5.1 | I read the README. I installed the packages. I need help with the tests. | Resposta modelo; o update deve ter três frases claras. |
