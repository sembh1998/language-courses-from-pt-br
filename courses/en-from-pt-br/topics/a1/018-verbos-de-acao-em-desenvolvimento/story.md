# História: Verbos de ação em desenvolvimento

Nível: A1

## História em inglês

Rafa works on a small team.

In the morning, he reads the ticket.

Then he writes code and fixes a bug.

Ana reviews the code after lunch.

Rafa pulls the latest changes and builds the app.

The team tests the login page.

At the end of the day, Rafa pushes his changes.

He does not merge the branch today.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Rafa works on a small team. | Rafa trabalha em uma equipe pequena. |
| In the morning, he reads the ticket. | De manhã, ele lê o ticket. |
| Then he writes code and fixes a bug. | Então ele escreve código e corrige um bug. |
| Ana reviews the code after lunch. | Ana revisa o código depois do almoço. |
| Rafa pulls the latest changes and builds the app. | Rafa puxa as alterações mais recentes e gera o build do app. |
| The team tests the login page. | A equipe testa a página de login. |
| At the end of the day, Rafa pushes his changes. | No fim do dia, Rafa envia suas alterações. |
| He does not merge the branch today. | Ele não faz merge da branch hoje. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| `reads` | lê | Forma com `s` para `he`. |
| `writes` | escreve | Verbo de ação comum. |
| `fixes` | corrige | `fix` vira `fixes` com `he/she`. |
| `pulls` | puxa alterações | Ação comum no Git. |
| `pushes` | envia alterações | `push` vira `pushes`. |

## Perguntas de compreensão

1. O que Rafa lê de manhã?
2. Que problema Rafa corrige?
3. Quem revisa o código?
4. O que a equipe testa?
5. Rafa faz merge da branch hoje?
