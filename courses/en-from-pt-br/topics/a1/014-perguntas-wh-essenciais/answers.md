# Respostas: Perguntas WH essenciais

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | where | `Where` pergunta lugar. |
| 1.2 | who | `Who` pergunta pessoa. |
| 1.3 | why | `Why` pergunta motivo. |
| 2.1 | When | A resposta `At ten` indica horário. |
| 2.2 | Where | A resposta `In the folder` indica lugar. |
| 2.3 | Why | A resposta com `because` indica motivo. |
| 3.1 | What is the problem? | `What` pergunta qual coisa ou informação. |
| 3.2 | Who is the owner? | `Who` é usado para pessoa. |
| 3.3 | How is the app? | `How` pergunta como está algo. |
| 4.1 | In the folder. | Resposta curta adequada para `where`. |
| 4.2 | At three. | Resposta curta adequada para `when`. |
| 5.1 | What is the task? | Resposta modelo com `what` e `task`. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | When is the demo? | `When` pergunta tempo ou horário. |
| 2 | how | `How` significa como. |
| 3 | Where is the ticket? | A ordem correta é `Where + is + sujeito`. |
| 4 | Who | A resposta `Ana` é uma pessoa. |
| 5 | Where is the bug? | `Where` pergunta lugar. |
| 6 | Why did the test fail? | `Why` pergunta motivo; esta forma é comum para falha passada. |
| 7 | It is a dashboard. | Resposta modelo para `What is this?`. |
| 8 | How is the system? | Resposta modelo com `how`. |
