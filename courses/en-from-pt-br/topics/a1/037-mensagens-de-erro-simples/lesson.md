# Leitura de mensagens de erro simples

Nível: A1

## Objetivo

Neste tópico, você vai ler mensagens de erro simples em inglês e identificar a ação sugerida. O foco é entender palavras como `error`, `warning`, `failed`, `missing`, `invalid`, `required` e `not found`.

## Explicação

Mensagens de erro em software costumam ser curtas. Elas dizem o problema e, às vezes, indicam o que fazer.

Padrões comuns:

- `Error: file not found.`
- `Warning: password is required.`
- `Login failed.`
- `Invalid email.`
- `Missing token.`

Para explicar o erro em uma frase simples, use:

`The problem is + problema.`

ou

`The user needs to + ação.`

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| Error: file not found. | Erro: arquivo não encontrado. | `not found` indica que algo não foi localizado. |
| Warning: storage is low. | Aviso: o armazenamento está baixo. | `warning` é alerta, nem sempre bloqueia. |
| Login failed. | O login falhou. | `failed` indica que a ação não deu certo. |
| Missing email. | E-mail ausente. | `missing` indica falta de informação. |
| Invalid password. | Senha inválida. | `invalid` indica formato ou valor errado. |
| Name is required. | Nome é obrigatório. | `required` indica campo obrigatório. |

## Erros comuns

1. Traduzir `failed` como uma pessoa falhou.

Em mensagens técnicas, `failed` normalmente significa que uma ação não funcionou: `Login failed` = o login falhou.

2. Confundir `missing` com `not found`.

`Missing` costuma indicar que algo não foi enviado. `Not found` indica que o sistema procurou e não encontrou.

3. Ignorar a ação sugerida.

Em `Email is required`, a ação é informar o e-mail.

## Prática guiada

Leia a mensagem e diga o problema:

- `Invalid email.` → o e-mail é inválido.
- `Password is required.` → a senha é obrigatória.
- `File not found.` → o arquivo não foi encontrado.

## Resumo

Para ler erros simples, identifique a palavra-chave e transforme em uma frase curta. `Required` pede preencher algo; `invalid` pede corrigir; `not found` pede verificar nome, caminho ou recurso.
