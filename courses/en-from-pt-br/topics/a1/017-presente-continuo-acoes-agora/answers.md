# Gabarito: Presente contínuo: o que está acontecendo agora

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | I am debugging now. | Presente contínuo com `I` usa `am` + verbo com `-ing`. |
| 1.2 | The build is running. | `The build` é singular, então usa `is`. |
| 1.3 | Are you testing? | Perguntas no presente contínuo colocam `are` antes de `you`. |
| 2.1 | am | Com `I`, use `am`. |
| 2.2 | are | Com `they`, use `are`. |
| 2.3 | is | `The pipeline` é singular, então usa `is`. |
| 2.4 | are | Com `we`, use `are`. |
| 3.1 | We are testing the fix. | `We are` + `testing` indica ação em andamento. |
| 3.2 | She is reviewing my code. | Com `she`, use `is` + `reviewing`. |
| 3.3 | I am not deploying now. | A negativa usa `not` depois de `am`. |
| 4.1 | I am debugging the app now. | Para ação atual, troque `debug` por `am debugging`. |
| 4.2 | They are testing the page now. | Com `they`, use `are testing`. |
| 4.3 | The build is running now. | Com sujeito singular, use `is running`. |
| 5.1 | I am working on a ticket. | Resposta-modelo com presente contínuo. |
| 5.2 | My team is testing now. | `My team` pode ser tratado como singular: `is testing`. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | I am fixing an error. | Presente contínuo com `I am` + `fixing`. |
| 2 | The page is loading. | `The page` é singular, então usa `is`. |
| 3 | Are they reviewing the code? | A pergunta começa com `Are they`. |
| 4 | are | Com `we`, use `are`. |
| 5 | is | Com `he`, use `is`. |
| 6 | They are waiting for the pipeline. | `They are waiting` expressa ação em andamento. |
| 7 | I am not reviewing now. | Negativa: `am not` + verbo com `-ing`. |
| 8 | Are you testing the login? | Para pergunta, coloque `Are` no início. |
| 9 | I am debugging. | Resposta-modelo curta para uma ação acontecendo agora. |

## Respostas da história

| Pergunta | Resposta | Observação |
