# Respostas: Conjunções básicas: and, but, or, because

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | but | `But` significa mas. |
| 1.2 | because | `Because` introduz uma causa. |
| 1.3 | or | `Or` indica escolha entre opções. |
| 2.1 | and | A frase soma duas qualidades: rápido e simples. |
| 2.2 | but | Há contraste entre funcionar e ser lento. |
| 2.3 | because | A segunda parte explica a causa da falha. |
| 3.1 | The API is fast and stable. | `And` conecta duas qualidades. |
| 3.2 | Use Docker or Node. | `Or` liga alternativas. |
| 3.3 | The app fails because the config is wrong. | `Because` explica a causa do problema. |
| 4.1 | The server is small and fast. | Duas descrições podem ser unidas com `and`. |
| 4.2 | The tool is old, but it works. | `But` marca contraste. |
| 4.3 | The login fails because the password is wrong. | A senha errada é a causa da falha. |
| 5.1 | The bug happens because the file is missing. | Resposta modelo; deve usar `because` para explicar causa. |
