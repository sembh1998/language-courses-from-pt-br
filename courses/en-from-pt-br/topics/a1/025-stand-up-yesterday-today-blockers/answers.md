# Respostas: Stand-up A1: yesterday, today, blockers

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Yesterday I worked on the API. | `Yesterday` marca uma ação de ontem. |
| 1.2 | Today I will update the ticket. | `Today I will...` apresenta plano para hoje. |
| 1.3 | I have a blocker. | Essa é a frase direta para comunicar impedimento. |
| 2.1 | Yesterday | A frase fala sobre o que foi feito ontem. |
| 2.2 | will | `Will` forma um plano futuro simples. |
| 2.3 | blocker | `Blocker` é impedimento. |
| 2.4 | access | `Need access` significa precisar de acesso. |
| 3.1 | Yesterday I fixed a bug. | `Fixed` é uma forma comum para “corrigi”. |
| 3.2 | Today I will review the code. | `Today I will...` indica plano. |
| 3.3 | I need help with the error. | Em inglês, dizemos `help with`. |
| 4.1 | Yesterday I worked on the ticket. | O verbo é `work on` para trabalhar em uma tarefa. |
| 4.2 | Today I will test the feature. | A transformação adiciona `today` e `will`. |
| 4.3 | What is the blocker? | Pergunta simples e clara para descobrir o impedimento. |
| 5.1 | Yesterday I worked on the dashboard. | Resposta modelo; precisa começar com `Yesterday` ou falar de ontem. |
| 5.2 | I need access to the server. | Resposta modelo; deve comunicar necessidade de acesso. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | Yesterday I reviewed a pull request. | A frase informa progresso de ontem. |
| 2 | will | `Will` completa a estrutura de plano para hoje. |
| 3 | Do you need help? | Pergunta adequada após alguém mencionar blocker. |
| 4 | blocker | O servidor fora do ar é um impedimento. |
| 5 | Today I will work on the ticket. | `Work on` é a combinação correta. |
| 6 | I need help with the error. | Usamos `with` depois de `help` nesse contexto. |
| 7 | Yesterday I worked on the bug. | Ordem natural para stand-up: tempo, sujeito, verbo e tarefa. |
