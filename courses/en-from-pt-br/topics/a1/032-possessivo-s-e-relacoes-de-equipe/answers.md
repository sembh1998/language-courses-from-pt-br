# Respostas: Possessivo com 's e relações de equipe

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | a branch da Ana | `Ana's` indica posse da Ana. |
| 1.2 | the user's password | A posse vem no dono: `user's`. |
| 1.3 | The team's backlog is full. | `The team's backlog` significa o backlog da equipe. |
| 2.1 | Ana's | O dono vem antes do item possuído. |
| 2.2 | user's | `The user's password` é a senha do usuário. |
| 2.3 | product's | `The product's page` é a página do produto. |
| 3.1 | The team's backlog is ready. | `Team's` marca posse da equipe. |
| 3.2 | Ana's task is urgent. | Para pessoa, use nome + `'s`. |
| 3.3 | The product's name is short. | `Product's name` é o nome do produto. |
| 4.1 | Ana's branch | Possessivo natural com pessoa. |
| 4.2 | the user's password | O dono é `user`, por isso recebe `'s`. |
| 4.3 | the team's backlog | A equipe é dona do backlog. |
| 5.1 | The team's backlog is full. | Resposta modelo; deve usar `team's` corretamente. |
