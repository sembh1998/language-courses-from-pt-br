# Possessivo com 's e relações de equipe

Nível: A1

## Objetivo

Usar o possessivo com `'s` para falar de branches, senhas, backlog, times e produtos em contextos simples de equipe.

## Explicação

Em inglês, para dizer que algo pertence a uma pessoa, usuário, equipe ou produto, use `'s` depois do dono.

- `Ana's branch` = a branch da Ana
- `the user's password` = a senha do usuário
- `the team's backlog` = o backlog da equipe
- `the product's name` = o nome do produto

Com coisas ou relações mais longas, `of` também pode aparecer, mas no A1 use `'s` para relações simples de posse.

- Natural: `Ana's branch`
- Também possível: `the branch of Ana`, mas soa menos natural para pessoas.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| Ana's branch is ready. | A branch da Ana está pronta. | Pessoa + `'s`. |
| The user's password is weak. | A senha do usuário é fraca. | `user` + `'s`. |
| The team's backlog is full. | O backlog da equipe está cheio. | `team` + `'s`. |
| The product's page is slow. | A página do produto está lenta. | Produto + `'s`. |

## Erros comuns

- Não use `'s` no objeto possuído: diga `Ana's branch`, não `Ana branch's`.
- Não confunda plural com posse. `users` é plural; `user's` é posse de um usuário.
- Para pessoas, prefira `'s`: `Marcos's task` ou `Marcos' task` podem aparecer; neste curso, use `Marcos's task` para manter simples.

## Prática guiada

Combine dono + item:

- Ana + branch = `Ana's branch`
- user + password = `the user's password`
- team + backlog = `the team's backlog`
- product + name = `the product's name`

## Resumo

Use `'s` para indicar posse simples: dono + `'s` + coisa possuída. Essa forma é muito comum para pessoas, usuários, equipes e produtos.
