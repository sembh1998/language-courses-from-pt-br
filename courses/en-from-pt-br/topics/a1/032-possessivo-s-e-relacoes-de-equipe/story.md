# História: Possessivo com 's e relações de equipe

Nível: A1

## História em inglês

Ana's branch is ready.
Marcos checks the team's backlog.
The user's password is weak.
The product's page is slow.
Ana opens the owner's task.
The team's plan is simple.
Marcos updates his branch.
The team is ready for review.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| Ana's branch is ready. | A branch da Ana está pronta. |
| Marcos checks the team's backlog. | Marcos verifica o backlog da equipe. |
| The user's password is weak. | A senha do usuário é fraca. |
| The product's page is slow. | A página do produto está lenta. |
| Ana opens the owner's task. | Ana abre a tarefa do responsável. |
| The team's plan is simple. | O plano da equipe é simples. |
| Marcos updates his branch. | Marcos atualiza a branch dele. |
| The team is ready for review. | A equipe está pronta para revisão. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| weak | fraco | Pode descrever senha insegura. |
| owner | responsável | Pessoa dona de uma tarefa ou item. |
| review | revisão | Checagem de trabalho. |

## Perguntas de compreensão

1. Whose branch is ready?
2. What does Marcos check?
3. What is weak?
4. What is slow?
5. Is the team's plan simple?
