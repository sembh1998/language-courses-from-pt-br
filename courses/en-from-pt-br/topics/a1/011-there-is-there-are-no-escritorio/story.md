# História: There is/there are no escritório e no sistema

Nível: A1

## História em inglês

It is Monday morning. There is a new bug in the app. There are two developers in the office. There is a meeting room near the team. There are some files in the project folder. There aren't any errors in the test report now.

## Tradução linha a linha

| Inglês | Tradução |
|---|---|
| It is Monday morning. | É segunda-feira de manhã. |
| There is a new bug in the app. | Há um bug novo no aplicativo. |
| There are two developers in the office. | Há dois desenvolvedores no escritório. |
| There is a meeting room near the team. | Há uma sala de reunião perto da equipe. |
| There are some files in the project folder. | Há alguns arquivos na pasta do projeto. |
| There aren't any errors in the test report now. | Não há erros no relatório de testes agora. |

## Vocabulário da história

| Inglês | Tradução | Observação |
|---|---|---|
| new bug | bug novo | Singular com `there is`. |
| two developers | dois desenvolvedores | Plural com `there are`. |
| test report | relatório de testes | Contexto de QA. |

## Perguntas de compreensão

1. What day is it?
2. Is there a new bug in the app?
3. How many developers are in the office?
4. Are there any errors in the test report now?
