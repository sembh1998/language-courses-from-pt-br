# There is/there are no escritório e no sistema

Nível: A1

## Objetivo

Neste tópico, você vai descrever o que existe em um ambiente físico ou digital usando `there is` e `there are`.

## Explicação

Use `there is` para uma coisa no singular. Use `there are` para duas ou mais coisas no plural.

| Inglês | Tradução | Observação |
|---|---|---|
| There is a bug. | Há um bug. | Uma coisa: `a bug`. |
| There are three files. | Há três arquivos. | Plural: `files`. |
| There isn't a printer. | Não há uma impressora. | Negativa no singular. |
| There aren't any errors. | Não há erros. | Negativa no plural com `any`. |
| Is there a meeting room? | Há uma sala de reunião? | Pergunta no singular. |
| Are there any tests? | Há testes? | Pergunta no plural com `any`. |

Use `some` em frases afirmativas quando a quantidade não é exata: `There are some files`. Use `any` em perguntas e negativas: `Are there any files?`, `There aren't any files`.

## Exemplos

| Inglês | Tradução | Observação |
|---|---|---|
| There is a laptop on the desk. | Há um notebook na mesa. | Singular. |
| There are two monitors in the office. | Há dois monitores no escritório. | Plural com número. |
| There is a bug in the app. | Há um bug no aplicativo. | Ambiente digital. |
| There are some comments in the code. | Há alguns comentários no código. | Afirmativa com `some`. |
| There aren't any tickets today. | Não há tickets hoje. | Negativa com `any`. |
| Are there any files in the folder? | Há arquivos na pasta? | Pergunta com `any`. |

## Erros comuns

1. Usar `there are` com singular: prefira `There is a file`.

2. Usar `some` em pergunta simples: prefira `Are there any bugs?`.

3. Esquecer o plural depois de números: `three files`, não `three file`.

## Prática guiada

Descreva sua tela: `There is a browser window. There are three tabs.` Depois faça uma pergunta: `Are there any errors?`

## Resumo

`There is` apresenta uma coisa. `There are` apresenta várias coisas. Em perguntas e negativas, `any` é a escolha mais comum.
