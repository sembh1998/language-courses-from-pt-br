# Respostas: There is/there are no escritório e no sistema

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | There is a bug. | `A bug` é singular, então usamos `there is`. |
| 1.2 | There are two files. | `Two files` é plural, então usamos `there are`. |
| 1.3 | Are there any errors? | Em perguntas no plural, `any` é a opção natural. |
| 2.1 | is | `A printer` é uma coisa no singular. |
| 2.2 | are | `Five tickets` é plural. |
| 2.3 | any | Em negativas, use `any`: `aren't any`. |
| 3.1 | There is a file in the folder. | `A file` pede `there is`. |
| 3.2 | There are some comments in the code. | Afirmativa plural com quantidade indefinida usa `some`. |
| 3.3 | There aren't any errors. | Negativa plural natural com `any`. |
| 4.1 | Is there a meeting room? | Em pergunta, `is` vem antes de `there`. |
| 4.2 | There aren't any bugs. | A negativa plural usa `aren't any`. |
| 5.1 | There is a browser on my screen. | Resposta modelo; qualquer frase correta com `There is` é aceitável. |

## Gabarito do teste

| ID | Resposta | Explicação |
|---|---|---|
| 1 | There is an issue. | `An issue` é singular. |
| 2 | There are four tabs. | `Four tabs` é plural. |
| 3 | There aren't any tests. | Negativas usam `any`, não `some`. |
| 4 | Are | Pergunta plural: `Are there any messages?` |
| 5 | There are two meetings today. | `Two meetings` exige `there are`. |
| 6 | There isn't a printer here. | `A printer` é singular; negativa: `isn't`. |
| 7 | Are there any files? | Resposta modelo com `any` em pergunta. |
| 8 | There are monitors. | No plural, trocamos `is` por `are` e removemos `a`. |

## Respostas da história

| Pergunta | Resposta | Observação |
|---|---|---|
| 1 | It is Monday. | A primeira frase diz que é segunda-feira de manhã. |
| 2 | Yes, there is. | A história diz que há um bug novo no aplicativo. |
| 3 | There are two developers. | A quantidade informada é dois. |
| 4 | No, there aren't. | A última frase diz que não há erros agora. |
