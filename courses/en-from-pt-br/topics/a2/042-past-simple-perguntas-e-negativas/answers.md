# Respostas: Past simple com perguntas e negativas

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Did you test the payment flow? | Did marca o passado e test volta à forma base. |
| 1.2 | Did Marta send the report? | Sent volta para send depois de did. |
| 1.3 | Did the pipeline finish at noon? | Finished volta para finish depois do auxiliar. |
| 1.4 | Did they find the missing file? | Found volta para find depois de did. |
| 2.1 | I didn't deploy the change. | Depois de didn't, deploy permanece na forma base. |
| 2.2 | She didn't write the incident summary. | Wrote volta para write. |
| 2.3 | We didn't run the end-to-end tests. | Ran volta para run após didn't. |
| 2.4 | The backup didn't finish successfully. | Finished volta para finish. |
| 3.1 | Did you check the logs? | A forma neutra usa did antes do sujeito e check na forma base. |
| 3.2 | The job didn't run. | Didn't é seguido por run. |
| 3.3 | Why did the service stop? | Why vem antes de did e stop fica na forma base. |
| 3.4 | No, we didn't. | A resposta curta termina com didn't sem repetir o verbo principal. |
| 4.1 | did | Perguntas no passado usam did antes do sujeito. |
| 4.2 | didn't | Didn't forma a negativa no passado. |
| 4.3 | restart | Depois de did, o verbo fica na forma base. |
| 4.4 | find | Depois de didn't, found volta para find. |
| 5.1 | Did you run the tests yesterday? | Run fica na forma base depois de did. |
| 5.2 | I didn't send the report. | Send fica na forma base depois de didn't. |
| 5.3 | Why did the build fail? | Why + did + sujeito + fail. |
| 5.4 | Where did you find this error? | Where vem antes de did, e find permanece na forma base. |
| 6.1 | Yes, I did. | A resposta curta afirmativa repete did. |
| 6.2 | No, it didn't. | A resposta curta negativa usa didn't. |
| 6.3 | Yes, they did. | O pronome they é seguido por did. |

## Respostas das perguntas de compreensão

| Pergunta | Resposta | Observação |
|---|---|---|
| What problem did the dashboard have? | It didn't show the latest sales data. | A segunda frase descreve o problema. |
| Did the nightly import finish? | No, it didn't. | A tarefa final não foi executada. |
| When did the previous step finish? | It finished at 2:14 a.m. | O horário aparece nos logs. |
| Did the team find a configuration error? | No, they didn't. | O texto afirma que a equipe não encontrou esse erro. |
| Who restarted the worker? | One engineer restarted the worker. | Um engenheiro confirmou a ação. |
| What didn't the engineer restart? | He didn't restart the scheduler. | A negativa identifica a ação que não aconteceu. |
