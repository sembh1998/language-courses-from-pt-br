# Respostas: Past simple: tarefas concluídas

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | fixed | Fix é regular e recebe -ed. |
| 1.2 | created | Create já termina em e e recebe apenas -d. |
| 1.3 | planned | Plan duplica a consoante final antes de -ed. |
| 1.4 | deleted | Delete recebe -d e forma deleted. |
| 2.1 | wrote | O passado irregular de write é wrote. |
| 2.2 | ran | O passado irregular de run é ran. |
| 2.3 | found | O passado irregular de find é found. |
| 2.4 | sent | O passado irregular de send é sent. |
| 3.1 | I reviewed the pull request yesterday morning. | Review é regular e recebe -ed; o marcador indica passado concluído. |
| 3.2 | She wrote the release notes last Friday. | Write muda para wrote. |
| 3.3 | They ran the security scan two hours ago. | Run muda para ran e ago posiciona a ação no passado. |
| 3.4 | We made a backup before the deployment. | Make muda para made. |
| 4.1 | I fixed two bugs yesterday. | A ordem neutra é sujeito, verbo, objeto e marcador de tempo. |
| 4.2 | Then, she sent the report. | Then introduz a ação seguinte e sent é o passado de send. |
| 4.3 | We ran the tests last night. | Ran expressa a execução concluída. |
| 5.1 | I found the cause in the logs yesterday. | Find muda para found. |
| 5.2 | We created three new tests last week. | Create é regular e forma created. |
| 5.3 | She wrote the documentation and sent the link. | Wrote e sent são formas irregulares. |
| 5.4 | First, I ran the tests. Then, I fixed the error. | First e then organizam duas ações concluídas. |

## Respostas das perguntas de compreensão

| Pergunta | Resposta | Observação |
|---|---|---|
| Why did Camila start work early? | Because the team prepared an important release. | A primeira frase explica o motivo. |
| What did she find in the staging config? | She found an incorrect environment variable. | O problema estava em uma variável de ambiente. |
| What did Camila create after she fixed the value? | She created a short regression test. | O teste foi criado após a correção. |
| Where did she send the test results? | She sent them to the release channel. | Os resultados foram enviados ao canal da versão. |
| What did the team deploy in the afternoon? | The team deployed the new version to production. | A ação principal da tarde foi o deploy. |
| How did Camila finish the day? | She finished the day tired but satisfied. | A última frase descreve seu estado. |
