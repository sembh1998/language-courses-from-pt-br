# História: Past simple: tarefas concluídas

Nível: A2

## História em inglês

### A busy morning

Yesterday, Camila started work early because the team prepared an important release. First, she read the deployment checklist and reviewed the open tickets. She found an incorrect environment variable in the staging config. Camila fixed the value and created a short regression test. Then, she ran the full test suite and checked the logs. All tests passed, so she sent the results to the release channel.

### The release update

In the afternoon, the team deployed the new version to production. Camila watched the monitoring dashboard for thirty minutes. She saw no new errors and wrote a clear status update. The product manager read the message and thanked the team. Finally, Camila updated the ticket and closed the release task. She finished the day tired but satisfied.

## Notas de vocabulário

- deployment checklist = lista de verificação usada antes de um deploy
- found = passado irregular de find
- ran the full test suite = executou a suíte completa de testes
- sent the results = enviou os resultados; sent é o passado de send
- wrote a status update = escreveu uma atualização de status
- closed the release task = encerrou a tarefa da versão
