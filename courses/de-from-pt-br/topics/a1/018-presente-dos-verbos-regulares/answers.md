# Respostas: Presente dos verbos regulares

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | lerne | Com **ich**, verbo regular recebe **-e**. |
| 1.2 | wohnst | Com **du**, verbo regular recebe **-st**. |
| 1.3 | spielt | **Anna** equivale a **sie**, então recebe **-t**. |
| 1.4 | machen | Com **wir**, a forma termina em **-en**. |
| 1.5 | macht | Com **ihr**, verbo regular recebe **-t**. |
| 1.6 | wohnen | **Sie** formal recebe a forma em **-en**. |
| 2.1 | spiele | **ich** pede a terminação **-e**. |
| 2.2 | machst | **du** pede a terminação **-st**. |
| 2.3 | wohnen | **wir** usa a forma igual ao infinitivo regular. |
| 2.4 | lernt | **ihr** pede a terminação **-t**. |
| 2.5 | macht | **er** pede a terminação **-t**. |
| 2.6 | spielen | **sie** plural pede a terminação **-en**. |
| 3.1 | ich lerne, du lernst, er lernt, wir lernen, ihr lernt, sie lernen | O radical é **lern-** e as terminações seguem o padrão regular. |
| 3.2 | ich mache, du machst, sie macht, wir machen, ihr macht, Sie machen | O radical é **mach-**; **Sie** formal usa **-en**. |
| 3.3 | ich wohne, du wohnst, es wohnt, wir wohnen, ihr wohnt, sie wohnen | O radical é **wohn-** e as terminações são regulares. |
| 4.1 | Ich wohne in Hamburg. | **Ich** usa **wohne**. |
| 4.2 | Du lernst Deutsch. | **Du** usa **lernst**. |
| 4.3 | Er spielt Fußball. | **Er** usa **spielt**. |
| 4.4 | Wir machen die Aufgabe. | **Wir** usa **machen**. |
| 4.5 | Ihr lernt schnell. | **Ihr** usa **lernt**. |
| 5.1 | e | A terminação de **ich** é **-e**. |
| 5.2 | st | A terminação de **du** é **-st**. |
| 5.3 | t | A terminação de **er** é **-t**. |
| 5.4 | en | A terminação de **wir** é **-en**. |
| 5.5 | t | A terminação de **ihr** é **-t**. |
| 5.6 | en | A terminação de **Sie** formal é **-en**. |
| 6.1 | Ich lerne Deutsch. | Com **ich**, não usamos o infinitivo **lernen**. |
| 6.2 | Du wohnst hier. | Com **du**, acrescente **-st** ao radical. |
| 6.3 | Wir machen Kaffee. | Com **wir**, a forma correta é **machen**. |
| 6.4 | Ihr spielt Tennis. | Com **ihr**, a forma correta termina em **-t**. |
| 6.5 | Sie machen die Übung. | **Sie** formal usa a forma em **-en**. |
