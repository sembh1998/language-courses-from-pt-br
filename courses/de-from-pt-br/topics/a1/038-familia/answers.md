# Respostas: Família

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | mein | `Vater` é masculino; use `mein`. |
| 1.2 | meine | `Mutter` é feminino; use `meine`. |
| 1.3 | Meine | `Schwester` é feminino; no início da frase, maiúscula. |
| 1.4 | Meine | `Eltern` é plural; use `meine`. |
| 2.1 | o irmão | `der Bruder` significa o irmão. |
| 2.2 | a mãe | `die Mutter` significa a mãe. |
| 2.3 | irmãos e irmãs | `Geschwister` é plural coletivo. |
| 3.1 | Mein Vater wohnt in São Paulo. | `Vater` pede `mein`. |
| 3.2 | Meine Mutter ist verheiratet. | `Mutter` pede `meine`. |
| 3.3 | Ich habe eine Schwester. | `eine Schwester` indica uma irmã. |
| 3.4 | Meine Eltern sind zu Hause. | `Eltern` é plural, então `meine`. |
| 4.1 | Mutter | A mãe da pessoa é `Mutter`. |
| 4.2 | Bruder | O irmão da pessoa é `Bruder`. |
| 4.3 | Eltern | Pai e mãe juntos são `Eltern`. |
| 5.1 | Ja, ich habe Geschwister. | Resposta-modelo com vocabulário correto. |
| 5.2 | Meine Mutter heißt Ana. | Resposta-modelo usando `meine Mutter`. |
