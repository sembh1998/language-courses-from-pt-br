## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Ich heiße Carla. | `Wie heißt du?` pergunta o nome. |
| 1.2 | Ich wohne in Recife. | `Wo wohnst du?` pergunta onde você mora. |
| 1.3 | Ich höre gern Musik. | `Was machst du gern?` pergunta atividade preferida. |
| 2.1 | komme | `Ich komme aus Brasilien` indica origem. |
| 2.2 | wohnst | Com `du`, use `wohnst`. |
| 2.3 | lernen | Com `wir`, use `lernen`. |
| 2.4 | hört | Com `sie`, use `hört`. |
| 3.1 | meine | `Mutter` é feminino. |
| 3.2 | einen | `Bruder` é masculino no acusativo. |
| 3.3 | Mein | `Zimmer` é neutro e usa `mein`. |
| 3.4 | Die | `Küche` é feminino. |
| 4.1 | Ich heiße Lara. | Ordem básica: sujeito + verbo + nome. |
| 4.2 | Ich lese gern. | `gern` fica naturalmente após o verbo. |
| 4.3 | Ich stehe um sieben Uhr auf. | Verbo separável: `auf` no fim. |
| 4.4 | Ich habe Deutsch gelernt. | Perfekt: auxiliar + particípio no fim. |
| 5.1 | Ich wohne in Salvador. | `wohnen in` indica moradia. |
| 5.2 | Meine Schwester ist nett. | `Schwester` usa `meine`. |
| 5.3 | Wir kaufen heute ein. | `einkaufen` separa `ein`. |
| 5.4 | Gestern habe ich gekocht. | Perfekt com `haben` e particípio. |
| 6.1 | Ich heiße Bruno und komme aus Brasilien. | Frase combina nome e origem. |
| 6.2 | Ich höre gern Musik. | `Musik hören` com `gern`. |
| 6.3 | Mein Zimmer ist klein. | Descrição simples com `sein`. |
| 7.1 | Ich heiße Ana. | Resposta-modelo para nome. |
| 7.2 | Ich komme aus Brasilien. | Resposta-modelo para origem. |
| 7.3 | Ich habe Deutsch gelernt. | Resposta-modelo no Perfekt. |
