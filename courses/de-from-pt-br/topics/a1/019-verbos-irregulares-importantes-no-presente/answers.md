# Respostas: Verbos irregulares importantes no presente

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | esse | Com **ich**, **essen** não muda para **isst**. |
| 1.2 | sprichst | Com **du**, **sprechen** muda **e -> i**. |
| 1.3 | liest | **Lena** equivale a **sie**, então **lesen** vira **liest**. |
| 1.4 | fahren | Com **wir**, **fahren** fica sem umlaut. |
| 1.5 | seht | Com **ihr**, **sehen** usa **seht**. |
| 1.6 | nimmt | **Max** equivale a **er**, então **nehmen** vira **nimmt**. |
| 1.7 | schläft | **Die Katze** equivale a **sie**, então **schlafen** recebe **ä**. |
| 2.1 | siehst | Com **du**, **sehen** vira **siehst**. |
| 2.2 | fährt | Com **er**, **fahren** vira **fährt**. |
| 2.3 | nimmt | Com **sie** singular, **nehmen** vira **nimmt**. |
| 2.4 | sprechen | Com **wir**, não há mudança vocálica especial. |
| 2.5 | esst | Com **ihr**, a forma correta de **essen** é **esst**. |
| 2.6 | schläfst | Com **du**, **schlafen** vira **schläfst**. |
| 2.7 | lese | Com **ich**, **lesen** usa **lese**. |
| 3.1 | ich spreche, du sprichst, er spricht, wir sprechen, ihr sprecht, sie sprechen | A mudança **e -> i** aparece em **du** e **er**. |
| 3.2 | ich esse, du isst, sie isst, wir essen, ihr esst, Sie essen | **essen** muda para **isst** em **du** e **sie** singular. |
| 3.3 | ich fahre, du fährst, er fährt, wir fahren, ihr fahrt, sie fahren | O umlaut aparece em **du** e **er**. |
| 3.4 | ich nehme, du nimmst, es nimmt, wir nehmen, ihr nehmt, sie nehmen | **nehmen** tem as formas especiais **nimmst** e **nimmt**. |
| 4.1 | Du liest ein Buch. | Com **du**, **lesen** vira **liest**. |
| 4.2 | Sie isst Brot. | **Ela** é **sie** singular, então **essen** vira **isst**. |
| 4.3 | Wir sehen einen Film. | Com **wir**, **sehen** fica no infinitivo conjugado. |
| 4.4 | Er fährt nach Berlin. | Com **er**, **fahren** vira **fährt**. |
| 4.5 | Ihr schlaft gut. | Com **ihr**, use **schlaft**, sem umlaut. |
| 4.6 | Ich nehme den Bus. | Com **ich**, **nehmen** vira **nehme**. |
| 5.1 | sprichst | **sprechen** muda **e -> i** com **du**. |
| 5.2 | liest | **lesen** muda **e -> ie** com **du**. |
| 5.3 | siehst | **sehen** muda **e -> ie** com **du**. |
| 5.4 | isst | **essen** muda **e -> i** com **du**. |
| 5.5 | fährst | **fahren** recebe umlaut com **du**. |
| 5.6 | schläfst | **schlafen** recebe umlaut com **du**. |
| 5.7 | nimmst | **nehmen** tem forma especial com **du**. |
| 6.1 | Du sprichst sehr leise. | A forma **sprechst** não é correta; use **sprichst**. |
| 6.2 | Er liest die Zeitung. | Com **er**, **lesen** vira **liest**. |
| 6.3 | Wir fahren morgen. | Com **wir**, não use umlaut em **fahren**. |
| 6.4 | Ihr schlaft lange. | Com **ihr**, a forma correta é **schlaft**. |
| 6.5 | Sie nimmt Tee. | **sie** singular usa **nimmt**. |
| 6.6 | Du siehst das Auto. | Com **du**, **sehen** vira **siehst**. |
| 7.1 | nimmt | **Der Mann** equivale a **er**, então **nehmen** vira **nimmt**. |
| 7.2 | schläft | **Meine Schwester** equivale a **sie**, então **schlafen** vira **schläft**. |
| 7.3 | esse | Com **ich**, **essen** usa **esse**. |
| 7.4 | sprechen | **Sie** formal usa a forma em **-en**. |
| 7.5 | sehen | **Die Kinder** é plural, então usa **sehen**. |
| 7.6 | fährst | Com **du**, **fahren** vira **fährst**. |
