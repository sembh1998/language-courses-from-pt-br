Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|

| 1.4 | Wir | Grupo com quem fala. |
| 2.1 | nós | wir = nós. |
| 2.2 | vocês informal | ihr = vocês informal. |
| 2.3 | formal | Sie maiúsculo é formal. |
| 3.1 | Sie ist hier. | Anna vira sie. |
| 3.2 | Er wohnt in Bonn. | Paul vira er. |
| 3.3 | Sie lernen. | Plural vira sie. |

| 5.1 | informal | du. |
| 5.2 | formal | Sie. |
| 5.3 | plural | wir. |
| 5.4 | plural informal | ihr. |
