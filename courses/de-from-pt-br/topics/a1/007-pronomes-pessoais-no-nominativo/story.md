# História: Pronomes pessoais no nominativo

Nível: A1

## História em alemão

Ich bin Ana. Du bist Leo. Er ist mein Freund. Sie ist meine Freundin. Es ist mein Buch. Wir lernen Deutsch. Ihr seid auch hier. Sie lernen auch Deutsch. Sie sind Herr Klein.

## Tradução linha por linha

| Alemão | Tradução |
|---|---|
| Ich bin Ana. | Eu sou Ana. |
| Du bist Leo. | Você é Leo. |
| Er ist mein Freund. | Ele é meu amigo. |
| Sie ist meine Freundin. | Ela é minha amiga. |
| Es ist mein Buch. | Ele/isso é meu livro. |
| Wir lernen Deutsch. | Nós aprendemos alemão. |
| Ihr seid auch hier. | Vocês também estão aqui. |
| Sie lernen auch Deutsch. | Eles/elas também aprendem alemão. |
| Sie sind Herr Klein. | O senhor é o senhor Klein. |

## Notas de vocabulário

- auch = também
- mein Freund = meu amigo
- meine Freundin = minha amiga
- das Buch = o livro; por isso pode virar es

## Perguntas de compreensão

1. Quem é Ana?
2. Qual pronome retoma das Buch?
3. Qual frase usa Sie formal?
