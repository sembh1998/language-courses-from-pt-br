# Pronomes pessoais no nominativo

Nível: A1

## Objetivo

Reconhecer e usar os pronomes pessoais alemães no nominativo: ich, du, er, sie, es, wir, ihr, sie e Sie.

## Explicação

O nominativo é o caso do sujeito: a pessoa ou coisa que faz a ação ou é descrita. Em frases simples de A1, o pronome geralmente vem antes do verbo.

Em alemão, existe diferença entre **sie** e **Sie**. Com letra minúscula, **sie** pode significar “ela” ou “eles/elas”. Com letra maiúscula, **Sie** significa “o senhor/a senhora/vocês” em contexto formal.

| Pronome | Português | Uso básico |
|---|---|---|
| ich | eu | quem fala |
| du | você/tu | uma pessoa, informal |
| er | ele | masculino singular |
| sie | ela | feminino singular |
| es | ele/ela/isso | neutro singular |
| wir | nós | grupo com quem fala |
| ihr | vocês/vós | grupo informal |
| sie | eles/elas | plural |
| Sie | o senhor/a senhora/vocês | formal |

## Exemplos

| Alemão | Tradução | Observação |
|---|---|---|
| Ich lerne Deutsch. | Eu aprendo alemão. | ich = eu |
| Du bist Anna. | Você é Anna. | du = informal |
| Er kommt aus Brasilien. | Ele vem do Brasil. | er = masculino |
| Sie wohnt in Berlin. | Ela mora em Berlim. | sie = ela |
| Es ist klein. | É pequeno. | es = neutro |
| Wir sprechen Portugiesisch. | Nós falamos português. | wir = nós |
| Ihr seid Freunde. | Vocês são amigos. | ihr = plural informal |
| sie sind hier. | eles/elas estão aqui. | sie minúsculo plural |
| Sie sind Herr Müller. | O senhor é o senhor Müller. | Sie formal |

## Erros comuns

- Confundir **sie** minúsculo com **Sie** formal. A letra maiúscula muda o sentido.
- Usar **du** em contexto formal. Com professores, desconhecidos ou atendimento, use **Sie**.
- Traduzir **es** sempre como “isso”. Muitas vezes ele retoma uma palavra neutra: **das Kind → es**.

## Resumo

Os pronomes no nominativo são a base para conjugar verbos. Memorize especialmente **ich, du, er/sie/es, wir, ihr, sie/Sie** e preste atenção ao **S** maiúsculo de **Sie**.
