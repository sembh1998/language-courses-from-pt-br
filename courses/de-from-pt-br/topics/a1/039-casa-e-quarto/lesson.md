# Casa e quarto

Nível: A1

## Objetivo

Aprender vocabulário básico para falar da casa, dos cômodos, móveis e objetos pessoais.

## Explicação

Use `das Haus` para casa como construção e `die Wohnung` para apartamento ou moradia. Cômodos importantes: `das Zimmer`, `die Küche`, `das Bad`, `das Schlafzimmer`. Para dizer que algo está em um lugar, em A1 use frases simples com `ist` ou `steht`.

## Exemplos

| Alemão | Tradução | Observação |
|---|---|---|
| Das ist mein Zimmer. | Este é meu quarto. | `Zimmer` é neutro. |
| Die Küche ist klein. | A cozinha é pequena. | Adjetivo simples. |
| Das Bett steht im Schlafzimmer. | A cama fica no quarto de dormir. | `steht` para móveis. |
| Mein Tisch ist neu. | Minha mesa é nova. | Objeto pessoal/móvel. |
| Ich wohne in einer Wohnung. | Eu moro em um apartamento. | Frase de moradia. |

## Erros comuns

- Confundir `Haus` com `Wohnung`: casa/prédio vs. apartamento/moradia.
- Esquecer artigo e gênero: `das Zimmer`, `die Küche`, `das Bad`.
- Usar sempre `ist` para móveis; `steht` é muito comum para móveis em pé ou posicionados.

## Prática guiada

| Palavra | Artigo | Tradução |
|---|---|---|
| Haus | das | casa |
| Wohnung | die | apartamento/moradia |
| Zimmer | das | quarto/cômodo |
| Küche | die | cozinha |
| Bad | das | banheiro |

## Resumo

Memorize as palavras com artigo. Para descrever sua casa em A1, use frases curtas: `Mein Zimmer ist klein`, `Das Bett steht hier`, `Ich wohne in einer Wohnung`.
