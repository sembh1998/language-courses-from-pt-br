# Respostas: Casa e quarto

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Das | `Haus` é neutro. |
| 1.2 | Die | `Küche` é feminino. |
| 1.3 | Der | `Tisch` é masculino. |
| 1.4 | Das | `Bad` é neutro. |
| 2.1 | o quarto/cômodo | `Zimmer` pode ser quarto ou cômodo. |
| 2.2 | o apartamento | `Wohnung` significa apartamento/moradia. |
| 2.3 | o armário | `Schrank` significa armário. |
| 3.1 | Mein Haus ist klein. | `Haus` é neutro e usa `mein`. |
| 3.2 | Die Küche ist schön. | `Küche` é feminino com `die`. |
| 3.3 | Das Bett ist neu. | `Bett` é neutro com `das`. |
| 3.4 | Der Tisch ist im Zimmer. | `Tisch` é masculino; `im Zimmer` indica no quarto. |
| 4.1 | Bett | Dorme-se na cama. |
| 4.2 | Küche | Cozinha-se na cozinha. |
| 4.3 | Bad | Toma-se banho no banheiro. |
| 5.1 | Mein Zimmer ist klein. | Resposta-modelo com vocabulário correto. |
| 5.2 | Ich wohne in einer Wohnung. | Resposta-modelo para moradia. |
