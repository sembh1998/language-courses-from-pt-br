# Respostas: Apresentações com sein e heißen

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Eu me chamo Ana. | `heiße` indica nome. |
| 1.2 | Como você se chama? | Pergunta informal sobre nome. |
| 1.3 | Quem é você? | `wer` = quem. |
| 1.4 | Esta é a Lena. | `Das ist...` apresenta alguém. |
| 2.1 | bin | `ich bin`. |
| 2.2 | bist | `du bist`. |
| 2.3 | ist | `er ist`. |
| 2.4 | ist | `sie ist`. |
| 2.5 | sind | Formal: `Sie sind`. |
| 2.6 | ist | `Das ist...`. |
| 3.1 | heiße | `ich heiße`. |
| 3.2 | heißt | `du heißt`. |
| 3.3 | heißt | `du heißt`. |
| 3.4 | heißt | `er heißt`. |
| 3.5 | heißt | `sie heißt`. |
| 3.6 | heißen | Formal: `Sie heißen`. |
| 4.1 | Ich heiße Ana. | Não junte `bin` e `heiße`. |
| 4.2 | Du bist Paul. | `du` usa `bist`. |
| 4.3 | Er heißt Lukas. | `er` usa `heißt`. |
| 4.4 | Wie heißen Sie? | Pergunta formal. |
| 5.1 | heiße | Resposta com nome. |
| 5.2 | Und | `Und du?` = e você? |
| 5.3 | bin | `Ich bin...`. |
| 5.4 | heißen | Formal: `Wie heißen Sie?` |
| 5.5 | ist | `Das ist...`. |
| 6.1 | Ich heiße Ana. | Tradução direta. |
| 6.2 | Wie heißt du? | Pergunta informal. |
| 6.3 | Ich bin Bruno. | Identidade com `sein`. |
| 6.4 | Sie ist Lena. | `sie ist`. |
| 6.5 | Er heißt Thomas. | `er heißt`. |
| 6.6 | Das ist Maria. | Apresentando outra pessoa. |
| 7.1 | Wie heißt du? | Pergunta com `du`. |
| 7.2 | Du bist Paul. | Troca `ich` para `du`. |
| 7.3 | Er heißt Lukas. | Troca para `er`. |
| 7.4 | Sie ist Clara. | Troca para `sie`. |
| 7.5 | Wie heißen Sie? | Forma formal. |
| 8.1 | Ich heiße Ana. | Correção mais natural. |
| 8.2 | Du bist Paul. | `du bist`. |
| 8.3 | Wie heißt du? | Nome usa `heißen`. |
| 8.4 | Er heißt Lukas. | `er heißt`. |
| 8.5 | Sie ist Lena. | `sie ist`. |
