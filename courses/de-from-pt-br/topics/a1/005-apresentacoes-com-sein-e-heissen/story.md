# História: Apresentações com sein e heißen

Nível: A1

## História em alemão

### Dialog 1: In der Schule

Hallo! Ich heiße Ana. Ich bin aus Brasilien. Und du?

Ich heiße Paul. Ich bin aus Deutschland.

Wer ist das?

Das ist Lena. Sie ist meine Freundin.

Hallo, Lena! Wie heißt du?

Ich heiße Lena. Sehr angenehm!

### Dialog 2: Im Büro

Guten Tag. Wie heißen Sie?

Ich heiße Herr Müller. Und Sie?

Ich heiße Frau Weber.

Sehr angenehm, Frau Weber.

Wer ist das?

Das ist Herr Klein. Er ist mein Kollege.

### Dialog 3: Kleine Vorstellung

Ich heiße Bruno. Ich bin aus Brasilien. Das ist Clara. Sie ist meine Freundin. Sie heißt Clara und sie ist sehr nett.

## Tradução linha por linha

| Alemão | Tradução |
|---|---|
| Hallo! Ich heiße Ana. | Olá! Eu me chamo Ana. |
| Ich bin aus Brasilien. | Eu sou do Brasil. |
| Und du? | E você? |
| Ich heiße Paul. | Eu me chamo Paul. |
| Ich bin aus Deutschland. | Eu sou da Alemanha. |
| Wer ist das? | Quem é esta pessoa? / Quem é isso? |
| Das ist Lena. | Esta é a Lena. |
| Sie ist meine Freundin. | Ela é minha amiga. |
| Hallo, Lena! Wie heißt du? | Olá, Lena! Como você se chama? |
| Ich heiße Lena. Sehr angenehm! | Eu me chamo Lena. Muito prazer! |
| Guten Tag. Wie heißen Sie? | Bom dia/boa tarde. Como o senhor/a senhora se chama? |
| Ich heiße Herr Müller. Und Sie? | Eu me chamo senhor Müller. E o senhor/a senhora? |
| Ich heiße Frau Weber. | Eu me chamo senhora Weber. |
| Sehr angenehm, Frau Weber. | Muito prazer, senhora Weber. |
| Das ist Herr Klein. | Este é o senhor Klein. |
| Er ist mein Kollege. | Ele é meu colega. |
| Ich heiße Bruno. | Eu me chamo Bruno. |
| Das ist Clara. | Esta é a Clara. |
| Sie heißt Clara und sie ist sehr nett. | Ela se chama Clara e é muito simpática. |

## Notas de vocabulário

- Ich heiße = eu me chamo; meu nome é
- Wie heißt du? = como você se chama? informal
- Wie heißen Sie? = como o senhor/a senhora se chama? formal
- Ich bin = eu sou; eu estou
- Das ist = este/esta é; isso é
- Wer ist das? = quem é esta pessoa?
- mein Freund = meu amigo
- meine Freundin = minha amiga
- mein Kollege = meu colega
- sehr angenehm = muito prazer
- und du? = e você?
- und Sie? = e o senhor/a senhora?

## Perguntas de compreensão

1. Wie heißt die Person aus Brasilien im ersten Dialog?
2. Wie heißt Paul?
3. Wer ist Lena?
4. Wie fragt man formal nach dem Namen?
5. Wer ist Herr Klein?
6. Como Bruno apresenta Clara?
