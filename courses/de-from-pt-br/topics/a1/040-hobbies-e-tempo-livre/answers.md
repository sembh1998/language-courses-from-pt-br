# Respostas: Hobbies e tempo livre

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | lese | Com `ich`, use `lese`. |
| 1.2 | hören | Com `wir`, use `hören`. |
| 1.3 | spielt | Com `sie`, use `spielt`. |
| 1.4 | kochst | Com `du`, use `kochst`. |
| 2.1 | passear | `spazieren gehen` significa passear. |
| 2.2 | ouvir música | `Musik hören` significa ouvir música. |
| 2.3 | jogar futebol | `Fußball spielen` significa jogar futebol. |
| 3.1 | Ich lese gern. | `gern` indica gostar de fazer algo. |
| 3.2 | Sie spielt Geige. | Para instrumentos, use `spielen`. |
| 3.3 | Wir hören Musik. | `hören` significa ouvir. |
| 3.4 | Er geht gern spazieren. | `spazieren gehen` precisa de `gehen`. |
| 4.1 | Ich koche gern. | A posição natural de `gern` é após o verbo conjugado. |
| 4.2 | Spielst du Geige? | Pergunta sim/não começa com verbo. |
| 4.3 | Wir gehen spazieren. | Expressão correta com `gehen`. |
| 5.1 | Ich höre gern Musik. | Resposta-modelo com hobby e `gern`. |
| 5.2 | Am Wochenende lese ich. | Resposta-modelo com tempo no início. |
