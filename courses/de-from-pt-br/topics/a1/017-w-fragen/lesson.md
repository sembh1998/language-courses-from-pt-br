# W-Fragen

Nível: A1

## Objetivo

Aprender perguntas com palavras interrogativas como wer, was, wo, woher, wohin, wann, wie, warum, wie viel e wie viele.

## Explicação

W-Fragen são perguntas abertas. Elas começam com uma palavra interrogativa, normalmente começada por W. A estrutura básica é: **W-Wort + verbo conjugado + sujeito + complemento?**

Exemplo: **Wo wohnst du?** = Onde você mora?

As perguntas mais importantes no A1 são:

- **Wer?** = Quem?
- **Was?** = O quê?
- **Wo?** = Onde?
- **Woher?** = De onde?
- **Wohin?** = Para onde?
- **Wann?** = Quando?
- **Wie?** = Como?
- **Warum?** = Por quê?
- **Wie viel?** = Quanto? para quantidade não contável ou preço
- **Wie viele?** = Quantos/quantas? para itens contáveis

Depois da palavra W, o verbo fica na posição 2. Compare: **Du wohnst in Berlin.** → **Wo wohnst du?**

## Exemplos

| Alemão | Tradução | Observação |
|---|---|---|
| Wer bist du? | Quem é você? | Pessoa |
| Was machst du? | O que você faz? | Ação ou coisa |
| Wo wohnst du? | Onde você mora? | Localização |
| Woher kommst du? | De onde você vem? | Origem |
| Wohin gehst du? | Para onde você vai? | Direção |
| Wann kommst du? | Quando você vem? | Tempo |
| Wie heißt du? | Como você se chama? | Nome |
| Warum lernst du Deutsch? | Por que você estuda alemão? | Motivo |
| Wie viel kostet das? | Quanto custa isso? | Preço |
| Wie viele Bücher hast du? | Quantos livros você tem? | Quantidade contável |

## Erros comuns

- Confundir **wo** com **woher**: **wo** = onde; **woher** = de onde.
- Confundir **wo** com **wohin**: **wohin** indica direção, “para onde”.
- Esquecer a posição do verbo: **Wo du wohnst?** não é a pergunta principal correta no A1; use **Wo wohnst du?**
- Usar **wie viel** com plural contável. Para “quantos livros”, use **wie viele Bücher**.

## Resumo

W-Fragen começam com palavra interrogativa. Depois vem o verbo conjugado e depois o sujeito: **Wo wohnst du?** Memorize as palavras W porque elas organizam conversas básicas.
