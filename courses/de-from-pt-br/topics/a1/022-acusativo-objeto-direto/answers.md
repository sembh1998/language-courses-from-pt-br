# Respostas: Acusativo: objeto direto

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | den | **der Mann** é masculino e recebe a ação de **sehen**, então vira **den Mann**. |
| 1.2 | das | **das Buch** é neutro; no acusativo básico o artigo não muda. |
| 1.3 | den | **der Kaffee** é masculino e objeto de **trinken**, então **der** vira **den**. |
| 1.4 | die | **die Frau** é feminino; no acusativo básico continua **die**. |
| 1.5 | die | **die Aufgabe** é feminino; como objeto direto continua **die**. |
| 1.6 | den | **der Apfel** é masculino; no acusativo definido é **den Apfel**. |
| 2.1 | einen | **der Hund** é masculino; com artigo indefinido, **ein** vira **einen**. |
| 2.2 | eine | **die Tasche** é feminino; **eine** não muda no acusativo básico. |
| 2.3 | ein | **das Buch** é neutro; **ein** fica **ein** no acusativo. |
| 2.4 | einen | **der Kuchen** é masculino; objeto direto com artigo indefinido usa **einen**. |
| 2.5 | einen | **der Stift** é masculino; depois de **brauchen**, o objeto direto é **einen Stift**. |
| 2.6 | ein | **das Wasser** é neutro; no acusativo fica **ein Wasser**. |
| 3.1 | Ich sehe den Lehrer. | **der Lehrer** é masculino e objeto direto; por isso **den Lehrer**. |
| 3.2 | Mia kauft einen Apfel. | **der Apfel** é masculino; **ein** vira **einen** no acusativo. |
| 3.3 | Ben liest das Buch. | **das Buch** é neutro; usar **den** ou **die** estaria errado. |
| 3.4 | Wir kaufen die Tasche. | **die Tasche** é feminino e não muda no acusativo básico. |
| 3.5 | Anna sieht die Kinder. | Plural com artigo definido continua **die** no acusativo básico. |
| 3.6 | Du trinkst den Kaffee. | **der Kaffee** é masculino; como objeto direto, usa **den Kaffee**. |
| 4.1 | objeto direto | **den Hund** recebe a ação de **sehen**. |
| 4.2 | sujeito | **Der Mann** faz a ação de ler. |
| 4.3 | objeto direto | **eine Tasche** é aquilo que Lena compra. |
| 4.4 | sujeito | **Das Kind** faz a ação de comer. |
| 4.5 | objeto direto | **den Kaffee** é aquilo que nós bebemos. |
| 5.1 | Ich kaufe den Stift. | **der Stift** é masculino; no acusativo definido vira **den Stift**. |
| 5.2 | Sie hat einen Hund. | **der Hund** é masculino; com artigo indefinido vira **einen Hund**. |
| 5.3 | Tom liest das Buch. | **das Buch** é neutro, então não usa **den**. |
| 5.4 | Wir sehen den Lehrer. | **der Lehrer** é masculino e objeto de **sehen**. |
| 5.5 | Mia kauft eine Tasche. | **die Tasche** é feminino; o artigo correto é **eine**. |
| 6.1 | Ich sehe den Mann. | **Mann** é masculino; objeto direto com artigo definido usa **den**. |
| 6.2 | Sie kauft einen Apfel. | **Apfel** é masculino; artigo indefinido no acusativo é **einen**. |
| 6.3 | Wir lesen das Buch. | **Buch** é neutro; o artigo definido continua **das**. |
| 6.4 | Er hat einen Hund. | **Hund** é masculino; com **haben**, o objeto direto usa **einen**. |
| 6.5 | Ich kaufe eine Tasche. | **Tasche** é feminino; artigo indefinido continua **eine**. |
| 7.1 | den Mann | **der Mann** no acusativo vira **den Mann**. |
| 7.2 | einen Lehrer | **ein Lehrer** é masculino; no acusativo vira **einen Lehrer**. |
| 7.3 | die Frau | Feminino com artigo definido não muda no acusativo básico. |
| 7.4 | das Brot | Neutro com artigo definido não muda no acusativo básico. |
| 7.5 | einen Kaffee | **ein Kaffee** é masculino; no acusativo vira **einen Kaffee**. |
| 7.6 | eine Aufgabe | Feminino com artigo indefinido não muda no acusativo básico. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1 | Mia vê **den Kellner**, o garçom. |
| 2 | O garçom traz **einen Kaffee**. |
| 3 | Ela compra **einen Kuchen**. |
| 4 | Ben tem **einen Hund**. |
| 5 | Ben pega **einen Ball**. |
| 6 | Mia lê **ein Buch**. |
| 7 | Porque **der Kellner** é masculino e recebe a ação de **sehen**; por isso vira **den Kellner**. |
| 8 | Porque **das Buch** é neutro; no acusativo básico **ein** não vira **einen**. |
| Prática 1 | den | **der Kaffee** é masculino e objeto direto. |
| Prática 2 | einen | **der Hund** é masculino; com artigo indefinido no acusativo fica **einen**. |
| Prática 3 | ein | **das Buch** é neutro; **ein** não muda no acusativo. |
| Prática 4 | den Kellner | Masculino definido no acusativo: **der → den**. |
| Prática 5 | einen Ball | Masculino indefinido no acusativo: **ein → einen**. |
| Prática 6 | Ich sehe den Kellner. | **der Kellner** recebe a ação de ver, então usa **den**. |
