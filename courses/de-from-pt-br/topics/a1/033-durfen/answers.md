# Respostas: Dürfen

Nível: A1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | darf | Ich usa a forma singular. |
| 1.2 | darfst | Du recebe -st. |
| 1.3 | darf | Er usa a mesma forma de ich. |
| 1.4 | dürfen | Wir usa a forma plural. |
| 2.1 | Ich darf heute arbeiten. | O infinitivo vai ao fim. |
| 2.2 | Du darfst Deutsch lernen. | O modal fica na segunda posição. |
| 2.3 | Wir dürfen morgen kommen. | Wir usa forma plural e infinitivo no fim. |
| 3.1 | Ich darf lernen. | Conjugue só o modal. |
| 3.2 | Du darfst gehen. | Du usa a forma própria. |
| 3.3 | Wir dürfen arbeiten. | Wir usa forma plural. |
| 4.1 | Ich darf heute arbeiten. | Modal em segunda posição. |
| 4.2 | Du darfst Deutsch lernen. | Infinitivo no fim. |
| 4.3 | Wir dürfen morgen gehen. | Wir + modal + tempo + infinitivo. |
| 5.1 | arbeiten | Depois do modal, use infinitivo. |
| 5.2 | lernen | O verbo principal não é conjugado. |
| 5.3 | gehen | Infinitivo no fim. |
| 5.4 | kommen | Infinitivo para plural também. |
