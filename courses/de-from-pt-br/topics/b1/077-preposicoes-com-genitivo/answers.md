# Respostas: Preposições com genitivo

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Während | A ideia é "durante o seminário"; **während** pede genitivo. |
| 1.2 | Wegen | A rua está fechada por causa da obra; **wegen** expressa causa. |
| 1.3 | Statt | A frase contrasta hotel e casa de amigos; **statt** significa "em vez de". |
| 1.4 | Außerhalb | A ideia é localização fora da cidade; use **außerhalb**. |
| 1.5 | Trotz | A pessoa vai trabalhar apesar da dor de cabeça; **trotz** expressa contraste. |
| 1.6 | Innerhalb | A frase fala de prazo; **innerhalb** significa "dentro de". |
| 2.1 | des Gewitters | **das Gewitter** é neutro; no genitivo fica **des Gewitters**. |
| 2.2 | der Erklärung | **die Erklärung** é feminino; no genitivo fica **der Erklärung**. |
| 2.3 | des Windes | **der Wind** é masculino; no genitivo padrão fica **des Windes**. |
| 2.4 | eines Saftes | **der Saft** é masculino; com artigo indefinido no genitivo: **eines Saftes**. |
| 2.5 | zweier Tage | Com plural e quantidade, **zweier Tage** é uma forma genitiva adequada. |
| 2.6 | der Parkzone | **die Parkzone** é feminino; no genitivo fica **der Parkzone**. |
| 3.1 | Während des Kurses benutzen wir keine Handys. | **der Kurs** vira **des Kurses** depois de **während**. |
| 3.2 | Wegen des Nebels fährt der Bus langsamer. | **der Nebel** vira **des Nebels** depois de **wegen**. |
| 3.3 | Trotz der wenigen Zeit besuchen wir das Museum. | **die Zeit** é feminino; o genitivo é **der Zeit**, aqui com adjetivo. |
| 3.4 | Statt eines neuen Sofas kaufen wir einen Tisch. | **das Sofa** é neutro; com **statt** usa-se **eines neuen Sofas**. |
| 3.5 | Innerhalb dreier Tage kommt die Antwort. | A expressão marca prazo e usa genitivo plural: **dreier Tage**. |
| 4.1 | es | **der Fehler** fica **des Fehlers** no genitivo. |
| 4.2 | er | **die Hitze** é feminino; genitivo: **der Hitze**. |
| 4.3 | es | **das Interview** é neutro; genitivo: **des Interviews**. |
| 4.4 | er | **die Pause** é feminino; com artigo indefinido: **einer langen Pause**. |
| 4.5 | er | Plural: **der Arbeitszeiten**. |
| 4.6 | es | **der Monat** é masculino; com artigo indefinido: **eines Monats**. |
| 5.1 | Trotz der Probleme bleibt Mia positiv. | **die Probleme** no plural viram **der Probleme** no genitivo. |
| 5.2 | Statt eines Geschenks bringt er Blumen. | **das Geschenk** é neutro; genitivo: **eines Geschenks**. |
| 5.3 | Außerhalb des Gebäudes ist Rauchen erlaubt. | **das Gebäude** é neutro; genitivo: **des Gebäudes**. |
| 5.4 | Wegen der Nachricht ist Tom nervös. | **die Nachricht** é feminino; genitivo: **der Nachricht**. |
| 6.1 | Während der Besprechung diskutieren wir den Plan. | **die Besprechung** é feminino; depois de **während** fica **der Besprechung**. |
| 6.2 | Wegen des Lärms schließe ich das Fenster. | **der Lärm** é masculino; genitivo: **des Lärms**. |
| 6.3 | Trotz des Regens läuft sie im Park. | **der Regen** é masculino; genitivo: **des Regens**. |
| 6.4 | Statt einer langen Antwort schreibt er einen kurzen Satz. | **die Antwort** é feminino; genitivo com indefinido: **einer langen Antwort**. |
| 6.5 | Außerhalb der Universität arbeitet er in einem Geschäft. | **die Universität** é feminino; genitivo: **der Universität**. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1 | A escola fica fora da cidade: **außerhalb der Stadt**. |
| 2 | Ela acorda cedo por causa do caminho longo: **wegen des langen Weges**. |
| 3 | Ela organiza uma gincana pela cidade, não uma aula normal. |
| 4 | Eles devem encontrar cinco lugares e escrever frases curtas em alemão dentro de uma hora. |
| 5 | Alguns querem voltar por causa da chuva: **wegen des Regens**. |
| 6 | Não. Apesar do mau tempo, cada grupo encontra pelo menos quatro lugares. |
| 7 | Exemplo: **Trotz der Müdigkeit freut sie sich auf die Kurse**; a ideia é continuar animada apesar do cansaço. |
| 8 | Exemplo possível: **Wegen des Regens bleibt Mara ruhig und erklärt den Plan noch einmal.** A resposta deve usar **wegen** com genitivo. |
| Prática 1 | **außerhalb**; indica localização fora da cidade. |
| Prática 2 | **der**; **die erste Woche** vira **der ersten Woche** no genitivo. |
| Prática 3 | **Wegen des langen Weges steht Mara früh auf.** Usa causa com genitivo. |
| Prática 4 | **Trotz des Regens arbeiten die Gruppen weiter.** Usa contraste com genitivo. |
| Prática 5 | **Innerhalb einer Stunde finden sie vier Orte.** Usa prazo com genitivo. |
| Prática 6 | **Statt eines Tests machen sie eine Aktivität in der Stadt.** Usa substituição com genitivo. |
