# Respostas: Orações relativas no acusativo e dativo

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | den | **der Film** é masculino e é objeto direto de **gesehen habe**; por isso acusativo masculino **den**. |
| 1.2 | die | **die E-Mail** é feminino acusativo; a forma fica igual a **die**. |
| 1.3 | das | **das Restaurant** é neutro acusativo; o relativo correto é **das**. |
| 1.4 | die | **die Fragen** é plural acusativo; usa-se **die**. |
| 1.5 | den | **der Laptop** é masculino e objeto direto de **gekauft habe**; usa-se **den**. |
| 2.1 | dem | **helfen** pede dativo; com **der Nachbar** o relativo é **dem**. |
| 2.2 | der | **danken** pede dativo; com feminino singular usa-se **der**. |
| 2.3 | dem | **gehören** pede dativo; **das Mädchen** é neutro, então **dem**. |
| 2.4 | denen | **antworten** pede dativo; plural dativo do relativo é **denen**. |
| 2.5 | dem | **vertrauen** pede dativo; **der Freund** vira **dem**. |
| 3.1 | den | **anrufen** usa objeto direto; masculino acusativo é **den**. |
| 3.2 | die | **grüßen** usa acusativo; feminino acusativo permanece **die**. |
| 3.3 | denen | **helfen** pede dativo e **die Kinder** é plural; forma correta: **denen**. |
| 3.4 | dem | **Zeit geben** pede dativo para quem recebe; **das Projekt** é neutro, então **dem**. |
| 3.5 | den | **besuchen** usa acusativo; **der Kurs** vira **den**. |
| 3.6 | der | **vertrauen** pede dativo; feminino singular dativo é **der**. |
| 4.1 | auf den | **warten auf** pede acusativo; **der Mann** vira **den**. |
| 4.2 | für die | **für** sempre pede acusativo; **die Firma** fica **die**. |
| 4.3 | in dem | Localização com **in** pede dativo; **das Zimmer** vira **dem**. |
| 4.4 | mit denen | **mit** pede dativo; plural relativo dativo é **denen**. |
| 4.5 | über das | **diskutieren über** pede acusativo; **das Thema** fica **das**. |
| 5.1 | Der Mann, den ich sehe, ist mein Onkel. | **den ich sehe** retoma **der Mann** como objeto direto. |
| 5.2 | Die Studentin, der ich helfe, wohnt neben mir. | **helfen** pede dativo; feminino singular é **der**. |
| 5.3 | Das Museum, das wir besuchen, ist neu. | **besuchen** usa acusativo; neutro acusativo é **das**. |
| 5.4 | Die Freunde, denen sie schreibt, leben in Bremen. | **schreiben** para alguém usa dativo; plural dativo é **denen**. |
| 5.5 | Der Techniker, mit dem ich spreche, repariert den Drucker. | **mit** pede dativo; **der Techniker** vira **dem**. |
| 6.1 | Der Kunde, den ich beraten habe, ist zufrieden. | **beraten** usa acusativo; masculino acusativo é **den**. |
| 6.2 | Die Nachbarin, der ich geholfen habe, ist krank. | **helfen** pede dativo; feminino singular dativo é **der**. |
| 6.3 | Das Auto, mit dem wir fahren, ist alt. | **mit** pede dativo; **das Auto** vira **dem**. |
| 6.4 | Die Kollegen, mit denen ich arbeite, sind nett. | Depois de **mit** no plural, o relativo é **denen**. |
| 6.5 | Der Brief, der auf dem Tisch liegt, ist wichtig. | Aqui o relativo é sujeito de **liegt**, então nominativo masculino **der**. |
| 7.1 | Die Frau, die ich kenne, arbeitet in der Bank. | **kennen** usa acusativo; feminino acusativo é **die**. |
| 7.2 | Der Mann, dem ich helfe, wohnt in München. | **helfen** pede dativo; masculino dativo é **dem**. |
| 7.3 | Das Buch, über das wir sprechen, ist interessant. | **sprechen über** pede acusativo; neutro acusativo é **das**. |
| 7.4 | Die Leute, mit denen wir reisen, sind freundlich. | **mit** pede dativo e **die Leute** é plural; usa-se **denen**. |
| 7.5 | Der Kurs, den ich besuche, beginnt um neun Uhr. | **besuchen** usa acusativo; masculino acusativo é **den**. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Welchen Verein besucht Lara? | Lara besucht einen Sprachverein. Em **den eine Freundin empfohlen hat**, **den** retoma **Sprachverein** no acusativo. |
| 2. Wen kennt Lara schon aus der Bibliothek? | Sie kennt Herrn Weber schon aus der Bibliothek. **den sie kennt** usa acusativo. |
| 3. Wem hilft Lara später bei der Anmeldung? | Sie hilft einer Studentin bei der Anmeldung. Como **helfen** pede dativo, aparece **der**. |
| 4. Mit wem spricht Lara kurz über die Regeln? | Sie spricht kurz mit einem Mitarbeiter. Na relativa: **mit dem Lara spricht**. |
| 5. In welchem Raum findet der Deutschkurs statt? | Er findet im Kursraum statt. Localização com **in** usa dativo: **in dem**. |
| 6. Über welchen Ordner hat die Lehrerin am Morgen gesprochen? | Sie hat über den Ordner gesprochen, den Lara danach sucht. **über den** retoma **der Ordner** no acusativo. |
| 7. Für welches Projekt wollen die Teilnehmerinnen zusammenarbeiten? | Sie wollen für ein gemeinsames Projekt zusammenarbeiten. Na relativa: **für das sie zusammenarbeiten wollen**. |
| 8. Com base na história, complete: Die Leute, mit ___ Lara gesprochen hat, wirken motiviert. | **denen**. **mit** pede dativo e **die Leute** está no plural. |
| Prática 1 | Lara kennt Herrn Weber, der in der Bibliothek ist. Aqui o relativo é sujeito de **ist**, então nominativo masculino **der**. |
| Prática 2 | der | **helfen** pede dativo; **die Studentin** vira **der**. |
| Prática 3 | dem | Localização com **in** pede dativo; **der Kursraum** vira **dem**. |
| Prática 4 | den | **sprechen über** pede acusativo; **der Ordner** vira **den**. |
| Prática 5 | **denen** aparece porque **zeigen** dá algo a alguém; os novos membros recebem a informação e estão no plural dativo. |
| Prática 6 | Exemplo: **Der Mitarbeiter, mit dem Lara spricht, ist freundlich.** Usa-se **mit dem** porque **mit** pede dativo e **Mitarbeiter** é masculino. |
