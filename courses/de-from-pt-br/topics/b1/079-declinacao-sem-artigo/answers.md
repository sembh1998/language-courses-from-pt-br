# Respostas: Declinação sem artigo

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | er | `Wein` é masculino nominativo sem artigo: `guter Wein`. |
| 1.2 | es | `Wasser` é neutro nominativo sem artigo: `kaltes Wasser`. |
| 1.3 | e | `Musik` é feminino nominativo sem artigo: `deutsche Musik`. |
| 1.4 | e | Plural nominativo sem artigo usa `-e`: `junge Leute`. |
| 1.5 | es | `Brot` é neutro nominativo sem artigo: `frisches Brot`. |
| 2.1 | en | `Kaffee` é masculino acusativo como objeto de `bestelle`: `schwarzen Kaffee`. |
| 2.2 | es | `Gemüse` é neutro acusativo sem artigo: `frisches Gemüse`. |
| 2.3 | e | `Musik` é feminino acusativo sem artigo: `moderne Musik`. |
| 2.4 | e | `Informationen` está no plural acusativo sem artigo: `wichtige Informationen`. |
| 2.5 | en | `Rat` é masculino acusativo sem artigo: `ruhigen Rat`. |
| 3.1 | Schwarzer | `Kaffee` é sujeito masculino nominativo, então `schwarzer`. |
| 3.2 | schwarzen Kaffee | Depois de `trinke`, `Kaffee` é masculino acusativo: `schwarzen Kaffee`. |
| 3.3 | frischem Gemüse | `mit` pede dativo; `Gemüse` é neutro: `frischem Gemüse`. |
| 3.4 | Mit deutscher Musik arbeitet er gern. | `mit` pede dativo; `Musik` é feminino: `deutscher Musik`. |
| 3.5 | Mit guten Ideen lösen wir das Problem. | Plural dativo sem artigo usa `-en`: `guten Ideen`. |
| 3.6 | schlechten Wetters | `wegen` costuma pedir genitivo; `Wetter` é neutro: `schlechten Wetters`. |
| 4.1 | Guter Wein ist teuer. | Sem artigo, masculino nominativo usa `-er`. |
| 4.2 | Ich trinke schwarzen Kaffee. | Sem artigo, masculino acusativo usa `-en`. |
| 4.3 | Kaltes Wasser steht dort. | Sem artigo, neutro nominativo usa `-es`. |
| 4.4 | Wir hören deutsche Musik. | Sem artigo, feminino acusativo usa `-e`. |
| 4.5 | Sie reist mit guten Freunden. | `mit` pede dativo plural: adjetivo `guten` e plural `Freunden`. |
| 5.1 | Guter Wein schmeckt mir. | Faltava a terminação forte `-er` no masculino nominativo. |
| 5.2 | Ich brauche guten Rat. | `Rat` é masculino acusativo sem artigo: `guten`. |
| 5.3 | Kaltes Wasser ist im Glas. | `Wasser` é neutro nominativo, não feminino: `kaltes`. |
| 5.4 | Mit deutscher Musik lerne ich besser. | `mit` pede dativo; feminino sem artigo usa `-er`. |
| 5.5 | Wir sprechen mit guten Freunden. | Dativo plural sem artigo: `guten Freunden`. |
| 5.6 | Wegen schlechten Wetters kommt er später. | Genitivo neutro sem artigo: `schlechten Wetters`. |
| 6.1 | Guter Wein passt zu Käse. | `Wein` é masculino nominativo sem artigo: `guter`. |
| 6.2 | Ich trinke schwarzen Kaffee. | `Kaffee` é masculino acusativo como objeto direto: `schwarzen`. |
| 6.3 | Kaltes Wasser hilft im Sommer. | `Wasser` é neutro nominativo sem artigo: `kaltes`. |
| 6.4 | Sie mag deutsche Musik. | `Musik` é feminino acusativo sem artigo: `deutsche`. |
| 6.5 | Mit guten Freunden wird die Reise besser. | `mit` pede dativo plural: `guten Freunden`. |
| 6.6 | Wegen schlechten Wetters bleiben wir zu Hause. | `wegen` + genitivo neutro: `schlechten Wetters`. |
| 7.1 | guter Planung | `mit` pede dativo; `Planung` é feminino: `guter Planung`. |
| 7.2 | kaltem Wasser | `aus` pede dativo; `Wasser` é neutro: `kaltem Wasser`. |
| 7.3 | starken Regens | `wegen` pede genitivo; `Regen` é masculino: `starken Regens`. |
| 7.4 | guter Freunde | Genitivo plural sem artigo usa `-er`: `guter Freunde`. |
| 7.5 | frisches Brot | `für` pede acusativo; `Brot` é neutro: `frisches Brot`. |

|---|---|
| 1 | Clara vai cedo à feira porque quer comprar pão fresco e bom queijo. |
| 2 | Ela toma café preto: `schwarzen Kaffee`. |
| 3 | Não. Ela não compra vinho, mas leva água fria para o passeio. |
| 4 | Ele quase chegou tarde por causa do mau tempo: `wegen schlechten Wetters`. |
| 5 | Ela preparou pão, legumes, comida quente e música alemã no fundo. |
| 6 | Ela quer fazer uma sopa com legumes frescos. |
| 7 | Ela escreve: boa comida, água fria, boa música e tempo suficiente. |
| 8 | Uma resposta possível é `mit guten Freunden`. |

| Prática 4 | mit guten Freunden; plural dativo com `mit`. |
