# História: Vorgangspassiv no presente

Nível: B1

## História em alemão

Im Stadtteil Linden wird seit Monaten ein altes Kino renoviert. Jeden Morgen werden große Fenster geputzt, neue Kabel gelegt und alte Sitze entfernt. Das Gebäude wird von einer kleinen Baufirma modernisiert, aber viele Ideen kommen von den Bewohnern. Im Eingang wird ein Plan aufgehängt, damit alle sehen können, was gerade gemacht wird.

Am Mittwoch wird die alte Bühne von zwei Technikern geprüft. Durch eine neue Lichtanlage wird der Raum heller und freundlicher gemacht. Die Wände werden gestrichen, und im Café wird schon eine kleine Speisekarte vorbereitet. Viele Nachbarn sind neugierig, weil das Kino bald wieder als Kulturzentrum genutzt wird.

Lisa besucht die Baustelle mit ihrer Klasse. Der Projektleiter erklärt, dass nicht alles sofort entschieden wird. Manche Fragen werden von der Stadtverwaltung geprüft, andere Probleme werden durch Gespräche mit den Anwohnern gelöst. Lisa findet es interessant, dass ein Ort nicht nur gebaut, sondern gemeinsam verändert wird.

## Notas de vocabulário

- renoviert = reformado; renovado
- entfernt = removido
- aufgehängt = pendurado
- geprüft = verificado; examinado
- die Lichtanlage = o sistema de iluminação
- gestrichen = pintado, no contexto de paredes
- die Stadtverwaltung = a administração municipal; prefeitura
- die Anwohner = os moradores da região
- gelöst = resolvido
- verändert = transformado; mudado
