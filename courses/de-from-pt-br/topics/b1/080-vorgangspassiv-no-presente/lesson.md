# Vorgangspassiv no presente

Nível: B1

## Objetivo

Entender e usar o Vorgangspassiv no presente para dizer que uma ação está sendo realizada, sem colocar o foco em quem faz a ação.

## Explicação

Em alemão, o **Vorgangspassiv** descreve um processo ou uma ação em andamento do ponto de vista do objeto afetado. No presente, ele é formado com:

**werden no presente + Partizip II**

O Partizip II fica no final da frase ou da oração.

| Ativa | Passiva | Ideia |
|---|---|---|
| Die Firma baut das Haus. | Das Haus wird gebaut. | A casa está sendo construída. |
| Der Techniker repariert den Computer. | Der Computer wird repariert. | O computador está sendo consertado. |
| Viele Menschen lesen den Artikel. | Der Artikel wird von vielen Menschen gelesen. | O artigo é lido por muitas pessoas. |

Conjugação de **werden** no presente:

| Pessoa | werden |
|---|---|
| ich | werde |
| du | wirst |
| er/sie/es | wird |
| wir | werden |
| ihr | werdet |
| sie/Sie | werden |

Na passiva, o objeto direto da frase ativa vira sujeito da frase passiva:

- Ativa: **Der Koch** kocht **die Suppe**.
- Passiva: **Die Suppe** wird gekocht.

O agente pode aparecer, mas muitas vezes é omitido. Use:

- **von + dativo** para pessoas, grupos, instituições ou causas vistas como agente direto.
- **durch + acusativo** para meio, instrumento, processo ou causa intermediária.

Exemplos:

- Der Brief wird **von Maria** geschrieben.
- Das Fenster wird **durch den Wind** geöffnet.
- Die Entscheidung wird **von der Regierung** getroffen.
- Die Nachricht wird **durch eine App** gesendet.

## Exemplos

| Alemão | Tradução | Observação |
|---|---|---|
| Das Haus wird gebaut. | A casa está sendo construída. | **wird + gebaut** |
| Die Tür wird geöffnet. | A porta está sendo aberta. | O foco é a porta, não a pessoa. |
| Die E-Mails werden jeden Morgen gelesen. | Os e-mails são lidos todas as manhãs. | Plural: **werden** |
| Der Patient wird vom Arzt untersucht. | O paciente é examinado pelo médico. | **vom = von dem** |
| Der Vertrag wird durch eine digitale Unterschrift bestätigt. | O contrato é confirmado por uma assinatura digital. | **durch** indica meio. |
| Wird das Problem heute gelöst? | O problema será resolvido hoje? | Pergunta: verbo conjugado em primeiro lugar. |

## Erros comuns

- Usar **sein** em vez de **werden** para uma ação: **Das Haus wird gebaut**, não **Das Haus ist gebaut** quando o foco é o processo.
- Esquecer que o Partizip II vai para o final: **Der Computer wird repariert**, não **Der Computer wird repariert den Techniker**.
- Manter o objeto direto como objeto: na passiva, ele vira sujeito. **Den Computer wird repariert** está errado; correto: **Der Computer wird repariert**.
- Confundir **von** e **durch**: normalmente **von Maria**, mas **durch eine Maschine**.

## Prática guiada

Transforme mentalmente seguindo três passos:

1. Encontre o objeto direto da frase ativa: **Die Schüler lesen den Text**.
2. Coloque esse objeto como sujeito: **Der Text**.
3. Use **werden + Partizip II**: **Der Text wird gelesen**.

Mais padrões úteis:

| Ativa | Passiva |
|---|---|
| Man bezahlt die Rechnung online. | Die Rechnung wird online bezahlt. |
| Jemand schließt das Büro um 18 Uhr. | Das Büro wird um 18 Uhr geschlossen. |
| Die Polizei kontrolliert die Straße. | Die Straße wird von der Polizei kontrolliert. |
| Ein Programm übersetzt den Text. | Der Text wird durch ein Programm übersetzt. |

Mini-checagem:

- **Die Fahrräder werden repariert.** Quem ou o quê está no foco? Os bicicletas; a pessoa que conserta não é importante.
- **Der Kurs wird von Frau Schneider geleitet.** Por que **von**? Porque Frau Schneider é a agente direta.
- **Die Tür wird durch einen Code geöffnet.** Por que **durch**? Porque o código é o meio usado para abrir.

## Resumo

No Vorgangspassiv presente, use **werden + Partizip II** para destacar a ação sofrida pelo sujeito. O agente pode ser omitido ou introduzido com **von**; meios e instrumentos geralmente aparecem com **durch**.
