# Respostas: Argumentação simples

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Meinung | A expressão fixa é `Ich bin der Meinung, dass ...`. |
| 1.2 | nach | A expressão fixa é `Meiner Meinung nach ...`. |
| 1.3 | finde | `Ich finde, dass ...` introduz uma opinião pessoal. |
| 1.4 | denke | `Deshalb denke ich, dass ...` liga conclusão e opinião. |
| 2.1 | Ich bin der Meinung, dass ein Sprachkurs nützlich ist. | Em oração com `dass`, o verbo conjugado vai ao final. |
| 2.2 | Meiner Meinung nach ist die App hilfreich. | `Meiner Meinung nach` ocupa a primeira posição, então o verbo vem logo depois. |
| 2.3 | Ein Nachteil ist, dass man wenig Zeit hat. | Com `dass`, o verbo `hat` fica no final da oração. |
| 2.4 | Einerseits ist das Projekt interessant, andererseits ist es stressig. | A estrutura apresenta dois lados com ordem verbal correta. |
| 3.1 | Ich finde, dass Online-Unterricht flexibel ist. | A opinião usa `dass` e coloca `ist` no final. |
| 3.2 | Ich bin der Meinung, dass Sport wichtig für die Gesundheit ist. | A expressão pede uma oração subordinada com verbo no final. |
| 3.3 | Meiner Meinung nach braucht die Stadt mehr Radwege. | Depois da expressão inicial, o verbo `braucht` vem na segunda posição. |
| 3.4 | Deshalb denke ich, dass wir weniger Plastik benutzen sollten. | Com `dass`, o modal `sollten` fica no final. |
| 4.1 | Vorteil | `Ein Vorteil ist, dass ...` introduz algo positivo. |
| 4.2 | Nachteil | `Ein Nachteil ist, dass ...` introduz algo negativo. |
| 4.3 | Einerseits | A dupla correta é `Einerseits ..., andererseits ...`. |
| 4.4 | andererseits | `andererseits` mostra o segundo lado do contraste. |
| 5.1 | Meiner Meinung nach sollte man jeden Tag ein bisschen Deutsch lernen. | Resposta-modelo correta; outras respostas semelhantes com a expressão também podem valer. |
| 5.2 | Ein Vorteil ist, dass man keinen langen Arbeitsweg hat. | Resposta-modelo que apresenta uma vantagem com `dass` e verbo no final. |
| 5.3 | Ein Nachteil ist, dass man die Produkte nicht direkt sehen kann. | Resposta-modelo que apresenta uma desvantagem com modal no final. |
| 5.4 | Einerseits ist das Leben im Zentrum praktisch, andererseits ist es oft laut. | A frase usa a estrutura de dois lados de forma adequada. |
| 6.1 | Ich bin der Meinung, dass ... | Essa expressão serve para dar opinião de forma mais formal. |
| 6.2 | Ein Vorteil ist, dass ... | Essa expressão introduz um ponto positivo. |
| 6.3 | Ein Nachteil ist, dass ... | Essa expressão introduz um ponto negativo. |
| 6.4 | Einerseits ..., andererseits ... | Essa estrutura apresenta contraste equilibrado. |
