# Respostas: Konjunktiv II: Desejos e hipóteses

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | hätte | Com `wenn` em hipótese irreal, usa-se Konjunktiv II; `ich` com `haben` vira `hätte`. |
| 1.2 | wäre | Para desejo com estado/lugar, usa-se `wäre gern`. |
| 1.3 | wären | O sujeito é `wir`; a forma de `sein` no Konjunktiv II é `wären`. |
| 1.4 | würdest | Pergunta com `du` e `gern` usa `würdest du ... lernen?`. |
| 1.5 | hättest | O sujeito é `du`; `haben` no Konjunktiv II vira `hättest`. |
| 1.6 | hätten | O sujeito é `wir`; `hätten gern` expressa desejo de ter algo. |
| 2.1 | Wenn ich Urlaub hätte, würde ich in die Schweiz reisen. | A oração com `wenn` coloca `hätte` no final e a oração principal usa `würde + infinitivo`. |
| 2.2 | Ich wäre gern zu Hause. | Com `sein`, a forma natural é `wäre`, não `würde ... sein` em uma frase simples. |
| 2.3 | Wir würden morgen einen Ausflug machen. | Depois de `würden`, o verbo principal fica no infinitivo no fim. |
| 2.4 | Wenn er Geld hätte, würde er umziehen. | Em oração com `wenn`, o verbo conjugado fica no final: `Geld hätte`. |
| 2.5 | Sie würde lieber später anfangen. | `lieber` expressa preferência e `anfangen` fica no infinitivo. |
| 2.6 | Ich hätte Zeit. | Para `teria`, `hätte` é a forma mais natural e frequente. |
| 3.1 | Wenn ich Zeit hätte, würde ich mehr Bücher lesen. | `habe` vira `hätte`; a ação hipotética usa `würde ... lesen`. |
| 3.2 | Wenn Lena Geld hätte, würde sie ein Fahrrad kaufen. | A condição irreal usa `hätte`; a consequência usa `würde + kaufen`. |
| 3.3 | Wenn wir in Köln wären, würden wir den Dom besuchen. | Para lugar/estado, `sein` vira `wären`. |
| 3.4 | Wenn du nicht müde wärst, würdest du mitkommen. | Com `du`, use `wärst` e `würdest`. |
| 3.5 | Wenn ich besser Deutsch könnte, würde ich mit den Nachbarn sprechen. | O modal `können` vira `könnte`; `sprechen` fica no infinitivo. |
| 4.1 | Wenn ich Zeit hätte, würde ich Sport machen. | O objeto `Zeit` vem antes de `hätte` no final da oração subordinada. |
| 4.2 | Ich wäre gern in Hamburg. | Para `gostaria de estar`, use diretamente `wäre gern`. |
| 4.3 | Wir würden gern das Museum besuchen. | O infinitivo `besuchen` vai para o final da oração. |
| 4.4 | Wenn sie mehr Geld hätte, würde sie reisen. | A frase fala de hipótese irreal; por isso `hat` vira `hätte`. |
| 4.5 | Du würdest lieber zu Hause bleiben. | Depois de `würdest`, o verbo fica no infinitivo: `bleiben`. |
| 5.1 | Ich hätte gern mehr Freizeit. | `hätte gern` expressa “gostaria de ter”. |
| 5.2 | Wenn wir in Berlin wären, würden wir das Museum besuchen. | `wären` expressa estar em Berlim como hipótese; `würden ... besuchen` mostra a consequência. |
| 5.3 | Könntest du mir morgen helfen? | `könntest` é a forma educada/hipotética para `você poderia`. |
| 5.4 | An deiner Stelle würde ich jeden Tag Deutsch lernen. | `An deiner Stelle` corresponde a “no seu lugar” e pede `würde + infinitivo`. |
| 5.5 | Sie würde lieber im Zentrum wohnen. | `lieber` indica preferência e `wohnen` fica no infinitivo. |
| 6.1 | Ich würde ins Kino gehen. | Resposta curta adequada com `würde + infinitivo`. |
| 6.2 | Ich wäre jetzt gern am Strand. | Desejo de estar em um lugar usa `wäre gern`. |
| 6.3 | Ich hätte gern eine größere Küche. | Desejo de ter algo usa `hätte gern`. |
| 6.4 | Ich würde lieber Tee trinken. | Preferência usa `lieber` e infinitivo no final. |
| 7.1 | B | `Ich wäre gern in Wien` significa “Eu gostaria de estar em Viena”. |
| 7.2 | A | `Wir hätten mehr Zeit` significa “Nós teríamos mais tempo”. |
| 7.3 | B | `An deiner Stelle würde ich anrufen` significa “No seu lugar, eu ligaria”. |
| 7.4 | A | `Würdest du mitkommen?` significa “Você iria junto?”. |
