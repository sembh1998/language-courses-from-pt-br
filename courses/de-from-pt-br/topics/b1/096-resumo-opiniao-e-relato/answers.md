# Respostas: Resumo, opinião e relato

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | In dem Artikel geht es um die Nutzung von Handys in der Schule. | A frase apresenta o tema de forma neutra, adequada para resumo. |
| 1.2 | Meiner Meinung nach sollten Kinder mehr draußen spielen. | `Meiner Meinung nach` introduz opinião pessoal. |
| 1.3 | Vor einem Jahr habe ich an einem Sprachkurs teilgenommen. | A frase começa um relato com tempo e experiência pessoal. |
| 2.1 | weil | `weil` introduz o motivo e leva o verbo conjugado ao fim. |
| 2.2 | Trotzdem | Mostra contraste: foi cansativo, mas houve aprendizado. |
| 2.3 | danach | Indica sequência depois de ler o texto. |
| 2.4 | jedoch | Apresenta contraste entre uso do carro e vantagem da bicicleta. |
| 2.5 | deshalb | Indica consequência: pouco tempo levou a um relato curto. |
| 3.1 | In dem Text geht es um Arbeit von zu Hause. | É uma abertura neutra de resumo com o tema indicado. |
| 3.2 | Meiner Meinung nach ist der öffentliche Verkehr nützlich. | A expressão introduz opinião de forma clara. |
| 3.3 | Ich finde das gut, weil es billig ist. | A opinião é justificada com `weil`. |
| 3.4 | Zusammenfassend kann man sagen, dass es Vorteile und Nachteile gibt. | A frase funciona como conclusão de resumo. |
| 4.1 | B D A C | A ordem natural é situação, primeira ação, segunda ação e resultado. |
| 4.2 | B C A D | Começa com opinião, dá exemplo, acrescenta vantagem e conclui. |
| 4.3 | B D A C | Primeiro apresenta o tema, depois vantagens, desvantagens e conclusão. |
| 5.1 | resumo | A frase relata o que o autor diz, sem opinião pessoal. |
| 5.2 | opinião | `Ich bin der Meinung` marca claramente uma opinião. |
| 5.3 | relato | A frase conta uma experiência pessoal em sequência temporal. |
| 5.4 | resumo | `wird erklärt` apresenta informação de um texto. |
| 5.5 | relato | A frase reflete sobre uma experiência vivida. |
| 6.1 | In dem Text geht es um gesundes Essen. | Resposta modelo: resume o tema com frase neutra. |
| 6.2 | Ich finde Deutschlernen wichtig, weil es neue Chancen bringt. | Resposta modelo: opinião com justificativa usando `weil`. |
| 6.3 | Zuerst war ich unsicher, dann habe ich viel geübt. | Resposta modelo: relato curto com sequência. |
| 6.4 | Deshalb denke ich, dass mehr Menschen Fahrrad fahren sollten. | Resposta modelo: conclusão de opinião com consequência. |

| 3 | Sie findet es sinnvoll, weil man helfen und neue Erfahrungen sammeln kann. |
| 4 | Sie sagt, dass man genug Zeit braucht. |
| 5 | Sie hat in einem Tierheim geholfen. |
| 6 | Sie war unsicher, weil sie die Arbeit nicht kannte. |
