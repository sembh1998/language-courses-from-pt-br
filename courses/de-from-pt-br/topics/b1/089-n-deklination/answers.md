# Respostas: N-Deklination

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Studenten | `den` indica acusativo; `der Student` vira `den Studenten`. |
| 1.2 | Patienten | Depois de `helfen` usamos dativo; `dem Patienten` recebe `-en`. |
| 1.3 | Kollegen | `anrufen` pede acusativo; `der Kollege` vira `den Kollegen`. |
| 1.4 | Jungen | Depois de `erklären` para a pessoa usamos dativo; `dem Jungen`. |
| 1.5 | Menschen | `mit` pede dativo; `der Mensch` vira `dem Menschen`. |
| 1.6 | Kunden | `beraten` pede acusativo; `der Kunde` vira `den Kunden`. |
| 2.1 | Ich treffe den Studenten im Park. | No acusativo singular, `Student` recebe `-en`. |
| 2.2 | Wir helfen dem Kollegen. | `helfen` pede dativo e `Kollege` recebe `-n`. |
| 2.3 | Die Mutter sucht den Jungen. | `suchen` pede acusativo; `der Junge` vira `den Jungen`. |
| 2.4 | Der Mensch braucht Zeit. | No nominativo singular não se adiciona `-en`. |
| 2.5 | Das Handy des Kollegen ist neu. | No genitivo singular, `Kollege` recebe `-n`. |
| 2.6 | Ich begrüße den Herrn Schneider. | `der Herr` vira `den Herrn` no acusativo. |
| 3.1 | den Studenten | `fragen` usa acusativo; `der Student` vira `den Studenten`. |
| 3.2 | dem Kollegen | `antworten` pede dativo; `der Kollege` vira `dem Kollegen`. |
| 3.3 | des Menschen | A expressão indica genitivo: `des Menschen`. |
| 3.4 | den Jungen | `loben` pede acusativo; `der Junge` vira `den Jungen`. |
| 3.5 | dem Kunden | A empresa escreve para o cliente; dativo: `dem Kunden`. |
| 3.6 | dem Nachbarn | `danken` pede dativo; `der Nachbar` vira `dem Nachbarn`. |
| 4.1 | Ich kenne den Menschen aus dem Büro. | `kennen` pede acusativo; `Mensch` recebe `-en`. |
| 4.2 | Die Lehrerin hilft dem Studenten. | `helfen` pede dativo; `Student` recebe `-en`. |
| 4.3 | Er spricht mit dem Kollegen aus Berlin. | `mit` pede dativo; `Kollege` recebe `-n`. |
| 4.4 | Das Fahrrad des Jungen ist rot. | Genitivo de `der Junge`: `des Jungen`. |
| 4.5 | Die Sekretärin ruft den Herrn Müller an. | `der Herr` vira `den Herrn` no acusativo. |
| 5.1 | nominativo | `Der Student` é sujeito da frase. |
| 5.2 | acusativo | `den Patienten` é objeto direto de `besuchen`. |
| 5.3 | dativo | `dem Kunden` recebe a resposta; é dativo. |
| 5.4 | genitivo | `des Kollegen` indica posse. |
| 5.5 | nominativo | `Der Junge` é sujeito da frase. |
| 5.6 | dativo | `zuhören` pede dativo: `dem Menschen`. |
| 6.1 | Ich helfe dem Studenten. | `helfen` pede dativo; por isso `dem Studenten`. |
| 6.2 | Sie trifft den Kollegen. | `treffen` pede acusativo; `Kollege` recebe `-n`. |
| 6.3 | Wir fragen den Kunden. | `fragen` pede acusativo; `Kunde` recebe `-n`. |
| 6.4 | Die E-Mail des Herrn ist kurz. | Genitivo de `der Herr`: `des Herrn`. |
| 6.5 | Der Lehrer erklärt dem Jungen die Regel. | A pessoa que recebe a explicação fica no dativo: `dem Jungen`. |
| 7.1 | Ich sehe den Studenten. | Resposta modelo; qualquer frase correta com `den Studenten` é aceitável. |
| 7.2 | Ich helfe dem Kollegen. | Resposta modelo; qualquer frase correta com `dem Kollegen` é aceitável. |
| 7.3 | Die Würde des Menschen ist wichtig. | Resposta modelo; `des Menschen` é genitivo correto. |
| 7.4 | Die Mutter ruft den Jungen. | Resposta modelo; `den Jungen` é acusativo correto. |
