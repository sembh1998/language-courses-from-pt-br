# Respostas: Vorgangspassiv em outros tempos

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | wurde | **Die alte Fabrik** é singular; no Präteritum passivo usa-se **wurde**. |
| 1.2 | wurden | **Die Gäste** é plural; por isso a forma correta é **wurden**. |
| 1.3 | wurde | **Der Vertrag** é singular e a frase está no passado narrativo. |
| 1.4 | wurden | **Wir** pede **wurden** no Präteritum de werden. |
| 1.5 | wurdest | Com **du**, a forma no Präteritum é **wurdest**. |
| 2.1 | ist | **Das Formular** é singular; no Perfekt passivo: **ist + particípio + worden**. |
| 2.2 | worden | No Perfekt do Vorgangspassiv, a última palavra é **worden**, não **geworden**. |
| 2.3 | ist | **Der Fehler** é singular; por isso usamos **ist**. |
| 2.4 | sind | **Die Pakete** é plural; no Perfekt passivo usa-se **sind**. |
| 2.5 | worden | A estrutura correta é **sind erklärt worden**. |
| 3.1 | Das Museum wurde renoviert. | Präteritum passivo: **wurde + Partizip II**. |
| 3.2 | Die Straße ist repariert worden. | Perfekt passivo usa **ist + Partizip II + worden**. |
| 3.3 | Die App wird nächste Woche getestet werden. | **nächste Woche** indica futuro; no passivo: **wird + Partizip II + werden**. |
| 3.4 | Die Briefe wurden verschickt. | **Die Briefe** é plural, então a forma é **wurden**. |
| 3.5 | Das Problem ist gelöst worden. | No Perfekt passivo, usa-se **worden**, não **geworden**. |
| 3.6 | Die Dokumente werden morgen geprüft werden. | **morgen** pede futuro; a estrutura termina com **werden**. |
| 4.1 | Die Wohnungen wurden gebaut. | O objeto ativo **die Wohnungen** vira sujeito plural no passivo. |
| 4.2 | Der Drucker ist repariert worden. | No Perfekt passivo singular: **ist repariert worden**. |
| 4.3 | Die Nachbarn werden informiert werden. | Futur I passivo: **werden + Partizip II + werden**. |
| 4.4 | Der Termin ist verschoben worden. | **Der Termin** vira sujeito e exige **ist** no Perfekt. |
| 4.5 | Die Halle wurde geöffnet. | Präteritum passivo singular: **wurde geöffnet**. |
| 5.1 | passado narrativo; foi controlado ou verificado | **wurde + particípio** marca Präteritum passivo singular. |
| 5.2 | Perfekt passivo; foi controlado ou verificado | **ist + particípio + worden** marca Perfekt passivo singular. |
| 5.3 | futuro passivo; será controlado ou verificado | **wird + particípio + werden** marca Futur I passivo. |
| 5.4 | plural no Präteritum passivo; foram informados | **wurden** é a forma plural no Präteritum. |
| 5.5 | plural no Perfekt passivo; foram informados | **sind ... worden** é Perfekt passivo com sujeito plural. |
| 6.1 | Die Tür ist geschlossen worden. | Troca-se **geworden** por **worden** no Perfekt passivo. |
| 6.2 | Der Bericht wurde gestern geschrieben. | No Präteritum passivo não se usa **worden**. |
| 6.3 | Die Kunden werden informiert werden. | **Die Kunden** é plural; no futuro passivo usa-se **werden**. |
| 6.4 | Das Paket ist geliefert worden. | No Perfekt passivo, o final correto é **worden**. |
| 6.5 | Die Fotos wurden gelöscht. | **Die Fotos** é plural; por isso **wurden**. |
| 7.1 | Die Brücke wurde 2005 gebaut. | Ano passado específico combina bem com Präteritum passivo. |
| 7.2 | Der Termin ist verschoben worden. | Ação concluída no passado em Perfekt passivo. |
| 7.3 | Die Dokumente werden morgen geschickt werden. | **morgen** indica futuro; estrutura: **werden + geschickt + werden**. |
| 7.4 | Das Fenster ist durch den Wind geöffnet worden. | **durch den Wind** indica causa/meio; o Perfekt termina com **worden**. |
| 7.5 | Die Schüler wurden von der Schule informiert. | O agente é expresso com **von + Dativ**. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Was wurde vor zwei Jahren von der Stadt gekauft? | Das Gebäude wurde von der Stadt gekauft. A forma **wurde gekauft** mostra Präteritum passivo. |
| 2. Was wurde nach dem Kauf repariert? | Die Fenster wurden repariert. Como **die Fenster** é plural, aparece **wurden**. |
| 3. Welche Form zeigt Perfekt passiv: „wurde modernisiert“ oder „ist modernisiert worden“? | **ist modernisiert worden**. A sequência **ist + particípio + worden** marca Perfekt passivo. |
| 4. Was wird nächsten Monat eingebaut werden? | Die Technik wird nächsten Monat eingebaut werden. É futuro passivo. |
| 5. Warum freuen sich viele Menschen? | Sie freuen sich, weil bald Filme gezeigt und Konzerte organisiert werden werden. A resposta reutiliza futuro passivo. |
| 6. Reescreva no Perfekt passivo: Die Stadt plant eine Ausstellung. | Eine Ausstellung ist geplant worden. O objeto ativo vira sujeito e o Perfekt termina em **worden**. |
| 7. Reescreva no Futur I passivo: Man richtet einen Raum ein. | Ein Raum wird eingerichtet werden. O verbo separável aparece como particípio **eingerichtet**. |
| 8. Encontre uma frase da história com wurden e explique por que está no plural. | Exemplo: **Danach wurden die Fenster repariert**. Usa-se **wurden** porque **die Fenster** é plural. |
| Prática 1 | wurde | **Das Gebäude** é singular e a frase fala de passado. |
| Prática 2 | worden | Perfekt passivo: **ist renoviert worden**. |
| Prática 3 | werden | Futur I passivo: **wird eingebaut werden**. |
| Prática 4 | Die alten Stühle wurden restauriert. | Passivo no Präteritum com sujeito plural: **wurden restauriert**. |
| Prática 5 | Filme werden gezeigt werden. | Futuro passivo plural: **werden + gezeigt + werden**. |
| Prática 6 | **wurde gekauft** é Präteritum passivo narrativo; **ist geplant worden** é Perfekt passivo, com foco em ação concluída. | A diferença principal está no tempo verbal e na forma de **werden**. |
