# Respostas: Declinação após artigo indefinido

Nível: B1

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | er | `Nachbar` é masculino nominativo depois de `ein`; o adjetivo recebe `-er`. |
| 1.2 | e | `Ärztin` é feminino nominativo; depois de `eine`, o adjetivo fica com `-e`. |
| 1.3 | es | `Café` é neutro nominativo; depois de `ein`, use `-es`. |
| 1.4 | e | Plural sem artigo no nominativo recebe `-e`: `junge Leute`. |
| 1.5 | er | `Computer` é masculino nominativo; `ein alter Computer`. |
| 2.1 | en | `Platz` é masculino acusativo com `einen`; o adjetivo recebe `-en`. |
| 2.2 | e | `Jacke` é feminino acusativo; a forma é `eine warme Jacke`. |
| 2.3 | es | `Regal` é neutro acusativo; depois de `ein`, use `-es`. |
| 2.4 | e | Plural sem artigo no acusativo recebe `-e`: `interessante Themen`. |
| 2.5 | en | `Garten` é masculino acusativo: `einen schönen Garten`. |
| 3.1 | Ein guter Lehrer erklärt die Aufgabe. | Masculino nominativo depois de `ein` exige `-er`. |
| 3.2 | Wir kaufen ein neues Sofa. | `Sofa` é neutro acusativo; a forma correta é `ein neues`. |
| 3.3 | Mit guten Ideen schaffen wir das. | Com `mit`, o plural está no dativo; plural dativo sem artigo usa `-en`. |
| 3.4 | Ich lese eine kurze E-Mail. | `E-Mail` é feminino; no acusativo feminino fica `eine kurze`. |
| 3.5 | Er braucht einen zuverlässigen Handwerker. | `Handwerker` é masculino acusativo; use `einen` e adjetivo com `-en`. |
| 3.6 | Alte Häuser kosten viel Geld. | Plural sem artigo no nominativo usa `-e`. |
| 4.1 | Nette Kollegen helfen mir. | No plural sem artigo nominativo, o adjetivo fica com `-e` e o verbo vai ao plural. |
| 4.2 | Ich sehe neue Filme. | Plural sem artigo no acusativo usa `-e`: `neue Filme`. |
| 4.3 | Wichtige Fragen bleiben offen. | `Fragen` é plural sem artigo no nominativo: `wichtige`. |
| 4.4 | Wir haben kleine Probleme. | Plural sem artigo no acusativo: `kleine Probleme`. |
| 4.5 | Sie spricht mit erfahrenen Experten. | `mit` exige dativo; plural dativo sem artigo usa `-en`. |
| 5.1 | Ein guter Freund kommt heute Abend. | Masculino nominativo depois de `ein` precisa de `-er`. |
| 5.2 | Ich brauche einen guten Rat. | `Rat` é masculino acusativo; use `einen guten`. |
| 5.3 | Das ist ein interessantes Angebot. | `Angebot` é neutro; use `ein interessantes`. |
| 5.4 | Mit guten Freunden reist man gern. | `mit` pede dativo; plural dativo: `guten Freunden`. |
| 5.5 | Sie wohnt in einer ruhigen Wohnung. | Localização com `in` usa dativo; `Wohnung` é feminino: `einer ruhigen`. |
| 5.6 | Wir lesen interessante Artikel in einer alten Zeitung. | `Artikel` é plural acusativo sem artigo; `Zeitung` após `in` local é dativo feminino. |
| 6.1 | Ein netter Kollege arbeitet mit mir. | `Kollege` é masculino nominativo; `ein netter`. |
| 6.2 | Ich suche ein helles Zimmer. | `Zimmer` é neutro acusativo; `ein helles`. |
| 6.3 | Wir besuchen eine alte Stadt. | `Stadt` é feminino acusativo; `eine alte`. |
| 6.4 | Mit guten Plänen wird alles leichter. | `mit` exige dativo plural; adjetivo com `-en`. |
| 6.5 | Sie trifft einen geduldigen Lehrer. | `Lehrer` é masculino acusativo; `einen geduldigen`. |
| 6.6 | Gute Fragen helfen im Kurs. | Plural sem artigo nominativo usa `-e`. |
| 7.1 | schlechten Wetters | `wegen` costuma reger genitivo; `Wetter` é neutro genitivo: `schlechten Wetters`. |
| 7.2 | einem freundlichen Team | `mit` exige dativo; depois de `einem`, o adjetivo recebe `-en`. |
| 7.3 | einen wichtigen Termin | `für` exige acusativo; `Termin` é masculino: `einen wichtigen`. |
| 7.4 | guter Argumente | `trotz` pode reger genitivo formal; plural sem artigo genitivo usa `-er`. |
| 7.5 | einem einfachen Grund | `aus` exige dativo; `Grund` é masculino: `einem einfachen`. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Onde Mara começa o estágio? | Ela começa em uma pequena empresa: `in einer kleinen Firma`. |
| 2. Quem mostra o escritório para Mara? | Um colega simpático mostra o escritório: `ein freundlicher Kollege`. |
| 3. Qual tarefa importante Mara recebe? | Ela deve reunir boas ideias para um novo projeto. |
| 4. Que pessoas Mara encontra no quarto claro? | Ela encontra uma designer experiente e um programador jovem. |
| 5. Por que alguns funcionários ficam em casa? | Eles ficam em casa por causa do mau tempo. |
| 6. O que ajuda a colaboração apesar do tempo ruim? | Boas videoconferências ajudam a colaboração. |
| 7. Escreva em alemão: “uma colega simpática”. | `eine nette Kollegin`; feminino com `eine` + `-e`. |
| 8. Escreva em alemão: “com boas videoconferências”. | `mit guten Videokonferenzen`; `mit` exige dativo plural. |
| Prática 1 | neues; `Praktikum` é neutro com `ein`. |
| Prática 2 | freundlicher; `Kollege` é masculino nominativo com `ein`. |
| Prática 3 | wichtigen; `Auftrag` é masculino acusativo com `einen`. |
| Prática 4 | gute; plural sem artigo no acusativo. |
| Prática 5 | geduldigen; dativo depois de `einem`. |
| Prática 6 | schlechten; genitivo em `wegen schlechten Wetters`. |
