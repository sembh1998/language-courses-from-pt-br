# Respostas: Partizip II como adjetivo

Nível: B2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | gekochte | Gekocht recebe -e depois de das no nominativo neutro. |
| 1.2 | unterschriebenen | Plural acusativo com die exige a terminação -en. |
| 1.3 | reparierten | Dativo masculino com einem exige -en; reparieren forma repariert sem ge-. |
| 1.4 | vorbereiteter | Masculino nominativo com ein recebe a terminação forte -er. |
| 2.1 | geschlossenen | Dativo feminino com der exige -en; a porta já estava fechada. |
| 2.2 | angekommenen | Plural com artigo definido recebe -en; angekommen indica chegada concluída. |
| 2.3 | gesunkenen | Gesunken descreve uma redução já ocorrida; plural acusativo com die recebe -en. |
| 2.4 | gelieferte | Das Paket é neutro acusativo com artigo definido, por isso gelieferte. |
| 3.1 | Die versehentlich gelöschte Datei konnte wiederhergestellt werden. | A relativa passiva vira gelöschte antes de Datei. |
| 3.2 | Der gestern veröffentlichte Bericht enthält neue Zahlen. | O advérbio gestern permanece antes do particípio no atributo estendido. |
| 3.3 | Die bereits angekommenen Mitarbeiter warten im Saal. | Ankommen forma o Perfekt com sein e permite um atributo ativo concluído. |
| 3.4 | Wir besprechen die vom Ausschuss beschlossenen Maßnahmen. | O agente com von fica antes de beschlossenen no atributo estendido. |
| 4.1 | das kochende Wasser | Partizip I descreve uma ação ativa em andamento. |
| 4.2 | das gekochte Wasser | Partizip II descreve o resultado da ação concluída. |
| 4.3 | die sinkenden Preise | Sinkend apresenta o processo atual de queda. |
| 4.4 | die gesunkenen Preise | Gesunken apresenta uma mudança já concluída. |
| 5.1 | Die geschriebene Nachricht enthält alle Details. | Geschrieben recebe -e com die no nominativo feminino. |
| 5.2 | Wir arbeiten mit den aktualisierten Daten. | Mit exige dativo, e o plural com den recebe -en. |
| 5.3 | Das verschwundene Dokument war im Archiv. | Verschwunden tem leitura ativa concluída e recebe -e com das. |
| 5.4 | Die gestern beschlossene Maßnahme tritt heute in Kraft. | Gestern precede o Partizip II em um atributo estendido. |

## Respostas das perguntas de compreensão

| Pergunta | Resposta | Observação |
|---|---|---|
| Was bemerkte Lara in der veröffentlichten Tabelle? | Sie bemerkte mehrere falsch berechnete Werte. | A segunda frase apresenta os valores calculados incorretamente. |
| Welches Problem hatte die vom externen Team gelieferte Datei? | Die Datei war beschädigt. | O material externo continha um arquivo danificado. |
| Was rekonstruierten Lara und ihre Kollegin? | Sie rekonstruierten die versehentlich gelöschten Formeln. | As fórmulas haviam sido apagadas por engano. |
| Welche zusätzliche Maßnahme beschloss das Team? | Das Team beschloss eine zusätzliche Qualitätskontrolle. | A decisão exigiu um controle de qualidade adicional. |
| Warum blieben die Kosten überschaubar? | Der Text nennt keinen genauen Grund; er sagt nur, dass die bereits entstandenen Kosten überschaubar blieben. | A resposta distingue informação explícita de inferência. |
| Nenne zwei Partizip-II-Attribute aus dem Text. | Zum Beispiel die aktualisierten Projektdaten und die gefundene Lösung. | Há várias respostas possíveis retiradas da história. |
