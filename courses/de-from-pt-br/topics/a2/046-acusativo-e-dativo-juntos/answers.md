# Respostas: Acusativo e dativo juntos

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | dir | A pessoa que recebe o caderno fica no dativo; **dir** significa “a você”. |
| 1.2 | den | **Hund** é masculino e objeto direto; no acusativo fica **den Hund**. |
| 1.3 | ihm | A pessoa que recebe a carta fica no dativo; **ihm** significa “a ele”. |
| 1.4 | ein | **Geschenk** é neutro; no acusativo com artigo indefinido fica **ein Geschenk**. |
| 2.1 | Ich gebe der Frau die Tasche. | Com dois substantivos, a ordem neutra é dativo (**der Frau**) + acusativo (**die Tasche**). |
| 2.2 | Sie zeigt ihm das Zimmer. | **ihm** é dativo para “a ele”; a coisa mostrada é **das Zimmer**. |
| 2.3 | Ich gebe es dir. | Com dois pronomes, a ordem comum é acusativo (**es**) antes de dativo (**dir**). |
| 3.1 | Ich gebe ihm das Buch. | Substituir **dem Lehrer** por **ihm** mantém o dativo antes do substantivo acusativo. |
| 3.2 | Maria zeigt sie ihrer Freundin. | Quando o acusativo vira pronome (**sie**), ele vem antes do substantivo dativo. |
| 3.3 | Wir schicken sie dem Chef. | **sie** substitui **die E-Mail** no acusativo e vem antes de **dem Chef**. |
| 3.4 | Tobias bringt ihr den Kuchen. | **ihr** substitui **seiner Oma** no dativo e vem antes do objeto acusativo substantivo. |
| 4.1 | Der Vater gibt dem Kind den Ball. | O verbo fica na segunda posição; com dois substantivos, use dativo + acusativo. |
| 4.2 | Anna erklärt mir die Aufgabe. | **mir** é dativo e vem antes do objeto acusativo **die Aufgabe**. |
| 4.3 | Der Lehrer zeigt es uns. | Com dois pronomes, **es** no acusativo vem antes de **uns** no dativo. |
| 4.4 | Lukas bringt ihr einen Tee. | O pronome dativo **ihr** vem antes do substantivo acusativo **einen Tee**. |
| 5.1 | dativo = meiner Tante, acusativo = das Paket | A tia recebe; o pacote é enviado. |
| 5.2 | dativo = dem Patienten, acusativo = das Problem | O paciente recebe a explicação; o problema é o conteúdo explicado. |
| 5.3 | dativo = mir, acusativo = den Termin | **mir** é a pessoa que recebe a informação; **den Termin** é a informação. |
| 5.4 | dativo = ihrem Bruder, acusativo = die Schuhe | O irmão recebe; os sapatos são levados. |
| 6.1 | Ich schicke dir das Foto. | **dir** é dativo para “a você”; **das Foto** é acusativo. |
| 6.2 | Wir kaufen ihr ein Geschenk. | A pessoa beneficiada fica no dativo; **ein Geschenk** é neutro no acusativo. |
| 6.3 | Zeigst du es mir? | Com dois pronomes, **es** vem antes de **mir**. |
| 6.4 | Er erklärt den Schülern die Regel. | **den Schülern** é dativo plural; **die Regel** é acusativo. |
| 7.1 | Ich gebe dir das Handy. | Para “te dar”, use dativo **dir**, não acusativo **dich**. |
| 7.2 | Sie zeigt es mir. | Com dois pronomes, acusativo **es** vem antes de dativo **mir**. |
| 7.3 | Wir schicken der Frau eine Karte. | **Frau** é feminino; no dativo singular fica **der Frau**. |
| 7.4 | Kannst du ihm den Schlüssel geben? | A frase já está correta: **ihm** é dativo e **den Schlüssel** é acusativo. |

| 4 | mir | A pessoa que recebe o calendário fica no dativo. | 1 |
| 5 | eine | **Geschichte** é feminino; como objeto acusativo com artigo indefinido fica **eine Geschichte**. | 1 |
| 6 | uns | **uns** pode ser dativo e significa “para nós” nesta frase. | 1 |
| 7 | Er gibt ihr den Bericht. | **ihr** substitui **der Kollegin** no dativo. | 2 |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1 | Leo dá um caderno a Mara: **Leo gibt ihr ein Heft.** |
| 2 | Ela explica a primeira tarefa à turma: **der Klasse**. |
| 3 | Ele compra um café para Mara. |
| 4 | Mara dá a Leo um biscoito do Brasil. |
| 5 | Em **Mara schickt es ihm**, **es** é acusativo e **ihm** é dativo. |
| 6 | A professora explica o caminho para Mara. |
| 7 | **Die Lehrerin erklärt ihr den Weg.** |
| 8 | **Leo zeigt ihn ihr auf dem Handy.** |
| Prática 1 | **Leo gibt ihr ein Heft.** **ihr** é dativo. |
| Prática 2 | **Die Lehrerin erklärt der Klasse die erste Aufgabe.** A coisa explicada fica no acusativo. |
| Prática 3 | **Mara schickt es Leo.** Com pronome acusativo e substantivo/nome dativo, o pronome vem primeiro. |
| Prática 4 | **Mara gibt ihm einen Keks.** **ihm** substitui Leo no dativo. |
| Prática 5 | O dativo é **der Lehrerin**, porque ela recebe os livros de volta. |
| Prática 6 | **Sie zeigt es ihm.** Com dois pronomes, acusativo antes do dativo. |
