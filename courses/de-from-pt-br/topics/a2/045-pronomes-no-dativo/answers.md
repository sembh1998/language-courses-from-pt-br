# Respostas: Pronomes no dativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | mir | **helfen** pede dativo; para **ich**, o dativo é **mir**. |
| 1.2 | dir | **schreiben** indica escrever para alguém; para **du**, use **dir**. |
| 1.3 | ihm | Com **gehören**, a pessoa que possui fica no dativo; para **er**, use **ihm**. |
| 1.4 | ihr | **antworten** pede dativo; para **sie** singular, use **ihr**. |
| 1.5 | uns | A pessoa a quem o café agrada fica no dativo; para **wir**, use **uns**. |
| 1.6 | Ihnen | No tratamento formal com **Sie**, o dativo é **Ihnen** com maiúscula. |
| 2.1 | mir | A pessoa que recebe a explicação fica no dativo; **ich** vira **mir**. |
| 2.2 | dir | Quem recebe a chave está no dativo; **du** vira **dir**. |
| 2.3 | ihr | A música agrada a ela; **sie** singular vira **ihr**. |
| 2.4 | uns | **helfen** pede dativo; **wir** vira **uns**. |
| 2.5 | euch | Para **ihr** plural, o dativo é **euch**. |
| 2.6 | ihnen | Para **sie** plural, o dativo é **ihnen**. |
| 3.1 | Ich helfe ihr. | **Anna** é feminino singular; no dativo, vira **ihr**. |
| 3.2 | Wir danken ihm. | **dem Lehrer** é masculino singular no dativo; o pronome é **ihm**. |
| 3.3 | Das Auto gehört ihnen. | **meinen Eltern** é plural; no dativo, use **ihnen**. |
| 3.4 | Er antwortet ihr. | **Frau Keller** é feminino singular; com **antworten**, use **ihr**. |
| 3.5 | Ich zeige ihnen den Park. | **Paul und Max** é plural; quem recebe a mostra fica no dativo: **ihnen**. |
| 4.1 | Ich danke dir. | **danken** pede dativo; “a você” informal é **dir**. |
| 4.2 | Der Film gefällt mir. | Com **gefallen**, a coisa é sujeito e a pessoa fica no dativo. |
| 4.3 | Wir helfen euch. | **helfen** pede dativo; “vocês” informal é **euch**. |
| 4.4 | Das Haus gehört ihnen. | **gehören** pede dativo; “a eles” é **ihnen**. |
| 4.5 | Kann ich Ihnen helfen? | Forma formal no dativo é **Ihnen** com maiúscula. |
| 5.1 | Ich helfe dir. | O correto é dativo depois de **helfen**: **dir**, não **dich**. |
| 5.2 | Das gefällt mir. | A pessoa a quem algo agrada fica no dativo: **mir**. |
| 5.3 | Wir danken ihr. | **danken** pede dativo; para “ela”, use **ihr**. |
| 5.4 | Der Laptop gehört ihm. | **gehören** pede dativo; para “ele”, use **ihm**. |
| 5.5 | Ich antworte Ihnen morgen. | Com tratamento formal, o dativo correto é **Ihnen**. |
| 6.1 | Ich helfe ihm. | **dein Bruder** é masculino singular; o dativo é **ihm**. |
| 6.2 | Ich schreibe ihr. | **deine Freundin** é feminino singular; o dativo é **ihr**. |
| 6.3 | Wir danken ihr. | **unsere Lehrerin** é feminino singular; com **danken**, use **ihr**. |
| 6.4 | Lisa zeigt uns die Wohnung. | **wir** no dativo vira **uns**, pois somos quem recebe a mostra. |
