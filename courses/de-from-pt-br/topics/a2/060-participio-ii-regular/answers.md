# Respostas: Particípio II regular

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | gelernt | Lernen é regular: ge- + lern + -t. |
| 1.2 | gemacht | Machen perde -en e recebe ge- e -t: gemacht. |
| 1.3 | gespielt | Spielen forma o Particípio II regular gespielt. |
| 1.4 | gekauft | Kaufen vira gekauft no padrão regular. |
| 1.5 | gewohnt | Wohnen vira gewohnt; o Particípio II fica no fim da frase. |
| 2.1 | gemacht | A forma correta de machen no Particípio II é gemacht. |
| 2.2 | gespielt | Spielen segue ge- + spiel + -t. |
| 2.3 | gekauft | Kaufen segue ge- + kauf + -t. |
| 2.4 | gehört | Hören vira gehört, com ge- e -t. |
| 2.5 | gefragt | Fragen vira gefragt; gefragen não é a forma padrão. |
| 2.6 | gewohnt | Wohnen vira gewohnt, não gewohnen. |
| 3.1 | gekocht | Kochen perde -en e recebe ge- e -t. |
| 3.2 | gesucht | Suchen vira gesucht no padrão regular. |
| 3.3 | gesagt | Sagen vira gesagt, sem manter -en. |
| 3.4 | getanzt | Tanzen vira getanzt. |
| 3.5 | gereist | Reisen vira gereist. |
| 3.6 | geliebt | Lieben vira geliebt. |
| 4.1 | habe gearbeitet | Com ich usa-se habe; arbeiten vira gearbeitet. |
| 4.2 | haben gewartet | Com wir usa-se haben; warten vira gewartet. |
| 4.3 | hat gehört | Com Lisa usa-se hat; hören vira gehört. |
| 4.4 | habt gespielt | Com ihr usa-se habt; spielen vira gespielt. |
| 4.5 | haben gekauft | Tom und Mia é plural; kaufen vira gekauft. |
| 5.1 | gelernt | Gelernen mistura infinitivo com Particípio II; correto é gelernt. |
| 5.2 | gemacht | Depois de hat, o verbo principal deve aparecer como Particípio II. |
| 5.3 | gespielt | Spielen vira gespielt, não gespielen. |
| 5.4 | gewartet | Warten precisa de -et no final por causa do radical terminado em -t. |
| 5.5 | gefragt | Fragen vira gefragt; gefragen não é a forma regular correta. |
| 6.1 | Ich habe Brot gekauft. | Comprar é kaufen; no Perfekt regular fica habe gekauft. |
| 6.2 | Wir haben Deutsch gelernt. | Com wir usa-se haben; lernen vira gelernt. |
| 6.3 | Sie hat Kaffee gemacht. | Com sie singular usa-se hat; machen vira gemacht. |
| 6.4 | Ihr habt Tennis gespielt. | Com ihr usa-se habt; spielen vira gespielt. |
| 6.5 | Er hat gestern gearbeitet. | Com er usa-se hat; arbeiten vira gearbeitet. |
