# Respostas: Contraste Wo vs Wohin

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Wo? | **im Büro** mostra localização; com Wo? usa-se dativo. |
| 1.2 | Wohin? | **geht in die Küche** mostra direção para a cozinha; usa-se acusativo. |
| 1.3 | Wo? | **hängt an der Wand** indica posição na parede; é dativo. |
| 1.4 | Wohin? | **stellen ... vor das Fenster** mostra colocar em direção a um lugar; é acusativo. |
| 2.1 | in der | **wohne** indica localização na cidade; **die Stadt** no dativo vira **der Stadt**. |
| 2.2 | in die | **geht** indica direção para a escola; **die Schule** no acusativo fica **die Schule**. |
| 2.3 | auf dem | **steht** indica posição; **der Tisch** no dativo vira **dem Tisch**. |
| 2.4 | auf den | **legt** indica colocar em direção à mesa; **der Tisch** no acusativo vira **den Tisch**. |
| 2.5 | unter dem | **schläft** indica posição; **das Sofa** no dativo vira **dem Sofa**. |
| 2.6 | unter das | **läuft** indica direção para baixo do sofá; **das Sofa** no acusativo fica **das Sofa**. |
| 3.1 | Das Buch liegt auf dem Bett. | O livro já está na cama: posição com dativo e verbo **liegen**. |
| 3.2 | Ich stelle die Tasche neben die Tür. | Colocar a bolsa para o lado da porta pede acusativo: **neben die Tür**. |
| 3.3 | Die Kinder spielen im Garten. | As crianças estão/brincam dentro do jardim; é localização, então dativo: **im Garten**. |
| 3.4 | Wir gehen an die Haltestelle. | Ir até a parada indica direção: **an die Haltestelle**. |
| 4.1 | ins | Transformação para direção: **in das Wohnzimmer** contrai para **ins Wohnzimmer**. |
| 4.2 | auf den | **legen** indica colocar sobre a cadeira; **der Stuhl** no acusativo é **den Stuhl**. |
| 4.3 | an die | Pendurar em direção à porta pede acusativo: **an die Tür**. |
| 4.4 | hinter den | Rolar para trás do armário indica direção; **der Schrank** no acusativo é **den Schrank**. |
| 4.5 | neben das | Colocar ao lado do sofá indica direção; **das Sofa** no acusativo fica **das Sofa**. |
| 5.1 | dem | **steht** mostra posição sobre a escrivaninha; dativo masculino/neutro usa **dem**. |
| 5.2 | den | **stelle** mostra direção para cima da escrivaninha; acusativo masculino usa **den**. |
| 5.3 | der | **sitzt** mostra posição na cozinha; dativo feminino usa **der**. |
| 5.4 | die | **geht** mostra direção para a cozinha; acusativo feminino usa **die**. |
| 5.5 | dem | A bicicleta está parada na frente da casa; posição com dativo: **vor dem Haus**. |
| 5.6 | das | Colocar a bicicleta para a frente da casa indica direção; acusativo neutro: **vor das Haus**. |
| 6.1 | Die Katze ist unter dem Tisch. | A pergunta **Wo?** pede posição com dativo: **unter dem Tisch**. |
| 6.2 | Die Katze läuft unter den Tisch. | A pergunta **Wohin?** pede direção com acusativo: **unter den Tisch**. |
| 6.3 | Paul wartet vor dem Kino. | Esperar em frente ao cinema é posição; **das Kino** no dativo vira **dem Kino**. |
| 6.4 | Anna fährt vor das Kino. | Dirigir/ir para a frente do cinema é direção; acusativo neutro fica **das Kino**. |
| 7.1 | Ich bin im Zimmer. | Com **bin** aqui há posição no quarto; deve ser **in dem Zimmer = im Zimmer**. |
| 7.2 | Sie geht in die Küche. | **gehen** indica direção; para feminino singular no acusativo: **in die Küche**. |
| 7.3 | Das Buch liegt auf dem Tisch. | Para posição usa-se **liegen**, não **legen**; e dativo: **auf dem Tisch**. |
| 7.4 | Ich lege den Schlüssel auf den Tisch. | **lege** indica direção para a mesa; masculino acusativo: **den Tisch**. |
