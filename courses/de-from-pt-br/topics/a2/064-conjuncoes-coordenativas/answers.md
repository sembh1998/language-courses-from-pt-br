# Respostas: Conjunções Coordenativas

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | und | Soma duas ações: ter fome e cozinhar sopa. |
| 1.2 | oder | Apresenta duas alternativas: arroz ou batatas. |
| 1.3 | aber | Mostra contraste entre cansaço e fazer a lição. |
| 1.4 | denn | Dá o motivo para pegar o ônibus, com verbo em ordem normal. |
| 1.5 | sondern | Corrige a ideia negativa: não é maçã, mas sim pera. |
| 2.1 | Ich esse jetzt, denn ich habe Hunger. | Depois de denn, o verbo não vai para o fim. |
| 2.2 | Sie trinkt keinen Saft, sondern Wasser. | Sondern é usado depois de negação para corrigir. |
| 2.3 | Wir gehen ins Kino oder ins Café. | Oder indica alternativa entre cinema e café. |
| 2.4 | Er ist krank, aber er geht zur Arbeit. | Aber marca contraste entre estar doente e ir trabalhar. |
| 3.1 | Ich gehe einkaufen, denn ich brauche Milch. | Denn introduz o motivo mantendo sujeito + verbo. |
| 3.2 | Maria arbeitet viel, aber sie hat wenig Zeit. | Aber liga duas ideias em contraste. |
| 3.3 | Wir besuchen Oma und Opa. | Und pode ligar dois objetos sem repetir o verbo. |
| 3.4 | Ich fahre nicht nach Berlin, sondern nach Leipzig. | Sondern corrige o destino depois de nicht. |
| 4.1 | soma | Und soma elementos. |
| 4.2 | alternativa | Oder indica escolha. |
| 4.3 | contraste | Aber contrasta informações. |
| 4.4 | motivo | Denn explica a razão. |
| 4.5 | correção | Sondern corrige uma afirmação negativa. |
| 5.1 | weil | Com weil, o verbo bin vai para o fim. |
| 5.2 | denn | Com denn, a ordem fica ich bin. |
| 5.3 | denn | A frase mantém er muss, então pede denn. |
| 5.4 | weil | A forma muss no fim indica oração com weil. |
| 6.1 | Ich trinke keinen Kaffee, sondern Tee. | Após negação com keinen, a correção pede sondern. |
| 6.2 | Wir bleiben hier, denn wir sind müde. | Com denn, o verbo sind vem depois do sujeito wir. |
| 6.3 | Sie kauft Brot oder Käse. | Quando a alternativa é entre objetos, não é preciso repetir a frase. |
| 6.4 | Tom hat Zeit, aber er kommt nicht. | Não há correção após negação; é contraste com aber. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1 | Eles planejam um fim de semana. |
| 2 | Porque ele está no supermercado. |
| 3 | Ela quer chá, não café. |
| 4 | A conjunção é aber, que mostra contraste. |
| 5 | Jonas nimmt Wasser, denn er trinkt heute keinen Kaffee. |
| 6 | Lena bestellt keinen Apfelkuchen, sondern Käsekuchen. |

| Prática 2 | und |
| Prática 3 | oder |
| Prática 4 | Porque há negação com keinen Kaffee e a segunda parte corrige para Tee. |
