# Respostas: Haben ou sein no Perfekt

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | bin | **gehen** indica movimento até outro lugar; por isso usa **sein**: ich bin. |
| 1.2 | hast | **lernen** é ação comum/transitiva; por isso usa **haben**: du hast. |
| 1.3 | ist | **aufstehen** indica mudança de estado; por isso usa **sein**. |
| 1.4 | haben | **kaufen** tem objeto direto (**einen Kaffee**); usa **haben**. |
| 1.5 | seid | **fahren** com deslocamento usa **sein**; com **ihr**, a forma é **seid**. |
| 2.1 | Ich bin nach Berlin gefahren. | **fahren** para uma cidade indica deslocamento; o auxiliar correto é **sein**. |
| 2.2 | Sie hat eine Tasche gekauft. | **kaufen** é transitivo e usa **haben**; com **sie**, a forma é **hat**. |
| 2.3 | Wir sind im Hotel geblieben. | **bleiben** normalmente usa **sein** no Perfekt. |
| 2.4 | Ich habe den Lehrer gesehen. | **sehen** é transitivo; o auxiliar correto é **haben**. |
| 3.1 | Ich bin zum Supermarkt gegangen. | **gehen** expressa movimento até um lugar; use **bin** e particípio no fim. |
| 3.2 | Er hat eine Fahrkarte gekauft. | **kaufen** usa **haben**; **er hat** é a conjugação correta. |
| 3.3 | Wir sind spät gekommen. | **kommen** indica chegada/movimento; usa **sein**. |
| 3.4 | Lea hat eine E-Mail gelesen. | **lesen** tem objeto direto; usa **haben**. |
| 3.5 | Die Kinder sind schnell eingeschlafen. | **einschlafen** é mudança de estado; usa **sein**. |
| 4.1 | sein | **gehen** indica deslocamento. |
| 4.2 | haben | **lernen** é ação comum/transitiva. |
| 4.3 | sein | **kommen** indica chegada. |
| 4.4 | haben | **machen** é ação comum e geralmente transitiva. |
| 4.5 | sein | **aufstehen** indica mudança de estado. |
| 4.6 | haben | **sehen** é transitivo e usa **haben**. |
| 5.1 | Ich bin gestern nach Wien gefahren. | Movimento para Viena exige **sein**. |
| 5.2 | Sie hat ein Buch gelesen. | **lesen** com objeto direto usa **haben**. |
| 5.3 | Wir sind früh aufgestanden. | **aufstehen** é mudança de estado; usa **sein**. |
| 5.4 | Paul hat die Hausaufgabe gemacht. | **machen** com objeto direto usa **haben**. |
| 6.1 | Ich bin gestern nach Hause gegangen. | Resposta-modelo correta: **gehen** com deslocamento usa **sein**. |
| 6.2 | Ich habe am Wochenende Deutsch gelernt. | Resposta-modelo correta: **lernen** usa **haben**. |
| 6.3 | Ich bin heute um sieben Uhr aufgestanden. | Resposta-modelo correta: **aufstehen** usa **sein**. |
| 6.4 | Ich habe im Supermarkt Brot gekauft. | Resposta-modelo correta: **kaufen** usa **haben**. |

| 1 | Tom ist zur Arbeit gegangen. | **gehen** até um lugar usa **sein**; com Tom/er, a forma é **ist**. | 1 |
| 2 | hat | **arbeiten** é ação comum, sem deslocamento; usa **haben**. | 1 |
| 3 | sein | Mudança de estado, como adormecer ou ficar cansado, normalmente usa **sein**. | 1 |
| 4 | sind | **fahren** para uma cidade indica deslocamento; com **wir**, use **sind**. | 1 |
| 5 | Ich habe ein Sandwich gegessen. | **essen** com objeto direto usa **haben**. | 1 |

| 7 | ist | **einschlafen** é mudança de estado; com **meine Schwester**, use **ist**. | 1 |
| 8 | aufstehen | **aufstehen** usa **sein** porque indica mudança de estado. | 1 |
| 9 | Er hat einen Brief geschrieben. | **schreiben** com objeto direto usa **haben**, não **sein**. | 1 |
| 10 | habt | **putzen** é ação comum/transitiva; com **ihr**, use **habt**. | 1 |
| 11 | Die Studenten sind nach Österreich gereist. | **reisen** para outro lugar indica deslocamento; com plural, use **sind**. | 1 |
| 12 | Ich habe meine Nachbarin gesehen. | **sehen** é transitivo; use **haben**. | 1 |
| 13 | hat | **beginnen** normalmente usa **haben** no Perfekt. | 1 |

| 15 | Ich bin nach Hamburg gefahren. | **fahren** para uma cidade indica deslocamento; use **sein**. | 1 |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Wann ist Nina aufgestanden? | Nina ist am Samstag früh aufgestanden. |
| 2. Wohin ist Nina mit dem Zug gefahren? | Sie ist nach Freiburg gefahren. |
| 3. Was hat Nina am Bahnhof gekauft? | Sie hat eine Fahrkarte gekauft. |
| 4. Was haben Nina und Lara in der Stadt besucht? | Sie haben ein kleines Museum besucht. |
| 5. Warum heißt es: Nina ist eingeschlafen? | Porque **einschlafen** indica mudança de estado e usa **sein** no Perfekt. |
| 6. Complete: Zu Hause ___ Nina Deutsch gelernt. | Zu Hause **hat** Nina Deutsch gelernt. |
| 7. Reescreva no Perfekt: Nina geht zu Lara. | Nina ist zu Lara gegangen. |
| 8. Dê um exemplo da história com haben e um exemplo com sein. | Exemplo: **hat gekauft** e **ist gefahren**. Outros exemplos corretos da história também servem. |
| Prática 1 | Nina **ist** mit dem Fahrrad zum Bahnhof gefahren. |
| Prática 2 | Lara **hat** Kaffee gemacht. |
| Prática 3 | Sie **haben** eine Fahrkarte **gekauft**. |
| Prática 4 | Sie **sind** in die Stadt **gegangen**. |
| Prática 5 | Nina **hat** eine Nachricht gelesen. |
| Prática 6 | Nina **ist** müde geworden. |
