# Respostas: Verbos com Dativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | mir | **helfen** exige dativo; para **ich**, o dativo é **mir**. |
| 1.2 | der Frau | **gefallen** exige dativo; feminino singular usa **der**. |
| 1.3 | unseren Freunden | **danken** exige dativo; plural dativo usa **-n**: **Freunden**. |
| 1.4 | meinem Vater | **gehören** exige dativo; masculino singular usa **meinem**. |
| 1.5 | dir | **antworten** exige dativo; para **du**, o dativo é **dir**. |
| 1.6 | meinem Kind | **passen** exige dativo; neutro singular usa **meinem**. |
| 2.1 | helfe | Com **ich**, o verbo **helfen** fica **helfe** e pede dativo. |
| 2.2 | gefällt | **Der Film** é singular; por isso **gefällt**. A pessoa está no dativo. |
| 2.3 | gehört | **Dieses Auto** é singular; **gehört** expressa posse com dativo. |
| 2.4 | passt | **Die Jacke** é singular; **passen** fica **passt**. |
| 2.5 | fehlen | **Meine Eltern** é plural; por isso **fehlen**. |
| 2.6 | antwortest | Com **du**, **antworten** vira **antwortest** e pede dativo. |
| 2.7 | danken | Com **wir**, usa-se **danken**; a pessoa agradecida fica no dativo. |
| 3.1 | Ich helfe ihr. | **Meiner Schwester** vira o pronome dativo feminino **ihr**. |
| 3.2 | Das Zimmer gefällt ihnen. | **Meinen Eltern** vira **ihnen**, pronome dativo plural. |
| 3.3 | Die Tasche gehört ihm. | **Dem Studenten** vira **ihm**, pronome dativo masculino. |
| 3.4 | Der Pullover passt ihr. | **Meiner Freundin** vira **ihr**, pronome dativo feminino. |
| 3.5 | Der Lehrer antwortet ihnen. | **Den Kindern** vira **ihnen**, pronome dativo plural. |
| 3.6 | Das Geld fehlt ihm. | **Meinem Vater** vira **ihm**, pronome dativo masculino. |
| 4.1 | Ich helfe dir am Samstag. | Depois de **helfen**, use dativo: **du** vira **dir**. |
| 4.2 | Sie dankt dem Arzt. | **danken** pede dativo; masculino singular é **dem Arzt**. |
| 4.3 | Das Kleid gefällt meiner Mutter. | **gefallen** pede dativo; **meine Mutter** vira **meiner Mutter**. |
| 4.4 | Der Schlüssel gehört mir. | **gehören** pede dativo; **ich** vira **mir**. |
| 4.5 | Die Hose passt meinem Bruder nicht. | **passen** pede dativo; **mein Bruder** vira **meinem Bruder**. |
| 4.6 | Bitte antworte deiner Chefin sofort. | **antworten** pede dativo; feminino singular usa **deiner**. |
| 5.1 | Ich helfe meinem Freund. | **helfen** exige dativo; masculino singular com possessivo: **meinem**. |
| 5.2 | Wir danken der Lehrerin. | **danken** exige dativo; feminino singular usa **der**. |
| 5.3 | Dieses Kleid gefällt ihr. | Com **gefallen**, a coisa é sujeito; a pessoa vai para o dativo. |
| 5.4 | Das Handy gehört meinem Bruder. | **gehören** pede dativo; **meinem Bruder** indica o dono. |
| 5.5 | Passt dir das Hemd? | **passen** pede dativo; para **du**, use **dir**. |
| 5.6 | Meine Oma fehlt mir. | Em **fehlen**, a pessoa que sente falta fica no dativo: **mir**. |
| 6.1 | Ich helfe meinem Nachbarn. | **Nachbar** é masculino fraco; no dativo: **meinem Nachbarn**. |
| 6.2 | Anna dankt dem Verkäufer. | **danken** pede dativo; masculino singular usa **dem**. |
| 6.3 | Der Laptop gehört meiner Tante. | **gehören** pede dativo; feminino singular usa **meiner**. |
| 6.4 | Die Brille passt dem Kind. | **passen** pede dativo; neutro singular usa **dem**. |
| 6.5 | Der Pass fehlt dem Touristen. | **Tourist** é masculino fraco; dativo: **dem Touristen**. |
| 6.6 | Ich antworte meiner Kollegin. | **antworten** pede dativo; feminino singular usa **meiner**. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| 1. Wem hilft der alte Mann? | Er hilft Lena/ihr. |
| 2. Wem dankt Lena an der Haltestelle? | Sie dankt dem Mann. |
| 3. Wem antwortet Lena im Kurs? | Sie antwortet der Lehrerin. |
| 4. Was passt Amir nicht? | Die erste Jacke passt ihm nicht. |
| 5. Wem gefällt die blaue Jacke? | Die blaue Jacke gefällt Amir/ihm. |
| 6. Wem gehört die Nachricht auf Lenas Handy? | Die Nachricht gehört ihrer Schwester. |
| 7. Wem gefällt das Essen am Abend? | Das Essen gefällt Sofia/ihrer Mitbewohnerin. |
| 8. Quem ajuda Sofia com a lição de casa? | Lena hilft Sofia mit den Hausaufgaben. |
| Prática 1 | Lena. Com nome próprio, a forma pode ficar igual; função é dativa após **helfen**. |
| Prática 2 | dem Mann. **danken** pede dativo masculino singular. |
| Prática 3 | der Lehrerin. **antworten** pede dativo feminino singular. |
| Prática 4 | ihm. **passen** pede dativo; Amir vira **ihm**. |
| Prática 5 | gefällt. A jaqueta agrada Amir; **die Jacke** é singular. |
| Prática 6 | ihrer Schwester. **gehören** pede dativo feminino singular com possessivo. |
| Prática 7 | Sofia. Nome próprio sem artigo pode ficar igual; função é dativa após **gefallen**. |
| Prática 8 | ihr. **danken** pede dativo; Lena vira **ihr**. |
