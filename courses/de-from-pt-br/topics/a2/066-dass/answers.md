# Respostas: Dass

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | dass | A conjunção **dass** introduz a ideia relatada depois de **Ich glaube**. |
| 1.2 | dass | Depois de **er sagt**, usamos **dass** para relatar o que ele diz. |
| 1.3 | dass | **Wir denken, dass...** relata pensamento ou opinião. |
| 1.4 | dass | **Ich hoffe, dass...** introduz aquilo que a pessoa espera. |
| 1.5 | dass | **Sie weiß, dass...** introduz a informação conhecida. |
| 2.1 | Ich glaube, dass er müde ist. | Em oração com **dass**, o verbo conjugado **ist** vai para o fim. |
| 2.2 | Sie sagt, dass sie Hunger hat. | O verbo **hat** deve ficar no final da oração subordinada. |
| 2.3 | Ich denke, dass wir helfen können. | Com modal, o modal conjugado **können** vai para o fim. |
| 2.4 | Ich finde, dass der Film gut ist. | A ordem correta é sujeito + complementos + verbo no fim. |
| 2.5 | Es ist wichtig, dass du jeden Tag lernst. | O verbo **lernst** fecha a oração com **dass**. |
| 2.6 | Ich weiß, dass du recht hast. | A expressão é **recht haben**; o verbo conjugado **hast** fica no fim. |
| 3.1 | Ich glaube, dass Lisa krank ist. | A frase **Lisa ist krank** vira subordinada; **ist** vai ao final. |
| 3.2 | Paul sagt, dass er morgen arbeitet. | O sujeito muda para **er** e o verbo **arbeitet** fica no fim. |
| 3.3 | Wir denken, dass das Restaurant teuer ist. | Em alemão, o adjetivo vem antes do verbo final **ist**. |
| 3.4 | Anna weiß, dass der Bus um acht kommt. | O verbo **kommt** aparece no final da informação relatada. |
| 3.5 | Ich hoffe, dass du am Samstag kommen kannst. | Com modal, o infinitivo vem antes e **kannst** fica no fim. |
| 4.1 | Ich glaube, dass Tom zu Hause ist. | A correção move **ist** para o fim da oração com **dass**. |
| 4.2 | Sie sagt, dass sie lernen muss. | O modal conjugado **muss** deve ficar no final. |
| 4.3 | Wir wissen, dass der Zug spät ist. | **ist** fecha a oração subordinada. |
| 4.4 | Er denkt, dass wir frei haben. | Em **frei haben**, o verbo conjugado **haben** fica no fim. |
| 4.5 | Ich finde, dass Deutsch nicht schwer ist. | A negação e o adjetivo vêm antes do verbo final **ist**. |
| 5.1 | Ich glaube, dass sie in Berlin wohnt. | **wohnt** é o verbo conjugado e fica no final. |
| 5.2 | Er sagt, dass er krank ist. | Depois de **dass**, usamos sujeito + complemento + verbo no fim. |
| 5.3 | Wir finden, dass der Kaffee gut ist. | **finden, dass...** expressa opinião; **ist** vai ao fim. |
| 5.4 | Ich weiß, dass du Zeit hast. | A expressão **Zeit haben** leva **hast** para o final. |
| 5.5 | Es ist wichtig, dass wir pünktlich kommen. | A oração com **dass** explica o que é importante. |
| 6.1 | dass er heute kommt. | Essa continuação completa naturalmente **Ich glaube** com verbo no final. |
| 6.2 | dass sie müde ist. | **ist** fica no fim e relata o que ela diz. |
| 6.3 | dass das Wetter gut bleibt. | **bleibt** fecha a oração com **dass**. |
| 6.4 | dass der Termin um neun beginnt. | A informação conhecida termina com o verbo **beginnt**. |
| 6.5 | dass du hier bist. | **Es ist gut, dass...** introduz o motivo; **bist** fica no fim. |
