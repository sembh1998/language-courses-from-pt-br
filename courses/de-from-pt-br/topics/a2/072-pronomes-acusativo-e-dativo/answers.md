# Respostas: Pronomes acusativo + dativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | ihn | `der Ball` é masculino; no acusativo vira `ihn`. |
| 1.2 | sie | `die Tasche` é feminino; no acusativo vira `sie`. |
| 1.3 | es | `das Buch` é neutro; no acusativo vira `es`. |
| 1.4 | ihn | `der Schlüssel` é masculino; no acusativo vira `ihn`. |
| 2.1 | mir | Para `ich` no dativo usamos `mir`. |
| 2.2 | dir | Para `du` no dativo usamos `dir`. |
| 2.3 | ihm | Para `er` no dativo usamos `ihm`. |
| 2.4 | ihr | Para `sie` singular no dativo usamos `ihr`. |
| 2.5 | ihnen | Para `sie` plural no dativo usamos `ihnen`. |
| 3.1 | Ich gebe es dir. | Com dois pronomes, a ordem natural é acusativo + dativo. |
| 3.2 | Sie zeigt ihn mir. | `ihn` é acusativo e vem antes de `mir`. |
| 3.3 | Wir schicken es ihnen. | `es` vem antes de `ihnen` porque há dois pronomes. |
| 4.1 | Ich gebe es dir. | `das Heft` vira `es`; `dir` já é dativo. |
| 4.2 | Sie zeigt ihn mir. | `den Brief` vira `ihn`; depois vem `mir`. |
| 4.3 | Wir kaufen sie ihr. | `die Blumen` vira `sie`; `Anna` vira `ihr`. |
| 4.4 | Er schickt es ihnen. | `das Bild` vira `es`; `den Kindern` vira `ihnen`. |
| 5.1 | Ich gebe ihn dir. | `ele` como objeto masculino é `ihn`; destinatário `você` é `dir`. |
| 5.2 | Sie kauft es ihm. | `isso` é `es`; `para ele` é `ihm`. |
| 5.3 | Wir zeigen sie euch. | `ela` é `sie`; `a vocês` é `euch`. |
| 5.4 | Schickst du es mir? | Em pergunta, os pronomes continuam na ordem `es mir`. |
| 6.1 | Ich gebe es dir. | A forma `dir es` não é a ordem natural com dois pronomes. |
| 6.2 | Anna zeigt ihn mir. | `ihn` acusativo deve vir antes de `mir` dativo. |
| 6.3 | Wir schicken sie euch. | `sie` acusativo vem antes de `euch` dativo. |
| 6.4 | Paul bringt ihn ihr. | `ihn` acusativo vem antes de `ihr` dativo. |
| 7.1 | Ich gebe es dir. | A resposta pede `es` como objeto e `dir` como destinatário. |
| 7.2 | Sie zeigt ihn dir. | A resposta direta para você usa `dir`; `den Plan` vira `ihn`. |
| 7.3 | Ich kaufe sie ihr. | `die Jacke` vira `sie`; Anna como destinatária vira `ihr`. |
