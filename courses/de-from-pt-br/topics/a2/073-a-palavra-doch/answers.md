# Respostas: A palavra doch

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | Doch, ich komme. | A pergunta é negativa e a pessoa quer dizer que vai; por isso, usa **Doch**. |
| 1.2 | Doch, ich habe Hunger. | **Doch** corrige a ideia de que a pessoa não tem fome. |
| 1.3 | Doch, er ist zu Hause. | A resposta positiva a **nicht** pede **Doch**. |
| 1.4 | Doch, ich mag Tee. | A pessoa gosta de chá, então corrige **keinen Tee** com **Doch**. |
| 2.1 | Doch | A frase completa diz que a pessoa bebe café, corrigindo a pergunta negativa. |
| 2.2 | Nein | Aqui a pessoa confirma que não tem tempo, então **Nein** é adequado. |
| 2.3 | Doch | Maria está presente; **Doch** corrige **nicht da**. |
| 2.4 | Nein | A pessoa confirma que não mora em Köln. |
| 2.5 | Doch | A pessoa trabalha amanhã; **Doch** corrige a negação. |
| 3.1 | Doch, ich lerne Deutsch. | Resposta positiva completa a uma pergunta negativa. |
| 3.2 | Doch, ich habe ein Fahrrad. | **Doch** corrige **kein Fahrrad** e usa **ein Fahrrad** na resposta positiva. |
| 3.3 | Doch, das ist mein Handy. | A resposta afirma que o celular é da pessoa. |
| 3.4 | Doch, wir gehen ins Kino. | A resposta positiva para **ihr** usa **wir**. |
| 4.1 | Doch, ich bin müde. | Corrige a ideia de que a pessoa não está cansada. |
| 4.2 | Doch, sie hat ein Auto. | Corrige **kein Auto** com uma frase positiva. |
| 4.3 | Doch, wir kommen am Samstag. | A resposta afirma que o grupo vem no sábado. |
| 4.4 | Doch, er ist offen. | A loja está aberta; **Doch** corrige a negação. |
| 5.1 | Doch, ich habe einen Stift. | A pessoa tem a caneta, então responde com **Doch**. |
| 5.2 | Doch, ich komme mit. | A fala anterior é negativa, mas a pessoa vai junto. |
| 5.3 | Doch, das ist mein Platz. | A resposta afirma que o lugar é seu. |
| 5.4 | Doch, ich lese das Buch. | A resposta positiva corrige a pergunta negativa. |
