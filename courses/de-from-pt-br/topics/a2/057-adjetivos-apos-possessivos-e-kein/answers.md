# Respostas: Adjetivos após possessivos e kein

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | er | `Freund` é masculino nominativo; depois de `mein`, o adjetivo recebe `-er`. |
| 1.2 | e | `Kollegin` é feminino nominativo; `meine` + adjetivo com `-e`. |
| 1.3 | es | `Auto` é neutro nominativo; depois de `mein`, o adjetivo recebe `-es`. |
| 1.4 | en | Plural com possessivo usa adjetivo com `-en`: `meine alten Schuhe`. |
| 1.5 | es | `Problem` é neutro; com `kein`, o adjetivo fica `großes`. |
| 2.1 | meinen | `Onkel` é masculino acusativo depois de `besuchen`; use `meinen`. |
| 2.2 | keinen | `Tisch` é masculino acusativo depois de `kaufen`; use `keinen`. |
| 2.3 | ihre | `Tasche` é feminino acusativo; a forma é `ihre`. |
| 2.4 | sein | `Buch` é neutro acusativo; a forma é `sein`. |
| 2.5 | keine | `Kinder` está no plural acusativo; a forma negativa é `keine`. |
| 3.1 | Mein guter Freund wohnt hier. | Masculino nominativo pede `mein guter Freund`. |
| 3.2 | Ich sehe meine neue Tasche. | Feminino acusativo mantém `meine` e adjetivo `-e`. |
| 3.3 | Keine kleinen Kinder spielen hier. | Plural com `keine` pede adjetivo `-en`. |
| 3.4 | Das ist kein kleines Problem. | `Problem` é neutro; com `kein`, use `kleines`. |
| 3.5 | Ich treffe meinen guten Lehrer. | `Lehrer` é masculino acusativo; use `meinen` e adjetivo `-en`. |
| 3.6 | Wir haben keine alten Stühle. | Plural acusativo com `keine` pede `alten`. |
| 4.1 | Das ist kein kleiner Hund. | Troca-se `ein` por `kein`; masculino nominativo mantém adjetivo `-er`. |
| 4.2 | Sie hat keine neue Wohnung. | `Wohnung` é feminino acusativo; use `keine neue`. |
| 4.3 | Ich kaufe keinen teuren Mantel. | `Mantel` é masculino acusativo; use `keinen` e `teuren`. |
| 4.4 | Wir brauchen kein großes Zimmer. | `Zimmer` é neutro acusativo; use `kein großes`. |
| 4.5 | Dort stehen keine alten Fahrräder. | Plural com `keine` pede adjetivo `-en`. |
| 5.1 | Meine gute Freundin kommt morgen. | `Freundin` é feminino; o possessivo correto é `meine`. |
| 5.2 | Ich sehe meinen netten Nachbarn. | Masculino acusativo: `meinen netten`; `Nachbar` costuma tomar `-n`. |
| 5.3 | Das ist kein kleines Problem. | `Problem` é neutro; use `kein`, não `keine`, e adjetivo `-es`. |
| 5.4 | Unsere neuen Bücher sind interessant. | Plural com possessivo usa adjetivo `-en`. |
| 5.5 | Er trägt seine alte Jacke. | `Jacke` é feminino acusativo; use `seine alte`. |
| 6.1 | Mein guter Freund wohnt in München. | `Freund` é masculino nominativo; `mein guter` é o padrão correto. |
| 6.2 | Ich kaufe keinen teuren Mantel. | `Mantel` é masculino acusativo; `keinen teuren` marca caso e adjetivo. |
| 6.3 | Ihre kleine Schwester ist zu Hause. | `Schwester` é feminino nominativo; use `ihre kleine`. |
| 6.4 | Wir haben ein helles Zimmer. | `Zimmer` é neutro acusativo; com `ein`, adjetivo `helles`. |
| 6.5 | Unsere netten Nachbarn helfen viel. | Plural com possessivo pede adjetivo `-en`. |
| 6.6 | Das ist kein kleines Problem. | `Problem` é neutro; a forma correta é `kein kleines`. |
