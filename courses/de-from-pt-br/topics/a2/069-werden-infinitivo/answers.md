# Respostas: Werden + infinitivo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | werde | Com **ich**, a forma correta é **werde**. |
| 1.2 | wirst | Com **du**, a forma correta é **wirst**. |
| 1.3 | wird | Com **Lena** no singular, usamos **wird**. |
| 1.4 | werden | Com **wir**, a forma é **werden**. |
| 1.5 | werdet | Com **ihr**, a forma correta é **werdet**. |
| 1.6 | werden | Com **meine Eltern** no plural, usamos **werden**. |
| 2.1 | Ich werde heute kochen. | Em **werden + infinitivo**, o verbo principal fica como infinitivo: **kochen**. |
| 2.2 | Wir werden morgen nach Hamburg fahren. | O infinitivo **fahren** deve aparecer no final da frase. |
| 2.3 | Du wirst Deutsch lernen. | Com **du**, usamos **wirst**; **lernen** fica no infinitivo. |
| 2.4 | Es wird morgen regnen. | Para previsão do tempo, **Es wird ... regnen** é uma forma correta. |
| 3.1 | Ich werde morgen lernen. | O presente com **morgen** foi transformado em futuro com **werde + lernen**. |
| 3.2 | Tim wird später Brot kaufen. | **Tim** é singular, então usamos **wird**; **kaufen** vai ao final. |
| 3.3 | Wir werden Anna am Sonntag treffen. | Com **wir**, usamos **werden** e o infinitivo **treffen**. |
| 3.4 | Ihr werdet nächste Woche einen Test machen. | Com **ihr**, usamos **werdet** e o infinitivo **machen**. |
| 3.5 | Sie wird ihre Großmutter am Abend besuchen. | **Sie** singular pede **wird**; o verbo principal é **besuchen**. |
| 4.1 | Ich werde dich morgen anrufen. | **anrufen** fica no infinitivo no final; **dich** traduz “você” como objeto. |
| 4.2 | Wir werden am Samstag üben. | Com **wir**, usamos **werden**; **üben** fica no infinitivo. |
| 4.3 | Es wird heute Abend kalt. | Para “vai ficar frio”, usa-se **Es wird kalt**. |
| 4.4 | Sie wird ein Geschenk kaufen. | **Sie** singular usa **wird** e **kaufen** fica no final. |
| 4.5 | Ihr werdet im Sommer reisen. | Com **ihr**, a forma correta é **werdet**. |
| 5.1 | futuro com werden | A frase contém **werde** + infinitivo **gehen**. |
| 5.2 | presente usado como futuro | **beginnt** está no presente, mas **am Montag** indica futuro. |
| 5.3 | futuro com werden | A frase usa **werden** + infinitivo **lernen**. |
| 5.4 | presente usado como futuro | **sehe** está no presente, com indicação futura **heute Abend**. |
| 5.5 | futuro com werden | A previsão usa **wird** + infinitivo **schneien**. |
| 6.1 | Ich werde morgen lesen. | A ordem correta é sujeito + **werde** + tempo + infinitivo. |
| 6.2 | Du wirst später essen. | Com **du**, usamos **wirst** e o infinitivo **essen**. |
| 6.3 | Wir werden am Freitag tanzen. | **tanzen** fica no final como infinitivo. |
| 6.4 | Es wird heute regnen. | Para clima, a forma **Es wird ... regnen** é natural. |
| 6.5 | Max wird nächste Woche umziehen. | **Max** é singular, então usamos **wird**; **umziehen** fica no final. |

## Perguntas da história

| Pergunta | Resposta |
|---|---|
| Was wird Mara am Montag machen? | Sie wird Deutsch lernen. |
| Warum wird Mara Deutsch lernen? | Weil sie bald einen Test hat. |
| Wen wird Mara am Dienstag anrufen? | Sie wird ihre Freundin Nele anrufen. |
| Wie wird das Wetter am Donnerstag? | Es wird regnen und kalt. |
| Wird Mara mit dem Fahrrad zur Arbeit fahren? | Nein, sie wird mit dem Bus fahren. |
| Qual frase da história usa presente com sentido de futuro? | **Morgen gehen wir ins Museum.** O presente **gehen** fala de futuro por causa de **morgen**. |
| Complete em alemão: Am Sonntag ___ Mara nichts ___. | **Am Sonntag wird Mara nichts machen.** Usa **wird** + infinitivo **machen**. |
| Prática 1 | **Mara wird morgen ins Museum gehen.** O infinitivo **gehen** vai ao final. |
| Prática 2 | **Es wird kalt.** Para previsão/resultado futuro, usa-se **wird**. |
| Prática 3 | **Wir werden zusammen üben.** Com **wir**, usamos **werden**. |
| Prática 4 | Exemplo: **Ich werde morgen Deutsch lernen.** A frase deve ter **Ich werde** + infinitivo no final. |
