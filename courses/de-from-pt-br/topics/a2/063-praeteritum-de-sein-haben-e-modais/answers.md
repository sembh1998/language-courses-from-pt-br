# Respostas: Präteritum de sein, haben e modais

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | war | Com **ich**, o Präteritum de **sein** é **war**. |
| 1.2 | waren | Com **wir**, o Präteritum de **sein** é **waren**. |
| 1.3 | hatte | Com **Lisa/sie**, o Präteritum de **haben** é **hatte**. |
| 1.4 | hattet | Com **ihr**, o Präteritum de **haben** é **hattet**. |
| 1.5 | warst | Com **du**, o Präteritum de **sein** é **warst**. |
| 2.1 | musste | **Musste** é a forma de passado de **müssen** para **ich**. |
| 2.2 | durften | Com **die Kinder/sie**, o passado de **dürfen** é **durften**. |
| 2.3 | konnte | **Konnte** é o Präteritum de **können** para **er**. |
| 2.4 | wollten | Com **wir**, o Präteritum de **wollen** é **wollten**. |
| 2.5 | solltest | Com **du**, o Präteritum de **sollen** é **solltest**. |
| 3.1 | Ich war krank. | **bin** vira **war** no Präteritum. |
| 3.2 | Wir hatten viel Arbeit. | **haben** com **wir** vira **hatten**. |
| 3.3 | Sie konnte heute nicht kommen. | O modal **kann** vira **konnte**; **kommen** fica no infinitivo. |
| 3.4 | Paul musste früh aufstehen. | **muss** vira **musste**; **aufstehen** fica no infinitivo final. |
| 3.5 | Die Kinder durften fernsehen. | **dürfen** com plural vira **durften** no passado. |
| 4.1 | Ich war gestern müde. | Para **estar** no passado, use **war**, não **warte**. |
| 4.2 | Wir hatten keine Tickets. | Com **wir**, a forma correta é **hatten**. |
| 4.3 | Sie musste lernen. | Depois de modal, o verbo principal fica no infinitivo: **lernen**. |
| 4.4 | Er konnte gestern nicht kommen. | Aqui é passado real de **können**: **konnte**, não **könnte**. |
| 4.5 | Ihr wart im Büro. | Com **ihr**, o Präteritum de **sein** é **wart**. |
| 5.1 | estava; era | **war** traduz **estava/era** conforme o contexto. |
| 5.2 | tinha | **hatte** é o passado de **haben** para **ich/er/sie/es**. |
| 5.3 | pude; podia; conseguia | **konnte** expressa capacidade ou possibilidade no passado. |
| 5.4 | tive que; precisava | **musste** expressa obrigação ou necessidade no passado. |
| 5.5 | pude; tinha permissão para | **durfte** expressa permissão no passado. |
| 5.6 | devia; deveria | **sollte** expressa dever, conselho ou expectativa no passado. |
| 6.1 | Ich war zu Hause. | **Eu estava** corresponde a **ich war**. |
| 6.2 | Wir hatten Zeit. | **Nós tínhamos** corresponde a **wir hatten**. |
| 6.3 | Sie wollte schlafen. | **Queria** é **wollte**; **schlafen** fica no infinitivo. |
| 6.4 | Ich musste lernen. | **Tive que** corresponde a **musste**; o verbo fica no infinitivo. |
| 6.5 | Sie durften spielen. | Como é permissão, use **durften**, não **konnten**. |
