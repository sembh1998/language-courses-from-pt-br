# Respostas: Comparativo

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | kleiner | **klein** forma o comparativo regular **kleiner**. |
| 1.2 | länger | **lang** recebe Umlaut no comparativo: **länger**. |
| 1.3 | schneller | O comparativo regular usa **-er**: **schneller**. |
| 1.4 | billiger | **billig** vira **billiger** com **-er**. |
| 1.5 | wärmer | **warm** recebe Umlaut: **wärmer**. |
| 2.1 | schwerer | Para “mais difícil”, use **schwerer**, não **mehr schwer**. |
| 2.2 | besser | **gut** é irregular e vira **besser**. |
| 2.3 | mehr | Para quantidade, **viel** vira **mehr**. |
| 2.4 | größer | **groß** vira **größer** com Umlaut. |
| 3.1 | als | Em comparativo de superioridade, “do que” é **als**. |
| 3.2 | als | **älter als** significa “mais velho do que”. |
| 3.3 | als | Depois do comparativo **interessanter**, use **als**. |
| 3.4 | als | **mehr Aufgaben als** expressa “mais tarefas do que”. |
| 4.1 | Mein Zimmer ist kleiner als die Küche. | A frase usa **kleiner als** para “menor do que”. |
| 4.2 | Dieser Film ist besser als die Serie. | **melhor** é **besser**; “do que” é **als**. |
| 4.3 | Ich habe mehr Zeit als Maria. | **mais tempo** é **mehr Zeit**. |
| 4.4 | Der Zug ist teurer als der Bus. | **teuer** vira **teurer** no comparativo. |
| 5.1 | Das Auto ist schneller als das Fahrrad. | O carro é comparado à bicicleta com **schneller als**. |
| 5.2 | Anna hat mehr Kaffee als Tom. | Para quantidade, use **mehr** antes do substantivo. |
| 5.3 | Der neue Test ist besser als der alte Test. | **gut** vira **besser** em comparação. |
| 5.4 | Die Tasche ist kleiner als der Koffer. | A bolsa é o item menor, então **kleiner als**. |
| 6.1 | Mein Buch ist kleiner als dein Buch. | Não se usa **mehr klein**; use **kleiner**. |
| 6.2 | Der Kaffee ist besser als der Tee. | **guter** não é a forma comparativa; o correto é **besser**. |
| 6.3 | Ich brauche mehr Wasser als du. | Para “mais água do que você”, a ordem é **mehr Wasser als du**. |
| 6.4 | Hamburg ist größer als Bremen. | Em comparativo de superioridade, use **als**, não **wie**. |
