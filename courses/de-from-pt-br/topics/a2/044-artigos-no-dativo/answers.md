# Respostas: Artigos no dativo

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | dem | **Bus** é masculino e aparece depois de **mit**, então usa dativo masculino: **dem**. |
| 1.2 | der | **Ärztin** é feminino; no dativo definido, **die** vira **der**. |
| 1.3 | dem | **Tisch** é masculino; com posição fixa, usa dativo: **dem Tisch**. |
| 1.4 | den | **Nachbarn** está no plural; o dativo definido plural usa **den**. |
| 1.5 | dem | **Büro** é neutro; depois de **aus**, usa dativo: **dem**. |
| 1.6 | den | **Kolleginnen** é plural; no dativo definido plural, o artigo é **den**. |
| 2.1 | dem | **Vater** é masculino no dativo: **dem Vater**. |
| 2.2 | dem | **Fahrrad** é neutro; depois de **mit**, usa **dem**. |
| 2.3 | den | **Freunden** é plural dativo: **den** + plural com **-n**. |
| 2.4 | der | **Lehrerin** é feminino; no dativo fica **der Lehrerin**. |
| 2.5 | dem | **Hund** é masculino; depois de **mit**, usa **dem Hund**. |
| 2.6 | der | **Arbeit** é feminino; depois de **von**, usa dativo feminino: **der**. |
| 3.1 | einem | **Freund** é masculino; artigo indefinido no dativo masculino: **einem**. |
| 3.2 | einer | **Tante** é feminino; artigo indefinido no dativo feminino: **einer**. |
| 3.3 | einem | **Museum** é neutro; artigo indefinido no dativo neutro: **einem**. |
| 3.4 | einer | **Nachbarin** é feminino; no dativo indefinido usa **einer**. |
| 3.5 | einem | **Lehrer** é masculino; com o verbo **schreiben** para uma pessoa, usa dativo. |
| 3.6 | einem | **Kind** é neutro; depois de **mit**, fica **einem Kind**. |
| 4.1 | mit dem Bruder | **Bruder** é masculino; no dativo definido usa **dem**. |
| 4.2 | bei der Oma | **Oma** é feminino; depois de **bei**, usa **der**. |
| 4.3 | zu dem Hotel | **Hotel** é neutro; depois de **zu**, usa **dem**. |
| 4.4 | aus der Schule | **Schule** é feminino; depois de **aus**, usa **der**. |
| 4.5 | von den Eltern | **Eltern** é plural; no dativo definido plural, usa **den**. |
| 4.6 | mit den Kindern | **Kinder** é plural e recebe **-n** no dativo: **Kindern**. |
| 5.1 | keinem | **Mann** é masculino; negativo no dativo masculino: **keinem**. |
| 5.2 | keiner | **Frau** é feminino; negativo no dativo feminino: **keiner**. |
| 5.3 | keinem | **Kind** é neutro; negativo no dativo neutro: **keinem**. |
| 5.4 | keinen | **Freunden** é plural; negativo no dativo plural: **keinen**. |
| 5.5 | keinem | **Auto** é neutro; depois de **mit**, usa **keinem Auto**. |
| 5.6 | keiner | **Kollegin** é feminino; depois de **mit**, usa **keiner Kollegin**. |
| 6.1 | Ich fahre mit dem Zug. | **Zug** é masculino; depois de **mit**, o correto é **dem Zug**. |
| 6.2 | Sie ist bei der Mutter. | **Mutter** é feminino; depois de **bei**, usa **der**. |
| 6.3 | Wir spielen mit dem Kind. | **Kind** é neutro singular; depois de **mit**, usa **dem**. |
| 6.4 | Er spricht mit den Freunden. | No plural dativo, usa **den** e acrescenta **-n**: **Freunden**. |
| 6.5 | Das Paket ist für den Lehrer. | **für** pede acusativo, não dativo; por isso é **den Lehrer**. |
| 6.6 | Ich komme aus dem Haus. | **Haus** é neutro; depois de **aus**, usa **dem Haus**. |
