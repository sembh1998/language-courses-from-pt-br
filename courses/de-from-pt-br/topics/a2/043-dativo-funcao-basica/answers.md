# Respostas: Dativo: função básica

Nível: A2

## Gabarito dos exercícios

| ID | Resposta | Explicação |
|---|---|---|
| 1.1 | dem | **Mann** é masculino; no dativo fica **dem Mann**. |
| 1.2 | der | **Frau** é feminino; no dativo fica **der Frau**. |
| 1.3 | dem | **Kind** é neutro; no dativo fica **dem Kind**. |
| 1.4 | den | **Schülern** é plural; no dativo plural usa **den**. |
| 1.5 | der | **Freundin** é feminino; no dativo fica **der Freundin**. |
| 2.1 | dem | **Bruder** é masculino e recebe a explicação; usa dativo. |
| 2.2 | der | **Nachbarin** é feminino; com **helfen**, a pessoa fica no dativo. |
| 2.3 | den | **Kindern** é plural dativo: **den Kindern**. |
| 2.4 | dem | **Vater** é masculino; a mensagem é escrita para ele. |
| 2.5 | den | **Kollegen** é plural aqui; objeto indireto no dativo plural. |
| 2.6 | der | **Schwester** é feminino; no dativo fica **der Schwester**. |
| 3.1 | mir | A pessoa que recebe ajuda é “eu”; o dativo de **ich** é **mir**. |
| 3.2 | dir | Para “você”, o pronome dativo é **dir**. |
| 3.3 | ihm | Tom é masculino; o pronome dativo correspondente é **ihm**. |
| 3.4 | ihr | Eva é feminino; o pronome dativo correspondente é **ihr**. |
| 3.5 | uns | Para “nós”, o pronome dativo é **uns**. |
| 3.6 | euch | Para **ihr** no sentido de “vocês”, o dativo é **euch**. |
| 4.1 | Ich gebe dem Touristen das Ticket. | **Tourist** é masculino fraco; aqui aparece como **dem Touristen** no dativo. |
| 4.2 | Sie erklärt der Klasse die Übung. | A classe recebe a explicação; feminino no dativo: **der Klasse**. |
| 4.3 | Wir schenken der Großmutter Blumen. | A avó recebe as flores; feminino no dativo: **der Großmutter**. |
| 4.4 | Er zeigt dem Team den Plan. | **Team** é neutro; no dativo fica **dem Team**. |
| 4.5 | Maria schreibt den Eltern eine Karte. | **Eltern** é plural; dativo plural usa **den**. |
| 5.1 | Ich gebe meiner Mutter ein Geschenk. | **Mutter** é feminino; com possessivo no dativo: **meiner Mutter**. |
| 5.2 | Ich helfe meinem Freund am Samstag. | **helfen** pede dativo; masculino com possessivo: **meinem Freund**. |
| 5.3 | Ich schreibe meiner Lehrerin eine Nachricht. | A professora é destinatária; feminino dativo: **meiner Lehrerin**. |
| 5.4 | Ich zeige meinen Freunden die Stadt. | Plural dativo com possessivo: **meinen Freunden**. |
| 6.1 | Ich helfe der Frau. | **Frau** é feminino; a forma dativa correta é **der Frau**. |
| 6.2 | Er gibt dem Kind den Saft. | **Kind** é neutro; a pessoa que recebe fica em **dem Kind**. |
| 6.3 | Wir erklären den Studenten die Regel. | Os estudantes recebem a explicação; plural dativo: **den Studenten**. |
| 6.4 | Sie schreibt der Freundin eine E-Mail. | **Freundin** é feminino; no dativo fica **der Freundin**. |
| 6.5 | Kannst du mir helfen? | O pronome dativo de **ich** é **mir**, não **ich**. |
