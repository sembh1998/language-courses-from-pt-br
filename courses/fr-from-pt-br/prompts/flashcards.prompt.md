# Flashcards Prompt

Generate `flashcards.yaml` for a French learning topic for Brazilian Portuguese speakers.

If flashcards are useful, create concise cards with `front` in French, `back` in Brazilian Portuguese, `example` in French, and `example_translation` in Brazilian Portuguese. If flashcards are not useful, use `cards: []` and a clear `note` explaining why.

Keep `front` and `example` clean French text with no Markdown or parenthetical Portuguese.
